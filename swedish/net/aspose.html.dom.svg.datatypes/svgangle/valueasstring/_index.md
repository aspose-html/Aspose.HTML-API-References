## SVGAngle.ValueAsString property

Vinkelvärdet som ett strängvärde, i enheterna uttryckta av unitType. Om du ställer in detta attribut kommer värde, valueInSpecifiedUnits och unitType att uppdateras automatiskt för att återspegla denna inställning.

```csharp
public string ValueAsString { get; set; }
```

### Fastighetsvärde

Värdet som sträng.
