## SVGTransform.SetRotate method

Ställer in transformeringstypen till SVG_TRANSFORM_ROTATE, med parametervinkeln som definierar rotationsvinkeln och parametrarna cx och cy definierar det valfria rotationscentrumet.

```csharp
public void SetRotate(float angle, float cx, float cy)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| angle | Single | Rotationsvinkeln. |
| cx | Single | X-koordinaten för rotationscentrum. |
| cy | Single | Y-koordinaten för rotationscentrum. |
