## SVGTransform.Angle property

Ett bekvämlighetsattribut för SVG_TRANSFORM_ROTATE, SVG_TRANSFORM_SKEWX och SVG_TRANSFORM_SKEWY. Den innehåller vinkeln som specificerades. För SVG_TRANSFORM_MATRIX, SVG_TRANSFORM_TRANSLATE och SVG_TRANSFORM_SCALE kommer vinkeln att vara noll.

```csharp
public float Angle { get; }
```
