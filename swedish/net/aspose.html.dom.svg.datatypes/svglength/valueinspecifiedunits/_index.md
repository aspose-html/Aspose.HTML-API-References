## SVGLength.ValueInSpecifiedUnits property

Värdet som ett flyttalsvärde, i enheterna uttryckta av unitType. Om du ställer in detta attribut kommer värde och valueAsString att uppdateras automatiskt för att återspegla denna inställning.

```csharp
public float ValueInSpecifiedUnits { get; set; }
```

### Fastighetsvärde

Värdet i angivna enheter.
