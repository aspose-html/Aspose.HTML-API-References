## Document.GetElementById method

Returnerar elementet som har ett ID-attribut med det angivna värdet. Om inget sådant element finns returnerar detta null. Om mer än ett element har ett ID-attribut med det värdet, är det som returneras odefinierat.

public Element GetElementById(string elementId)
