## IXPathNSResolver.LookupNamespaceURI method

Slå upp namnutrymmes-URI som är kopplat till det givna namnområdesprefixet. XPath-utvärderaren får aldrig kalla detta med en`null` eller tomt argument, eftersom resultatet av att göra detta är odefinierat.

```csharp
public string LookupNamespaceURI(string prefix)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| prefix | String | Prefixet att leta efter. |

### Returvärde

Returnerar den associerade namnutrymmes-URI eller`null` om ingen hittas.
