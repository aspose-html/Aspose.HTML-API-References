## HtmlRenderer.Render method

Definierar metod för att rendera flera[`Document`](../../../aspose.html.dom/document/) är i specifika[`IDevice`](../../idevice/) . Renderingen kommer att utföras när det inte finns några nätverksoperationer för att ladda resurser, aktiva timers, animeringsuppgifter eller specificerad timeout har förflutit.

```csharp
public override void Render(IDevice device, TimeSpan timeout, params Document[] documents)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| device | IDevice | Utmatningsenheten. |
| timeout | TimeSpan | ATimeSpan som representerar antalet millisekunder att vänta, eller aTimeSpan som representerar -1 millisekund för att vänta på obestämd tid. |
| documents | Document[] | Dokumenten som ska återges. |
