## Arc(double, double, double, double, double) {#arc}

Lägger till en båge till banan som är centrerad vid (x, y) position med radie r som börjar vid startAngle och slutar vid endAngle går i den angivna riktningen moturs (standard till medurs).

```csharp
public void Arc(double x, double y, double radius, double startAngle, double endAngle)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x | Double | X-koordinaten för bågens centrum. |
| y | Double | Y-koordinaten för bågens centrum. |
| radius | Double | Bågens radie. |
| startAngle | Double | Vinkeln med vilken bågen börjar, mätt medurs från den positiva x-axeln och uttryckt i radianer. |
| endAngle | Double | Vinkeln med vilken bågen slutar, mätt medurs från den positiva x-axeln och uttryckt i radianer. |
