## ICanvasPathMethods.BezierCurveTo method

Lägger till en kubisk Bézier-kurva till sökvägen. Det krävs tre poäng. De två första punkterna är kontrollpunkter och den tredje är slutpunkten. Startpunkten är den sista punkten i den aktuella sökvägen, som kan ändras med moveTo() innan Bézier-kurvan skapas.

```csharp
public void BezierCurveTo(double cp1x, double cp1y, double cp2x, double cp2y, double x, double y)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| cp1x | Double | X-axeln för koordinaten för den första styrpunkten. |
| cp1y | Double | Y-axeln för koordinaten för den första styrpunkten. |
| cp2x | Double | X-axeln för koordinaten för den andra styrpunkten. |
| cp2y | Double | Y-axeln för koordinaten för den andra styrpunkten. |
| x | Double | X-axeln för koordinaten för slutpunkten. |
| y | Double | Y-axeln för koordinaten för slutpunkten. |
