## CreateImageData(double, double) {#createimagedata_1}

Skapar ett nytt tomt ImageData-objekt med de angivna måtten. Alla pixlar i det nya objektet är transparenta svarta.

```csharp
public IImageData CreateImageData(double sw, double sh)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| sw | Double | Bredden för att ge det nya ImageData-objektet. |
| sh | Double | Höjden för att ge det nya ImageData-objektet. |

### Returvärde

Ett nytt ImageData-objekt med angiven bredd och höjd. Det nya objektet är fyllt med genomskinliga svarta pixlar.
