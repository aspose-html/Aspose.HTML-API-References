## ICanvasRenderingContext2D.GetImageData method

Returnerar ett ImageData-objekt som representerar underliggande pixeldata för området på duken som betecknas av rektangeln som börjar på (sx, sy) och har en sw bredd och sh höjd. Denna metod påverkas inte av arbetsytans transformationsmatris.

```csharp
public IImageData GetImageData(double sx, double sy, double sw, double sh)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| sx | Double | X-koordinaten för det övre vänstra hörnet av rektangeln från vilken ImageData kommer att extraheras. |
| sy | Double | Y-koordinaten för det övre vänstra hörnet av rektangeln från vilken ImageData kommer att extraheras. |
| sw | Double | Bredden på rektangeln från vilken ImageData kommer att extraheras. |
| sh | Double | Höjden på rektangeln från vilken ImageData kommer att extraheras. |

### Returvärde

Ett ImageData-objekt som innehåller bilddata för den givna rektangeln på duken.
