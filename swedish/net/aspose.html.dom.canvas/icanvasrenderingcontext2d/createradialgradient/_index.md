## ICanvasRenderingContext2D.CreateRadialGradient method

Skapar en radiell gradient som ges av koordinaterna för de två cirklarna som representeras av parametrarna.

```csharp
public ICanvasGradient CreateRadialGradient(double x0, double y0, double r0, double x1, double y1, 
    double r1)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| x0 | Double | X-axeln för startcirkelns koordinat. |
| y0 | Double | Y-axeln för startcirkelns koordinat |
| r0 | Double | Startcirkelns radie. |
| x1 | Double | X-axeln för koordinaten för ändcirkeln. |
| y1 | Double | Y-axeln för koordinaten för ändcirkeln. |
| r1 | Double | Radien för ändcirkeln. |

### Returvärde

En radiell CanvasGradient initierad med de två angivna cirklarna.
