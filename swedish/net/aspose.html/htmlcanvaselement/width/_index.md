## HTMLCanvasElement.Width property

Är ett positivt heltal som återspeglar width HTML-attributet för canvas-elementet tolkat i CSS-pixlar. När attributet inte är specificerat, eller om det är inställt på ett ogiltigt värde, som ett negativt, används standardvärdet 300.

```csharp
public ulong Width { get; set; }
```
