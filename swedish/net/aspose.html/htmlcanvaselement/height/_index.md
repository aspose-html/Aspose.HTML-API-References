## HTMLCanvasElement.Height property

Är ett positivt heltal som återspeglar höjd HTML-attributet för canvas-elementet tolkat i CSS-pixlar. När attributet inte är specificerat, eller om det är inställt på ett ogiltigt värde, som ett negativt, används standardvärdet 150.

```csharp
public ulong Height { get; set; }
```
