## HTMLCanvasElement.GetContext method

Returnerar en ritkontext på arbetsytan, eller null om kontext-ID:t inte stöds. En ritkontext låter dig rita på duken.

```csharp
public object GetContext(string contextId)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| contextId | String | Är en sträng som innehåller kontextidentifieraren som definierar ritningskontexten som är kopplad till arbetsytan |

### Returvärde

En renderingskontext
