## HTMLTableSectionElement.DeleteRow method

Ta bort en rad från det här avsnittet.

```csharp
public void DeleteRow(int index)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| index | Int32 | Indexet för raden som ska raderas, eller -1 för att ta bort den sista raden. Detta index börjar från 0 och är endast relativt till raderna som finns i det här avsnittet, inte alla rader i tabellen. |

### Undantag

| undantag | skick |
| --- | --- |
| [DOMException](../../../aspose.html.dom/domexception/) | INDEX_SIZE_ERR: Ökas om det angivna indexet är större än eller lika med antalet rader eller om indexet är ett negativt tal annat än -1. @version DOM Nivå 2 |
