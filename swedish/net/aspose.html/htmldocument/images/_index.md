## HTMLDocument.Images property

En samling av alla`IMG` element i ett dokument. Beteendet är begränsat till`IMG` element för bakåtkompatibilitet . Som föreslagits av [[HTML 4.01](http://www.w3.org/TR/1999/REC-html401-19991224)], för att inkludera bilder kan författare använda the`OBJEKT` element eller`IMG` element. Därför rekommenderas det att inte använda detta attribut för att hitta -bilderna i dokumentet, men`getElementsByTagName` med HTML 4.01 eller`getElementsByTagNameNS` med XHTML 1.0.

public HTMLCollection Images { get; }
