## HTMLInputElement.Files property

Files IDL-attributet tillåter skript att komma åt elementets valda filer. Om IDL-attributet gäller måste det returnera ett FileList-objekt som representerar de aktuella valda filerna. Samma objekt måste returneras tills listan över valda filer ändras. Om IDL-attributet inte gäller måste det istället returnera null. [FILEAPI]

```csharp
public IFileList Files { get; }
```
