## HTMLInputElement.DefaultValue property

När`typ`elementets attribut har värdet "text", "file" eller "lösenord", detta representerar HTML-värdet attribut för elementet. Värdet på detta attribut ändras inte om innehållet i motsvarande formulärkontroll, i en interaktiv användaragent, ändras. Se definitionen av värdeattributet i HTML 4.01.

```csharp
public string DefaultValue { get; set; }
```
