## HTMLInputElement.DefaultChecked property

När`typ` har värdet "radio" eller "checkbox", representerar denna det HTML-kontrollerade attributet för elementet. Värdet på detta attribut ändras inte om tillståndet för motsvarande form kontroll, i en interaktiv användaragent, ändras. Se den markerade -attributdefinitionen i HTML 4.01.

```csharp
public bool DefaultChecked { get; set; }
```
