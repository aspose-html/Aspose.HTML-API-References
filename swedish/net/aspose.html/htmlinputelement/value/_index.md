## HTMLInputElement.Value property

När`typ` elementets attribut har värdet "text", "fil" eller "lösenord", detta representerar det aktuella innehållet i motsvarande formulärkontroll, i en interaktiv användaragent. Ändring av detta attribut ändrar innehållet i formulärkontrollen, men ändrar inte värdet på HTML-värdeattributet för elementet. När`typ`elementets attribut har värdet "button", "hidden", "submit", "reset", "image", "checkbox" eller "radio", detta representerar elementets HTML-värdeattribut. Se definitionen av värdeattributet i HTML 4.01.

```csharp
public string Value { get; set; }
```
