## HTMLTextAreaElement.Value property

Representerar det aktuella innehållet i motsvarande formulärkontroll, i en interaktiv användaragent. Ändring av detta attribut ändrar innehållet i formulärkontrollen, men ändrar inte innehållet i elementet . Om hela data inte får plats i en enda `DOMString` , kan implementeringen trunkera data.

```csharp
public string Value { get; set; }
```
