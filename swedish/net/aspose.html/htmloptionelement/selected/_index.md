## HTMLOptionElement.Selected property

Representerar det aktuella tillståndet för motsvarande formulärkontroll, i en interaktiv användaragent. Om du ändrar det här attributet ändras statusen för formulärkontrollen, men ändrar inte värdet på det HTML-valda -attributet för elementet.

```csharp
public bool Selected { get; set; }
```
