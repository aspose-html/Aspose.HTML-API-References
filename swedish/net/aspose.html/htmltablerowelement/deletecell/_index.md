## HTMLTableRowElement.DeleteCell method

Ta bort en cell från den aktuella raden.

```csharp
public void DeleteCell(int index)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| index | Int32 | Indexet för cellen som ska raderas, med början från 0. Om -indexet är -1 tas den sista cellen i raden bort. |

| undantag | skick |
| --- | --- |
