## HTMLTableElement.DeleteRow method

Ta bort en tabellrad.

```csharp
public void DeleteRow(int index)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| index | Int32 | Indexet för raden som ska raderas. Detta index börjar från 0 och är relativt den logiska ordningen (inte dokumentordningen) för alla rader som finns i tabellen. Om indexet är -1 raderas den sista raden i tabellen. |

### Undantag

| undantag | skick |
| --- | --- |
| [DOMException](../../../aspose.html.dom/domexception/) | INDEX_SIZE_ERR: Ökas om det angivna indexet är större än eller lika med antalet rader eller om indexet är ett negativt tal annat än -1. @version DOM Nivå 2 |
