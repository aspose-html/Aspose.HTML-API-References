## FontEmbeddingRule enumeration

Representerar reglerna för inbäddning av teckensnitt.

```csharp
public enum FontEmbeddingRule
```

### Värderingar

| namn | Värde | Beskrivning |
| --- | --- | --- |
| Full | `1` | Alla använda typsnitt kommer att bäddas in som de är. Var medveten om att vissa teckensnitt kan vara licensierade, detta kommer att göra det resulterande dokumentet oredigerbart. Även de inbäddade teckensnittsfilerna kan öka storleken på det resulterande dokumentet avsevärt. |
| None | `2` | Teckensnitt kommer inte att bäddas in. |
