## ITreeWalker.PreviousNode method

Flyttar TreeWalker till föregående synliga nod i dokumentordning i förhållande till den aktuella noden och returnerar noden new . Om den aktuella noden inte har någon tidigare nod, eller om sökningen efter previousNode försöker kliva uppåt från TreeWalkers rotnod, returnerar null och behåller den nuvarande noden.

```csharp
public Node PreviousNode()
```

### Returvärde

Den nya noden, eller null om den nuvarande noden har no föregående nod i TreeWalkers logiska vy.
