## IStyleSheet.Disabled property

false om stilmallen tillämpas på dokumentet. sant om det inte är det. Ändring av detta attribut kan orsaka en ny stilupplösning för dokumentet. En stilmall gäller endast om både en lämplig mediumdefinition finns och attributet disabled är falskt. Så om mediet inte gäller den aktuella användaragenten ignoreras attributet disabled.

```csharp
public bool Disabled { get; set; }
```

### Fastighetsvärde

`Sann` om inaktiverad; annat,`falsk` .
