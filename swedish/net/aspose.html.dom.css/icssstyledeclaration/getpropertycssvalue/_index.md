## ICSSStyleDeclaration.GetPropertyCSSValue method

Används för att hämta objektrepresentationen av värdet på en CSS-egenskap om den uttryckligen har ställts in i detta deklarationsblock. Denna metod returnerar null om egenskapen är en stenografisk egenskap. Egenskapsvärden för stenografi kan endast nås och ändras som strängar med metoderna getPropertyValue och setProperty.

```csharp
public CSSValue GetPropertyCSSValue(string propertyName)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| propertyName | String | Fastighetens namn. |

### Returvärde

Returnerar egenskapens värde
