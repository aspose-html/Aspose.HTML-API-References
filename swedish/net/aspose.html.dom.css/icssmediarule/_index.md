## ICSSMediaRule interface

CSSMediaRule-gränssnittet representerar en @media-regel i en CSS-stilmall. En @mediaregel kan användas för att avgränsa stilregler för specifika mediatyper.

public interface ICSSMediaRule : ICSSRule
