## TemplateLoadOptions.NamesAreCaseSensitive property

テンプレートとデータ要素の名前が大文字と小文字に関係なく一致するかどうかを指定するプロパティ。 true の場合、一致は大文字と小文字を区別します。 false の場合、大文字小文字は無視されます。 デフォルト値は true です。

```csharp
public bool NamesAreCaseSensitive { get; set; }
```
