**Aspose.Html.Saving**名前空間は、 変換/保存プロセスでの特定の保存オプションの説明のために、 データ クラスによって提示されます。

## クラス

| クラス | 説明 |
| --- | --- |
| [DocSaveOptions](./docsaveoptions/) | 特定のオプション データ クラス. |
| [HTMLSaveOptions](./htmlsaveoptions/) | HTML 保存オプションを表します。 |
| [ImageSaveOptions](./imagesaveoptions/) | 特定のオプション データ クラス. |
| [MarkdownSaveOptions](./markdownsaveoptions/) | Markdown 保存オプションを表します。 |
| [MHTMLSaveOptions](./mhtmlsaveoptions/) | MHTML 保存オプションを表します。 |
| [PdfSaveOptions](./pdfsaveoptions/) | 特定のオプション データ クラス. |
| [ResourceHandlingOptions](./resourcehandlingoptions/) | リソース処理オプションを表します。 |
| [SaveOptions](./saveoptions/) | これは、ドキュメントを特定の形式で保存するときにユーザーが追加オプションを指定できるようにするクラスの抽象基本クラスです。 |
| [XpsSaveOptions](./xpssaveoptions/) | 特定のオプション データ クラス. |
## 列挙

| 列挙 | 説明 |
| --- | --- |
| [HTMLSaveFormat](./htmlsaveformat/) | ドキュメントを保存する形式を指定します。 |
| [MarkdownFeatures](./markdownfeatures/) | A[`MarkdownFeatures`](../aspose.html.saving/markdownfeatures/)フラグ セットは、次のフラグの 0 個以上のセットであり、markdown に変換される要素を選択するために使用されます。 |
| [MarkdownFormatter](./markdownformatter/) | 出力のフォーマット方法を指定します。 |
| [ResourceHandling](./resourcehandling/) | この列挙型は、リソース処理オプションを表します。 |
| [UrlRestriction](./urlrestriction/) | この列挙型は、処理されたリソースの URL に適用される制限を表します。 |
