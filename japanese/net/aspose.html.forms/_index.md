名前空間は、Web ページ コンポーネントとしてフォーム内の ユーザーの対話性を担う 多数のオブジェクト (要素) へのアクセスを提供します.

## クラス

| クラス | 説明 |
| --- | --- |
| [ButtonElement](./buttonelement/) | ButtonElement は、HTMLButtonElement に関連付けられたラッパーを表します。 |
| [DataListElement](./datalistelement/) | DataListElement は、HTMLDataListElement に関連付けられているラッパーを表します |
| [FormEditor](./formeditor/) | このクラスは、[`HTMLFormElement`](../aspose.html/htmlformelement/)これにより、.net 開発者が HTML フォームを簡単に編集できるようになります。 |
| [FormElement](./formelement/) | フォーム要素の基本クラスを表します。 |
| [FormElement&lt;T&gt;](./formelement-1/) | [`FormElement`](../aspose.html.forms/formelement/)一般的な field を表します |
| [FormSubmitter](./formsubmitter/) | このクラスでは、指定された準備を行うことができます[`HTMLFormElement`](../aspose.html/htmlformelement/)、フォーム要素から値を収集し、それらをリモートサーバーに送信して応答を受け取ります. |
| [InputElement](./inputelement/) | InputElement は、HTMLInputElement. に関連付けられたラッパーを表します。 |
| [OptionCollection](./optioncollection/) | OptionElements は、IHTMLOptionsCollection に関連付けられているラッパーを表します |
| [OptionElement](./optionelement/) | OptionElement は、HTMLOptionElement に関連付けられているラッパーを表します |
| [SelectElement](./selectelement/) | SelectElement は、HTMLSelectElement に関連付けられているラッパーを表します |
| [SubmissionResult](./submissionresult/) | このクラスは、フォーム データをサーバーに送信した結果を表します。 |
| [TextAreaElement](./textareaelement/) | TextAreaElement は、HTMLTextAreaElement に関連付けられているラッパーを表します |
## 列挙

| 列挙 | 説明 |
| --- | --- |
| [ButtonType](./buttontype/) | この列挙は、ボタンの状態を表します. |
| [FormElementType](./formelementtype/) | フォーム要素タイプの列挙と、HTML 要素に対応するものを表します |
| [InputElementType](./inputelementtype/) | 入力フィールドの状態を表します。 |
