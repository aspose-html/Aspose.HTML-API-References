## OptionElement.Selected property

対話型ユーザー エージェントで、対応するフォーム コントロールの現在の状態を表します。この属性を変更すると、フォーム コントロールの状態が変更されますが、要素の HTML selected 属性の値は変更されません.

```csharp
public bool Selected { get; set; }
```
