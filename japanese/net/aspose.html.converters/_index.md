**Aspose.Html.Converters**名前空間の目標は、変換メソッドに簡単にアクセスできることです。 PDF、XPS、画像形式などの一般的な形式への幅広い変換を提供します。 より具体的な変換 (レンダリング、保存) のユーザー ケースは、サブジェクト指向の名前空間でよく知られ、文書化されている低レベル API 関数によって 提示されます。

## クラス

| クラス | 説明 |
| --- | --- |
| [Converter](./converter/) | ほとんどの場合の変換シナリオのみの共有ファサード。 PDF、XPS、画像形式などの一般的な形式への幅広い変換を提供します。 より具体的な変換 (レンダリング、保存) のユーザー ケースは、よく知られているものによって提示されます。および文書化された低レベル API 関数. |
| [TemplateContentOptions](./templatecontentoptions/) | マージ処理用インラインコンテンツオブジェクト. |
| [TemplateData](./templatedata/) | マージ (ユーザー) データ オブジェクト. |
