## Path2D.BezierCurveTo method

3 次ベジエ曲線をパスに追加します。 3点必要です。 最初の 2 点は制御点で、3 番目の点は終点です。 開始点は現在のパスの最後の点 であり、ベジエ曲線を作成する前に moveTo() を使用して変更できます.

```csharp
public void BezierCurveTo(double cp1x, double cp1y, double cp2x, double cp2y, double x, double y)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| cp1x | Double | 最初のコントロール ポイントの座標の x 軸。 |
| cp1y | Double | 最初のコントロール ポイントの座標の y 軸。 |
| cp2x | Double | 2 番目の制御点の座標の x 軸。 |
| cp2y | Double | 2 番目の制御点の座標の y 軸。 |
| x | Double | 終点の座標の x 軸。 |
| y | Double | 終点の座標の y 軸。 |
