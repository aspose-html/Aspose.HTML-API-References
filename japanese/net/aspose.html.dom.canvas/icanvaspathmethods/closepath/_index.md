## ICanvasPathMethods.ClosePath method

ペンのポイントを現在のサブパスの開始点に戻します。 現在点から始点まで直線を引こうとします。 シェイプがすでに閉じているか、ポイントが 1 つしかない場合、この関数は何もしません。

```csharp
public void ClosePath()
```
