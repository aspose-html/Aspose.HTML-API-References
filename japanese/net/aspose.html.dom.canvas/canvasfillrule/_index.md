## CanvasFillRule enumeration

この列挙は、点がパスの内側にあるか外側にあるかを決定する塗りつぶしルール アルゴリズムを選択するために使用されます.

```csharp
public enum CanvasFillRule
```

### 値

| 名前 | 価値 | 説明 |
| --- | --- | --- |
| NonZero | `0` | 値「非ゼロ」値は非ゼロのワインディング ルールを示します。ここで、ある点から引き出された半無限直線が、一方向に進む 形状のパスと交差する回数が 形状の外側にあると見なされます反対方向に進むパスを横切る回数に等しい. |
| EvenOdd | `1` | 「evenodd」値は偶奇規則を示します。この場合、点から引かれた半無限直線が形状のパスと交差する 回数が偶数である場合、その点は形状の外側にあると見なされます. |
