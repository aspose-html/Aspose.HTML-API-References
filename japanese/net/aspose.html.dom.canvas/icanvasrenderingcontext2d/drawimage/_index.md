## DrawImage(HTMLImageElement, double, double, double, double) {#drawimage_4}

指定した画像を描画します。

```csharp
public void DrawImage(HTMLImageElement image, double dx, double dy, double dw, double dh)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| image | HTMLImageElement | コンテキストに描画する HTMLImageElement。 |
| dx | Double | ソース イメージの左上隅を配置する宛先キャンバスの X 座標。 |
| dy | Double | ソース イメージの左上隅を配置する宛先キャンバスの Y 座標。 |
| dw | Double | 宛先キャンバスに画像を描画する幅。これにより、描画された画像のスケーリングが可能になります。指定しない場合、描画時にイメージの幅は拡大縮小されません。 |
| dh | Double | 宛先キャンバスに画像を描画する高さ。これにより、描画された画像のスケーリングが可能になります。指定しない場合、画像は描画時に高さを調整されません。 |

```csharp
public void DrawImage(HTMLImageElement image, double sx, double sy, double sw, double sh, 
    double dx, double dy, double dw, double dh)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| image | HTMLImageElement | コンテキストに描画する HTMLImageElement。 |
| sx | Double | 宛先コンテキストに描画するソース イメージのサブ長方形の左上隅の X 座標。 |

| sh | Double | 宛先コンテキストに描画するソース イメージのサブ長方形の高さ。 |
| dx | Double | ソース イメージの左上隅を配置する宛先キャンバスの X 座標。 |
| dy | Double | ソース イメージの左上隅を配置する宛先キャンバスの Y 座標。 |
| dw | Double | 宛先キャンバスに画像を描画する幅。これにより、描画された画像のスケーリングが可能になります。指定しない場合、描画時にイメージの幅は拡大縮小されません。 |
| dh | Double | 宛先キャンバスに画像を描画する高さ。これにより、描画された画像のスケーリングが可能になります。指定しない場合、画像は描画時に高さを調整されません。 |
