## ICanvasRenderingContext2D.GetImageData method

(sx, sy) で始まり、幅が sw で高さが sh の長方形で示されるキャンバスの領域の基礎となるピクセル データを表す ImageData オブジェクトを返します。 このメソッドは、キャンバスの変換行列の影響を受けません。

```csharp
public IImageData GetImageData(double sx, double sy, double sw, double sh)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| sx | Double | ImageData が抽出される四角形の左上隅の x 座標。 |
| sy | Double | ImageData が抽出される四角形の左上隅の y 座標。 |
| sw | Double | ImageData が抽出される四角形の幅。 |
| sh | Double | ImageData が抽出される四角形の高さ。 |

### 戻り値

キャンバスの指定された長方形の画像データを含む ImageData オブジェクト。
