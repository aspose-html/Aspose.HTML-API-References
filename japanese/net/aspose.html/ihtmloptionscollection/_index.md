## IHTMLOptionsCollection interface

アン`HTMLOptionsCollection`HTML オプション要素を表すノードのリストです。個々のノードは、序数の インデックスまたはノードの`名前`また`ID`属性。 HTML DOM のコレクションはライブであると見なされます。つまり、基になるドキュメントが変更されると 自動的に更新されます。
