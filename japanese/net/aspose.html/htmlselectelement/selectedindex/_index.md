## HTMLSelectElement.SelectedIndex property

0 から始まる、選択されたオプションの序数インデックス。要素が選択されていない場合、値 -1 が返されます。複数のオプションが選択されている場合、最初に選択されたオプションのインデックスが返されます。

```csharp
public int SelectedIndex { get; set; }
```
