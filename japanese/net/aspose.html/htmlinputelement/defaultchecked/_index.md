## HTMLInputElement.DefaultChecked property

いつ`タイプ`値が「radio」または「checkbox」で、この は要素の HTML のチェック済み属性を表します。 この属性の値は、対話型ユーザー エージェント内の対応するフォーム コントロールの状態が変化しても変化しません。 HTML 4.01. のチェック済み 属性定義を参照してください。

```csharp
public bool DefaultChecked { get; set; }
```
