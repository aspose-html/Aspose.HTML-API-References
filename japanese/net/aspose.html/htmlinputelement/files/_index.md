## HTMLInputElement.Files property

files IDL 属性により、スクリプトは要素の選択されたファイルにアクセスできます。 取得時に、IDL 属性が適用される場合、現在選択されているファイルを表す FileList オブジェクトを返す必要があります。 選択したファイルのリストが変更されるまで、同じオブジェクトを返す必要があります。 IDL 属性が適用されない場合は、代わりに null を返す必要があります。 [FILEAPI]

```csharp
public IFileList Files { get; }
```
