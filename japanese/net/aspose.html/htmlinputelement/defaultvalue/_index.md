## HTMLInputElement.DefaultValue property

`タイプ`要素の属性の値は "text"、"file"、または "password" で、要素の HTML 値 属性を表します。この属性の値は、対話型の ユーザー エージェントで、対応するフォーム コントロールのコンテンツが変更された場合でも 変更されません。 HTML 4.01. の value 属性の定義を参照してください。

```csharp
public string DefaultValue { get; set; }
```
