**Aspose.Html**名前空間には、 HTML ドキュメントを操作するためのクラスとメソッドが含まれています。 **Aspose.HTML**for .NET を使用すると、開発者は、挿入、削除、 HTML ノードの置換、CSS スタイル情報の抽出、 トラバーサル仕様によって提供される NodeIterator、 TreeWalker、 XPath または CSS セレクター クエリのいずれかによって、HTML ドキュメントをナビゲートできます。また、to が JavaScript を介して HTML DOM を操作できるようにするスクリプトも提供します。この API は、HTML だけでなく、EPUB と MHTML をロードする機能も提供します 。 Aspose API は インター ファイル フォーマット変換機能で有名であり、この API は HTML ファイルをロードして出力を PDF、 XPS および JPEG、PNG、BMP、TIFF などのラスター イメージ フォーマットでレンダリングする機能も提供します.
