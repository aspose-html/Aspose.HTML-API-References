## ToDataURL() {#todataurl}

は、type パラメーターで指定された形式 (デフォルトは PNG) で画像の表現を含むデータ URI を返します。 返された画像の解像度は 96 dpi です。

```csharp
public string ToDataURL()
```

### 戻り値

要求されたデータ URI を含む DOMString。
