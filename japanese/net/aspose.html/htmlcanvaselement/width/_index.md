## HTMLCanvasElement.Width property

CSS ピクセルで解釈される canvas 要素の幅 HTML 属性を反映する正の整数です。 属性が指定されていない場合、または負の値などの無効な値に設定されている場合、デフォルト値の 300 が使用されます。

```csharp
public ulong Width { get; set; }
```
