## HTMLLinkElement.Disabled property

リンクを有効/無効にします。これは現在、スタイル シート リンクにのみ使用され、スタイル シートをアクティブ化または非アクティブ化するために使用できます。

```csharp
public bool Disabled { get; set; }
```
