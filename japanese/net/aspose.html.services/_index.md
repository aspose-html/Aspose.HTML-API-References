**Aspose.Html.Services**名前空間には、サービス実装のプロトコルとして インターフェイスが含まれています.

## インターフェース

| インターフェース | 説明 |
| --- | --- |
| [IDeviceInformationService](./ideviceinformationservice/) | 環境が記述されているインターフェース。[`Document`](../aspose.html.dom/document/)ユーザーに提示されます. |
| [INetworkService](./inetworkservice/) | ネットワーク操作用のインターフェイスを提供します。 |
| [IRuntimeService](./iruntimeservice/) | このサービスは、ランタイム関連のプロパティを構成するために使用されます. |
| [IService](./iservice/) | サービス実装の基本インターフェースを定義します。 |
| [IServiceContainer](./iservicecontainer/) | サービスのコンテナーを提供します。 |
| [IServiceProvider](./iserviceprovider/) | サービスの不変コンテナーを提供します。 |
| [IUserAgentService](./iuseragentservice/) | ユーザー エージェント環境を表すインターフェイス。 |
