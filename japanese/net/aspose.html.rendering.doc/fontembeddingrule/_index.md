## FontEmbeddingRule enumeration

フォントの埋め込み規則を表します。

```csharp
public enum FontEmbeddingRule
```

### 値

| 名前 | 価値 | 説明 |
| --- | --- | --- |
| Full | `1` | 使用されているすべてのフォントがそのまま埋め込まれます。一部のフォントはライセンスされている可能性があることに注意してください。これにより、結果のドキュメントが編集できなくなります。また、埋め込まれたフォント ファイルにより、結果のドキュメントのサイズが大幅に増加する可能性があります. |
| None | `2` | フォントは埋め込まれません. |
