**Aspose.Html.Rendering.Doc**名前空間は、 固有のデバイス クラスと、DOCX ドキュメントへのレンダリングを担当するいくつかの render オプション クラスを提供します。

## クラス

| クラス | 説明 |
| --- | --- |
| [DocDevice](./docdevice/) | DOCX ドキュメントへのレンダリングを表します。 |
| [DocRenderingOptions](./docrenderingoptions/) | のレンダリング オプションを表します[`DocDevice`](../aspose.html.rendering.doc/docdevice/). |
## 列挙

| 列挙 | 説明 |
| --- | --- |
| [DocumentFormat](./documentformat/) | 出力ドキュメントのファイル形式を表します。 |
| [FontEmbeddingRule](./fontembeddingrule/) | フォントの埋め込み規則を表します。 |
