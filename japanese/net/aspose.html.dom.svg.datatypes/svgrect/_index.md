## SVGRect class

長方形のジオメトリを表します。長方形は、X の最小値、Y の最小値、および幅と高さを識別する (x,y) 座標ペアで構成されるものとして定義されます。これらは通常、負でない値に制限されます。

```csharp
public class SVGRect : SVGValueType
```
