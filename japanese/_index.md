## Aspose.HTML for .NET

{{% alert color="primary" %}} 

![Aspose.HTML for .NET 製品ロゴ](../home_1.png)


Aspose.HTML for .NET は、アプリケーションが幅広い HTML 操作タスクを実行できるようにするクロスプラットフォーム クラス ライブラリです。 HTML は一般に他の技術と組み合わせて使用されるため、Aspose.HTML は HTML5 だけでなく、CSS3、SVG、および HTML Canvas 仕様もサポートしています。 Aspose.HTML を使用すると、Web ページの読み込み、コンテンツの分析と編集、ページの保存と次の形式への変換を行うことができます: HTML、XHTML、MHTML、SVG、Markdown、PDF、XPS、
EPUB、PNG、TIFF、JPEG、BMP、その他の形式。

これらは、いくつかの有用なリソースへのリンクです。

- [Aspose.HTML for .NET API チュートリアル](/tutorials/html/net/)

## Aspose.HTML for Java

{{% alert color="primary" %}} 

![Aspose.HTML for Java 製品ロゴ](../home_2.png)


Aspose.HTML for Java は、Java アプリケーション内でさまざまな HTML 操作タスクを直接実行するために構築された高度な HTML 操作 API です。 この API は、PDF や XPS などの固定レイアウト形式、および多数のラスター イメージ形式用の忠実度の高いレンダリング エンジンも提供します。

{{% /alert %}} 

これらは、いくつかの有用なリソースへのリンクです。
- [Aspose.HTML for .Java API リファレンス](/html/java/)
- [Aspose.HTML for Java API チュートリアル](/tutorials/html/java/)
