## Aspose.HTML per .NET

{{% alert color="primary" %}} 

![Aspose.HTML per il logo del prodotto .NET](../home_1.png)


Aspose.HTML per .NET è una libreria di classi multipiattaforma che consente alle applicazioni di eseguire un'ampia gamma di attività di manipolazione HTML. Poiché l'HTML viene generalmente utilizzato in combinazione con altre tecnologie, Aspose.HTML supporta non solo le specifiche HTML5 ma anche CSS3, SVG e HTML Canvas. Usando Aspose.HTML, puoi caricare pagine web, analizzare e modificare il contenuto, salvare e convertire pagine da e verso i seguenti formati: HTML, XHTML, MHTML, SVG, Markdown, PDF, XPS,
EPUB, PNG, TIFF, JPEG, BMP e altri formati.

## Aspose.HTML per Java

{{% alert color="primary" %}} 

![Aspose.HTML per il logo del prodotto Java](../home_2.png)


Aspose.HTML per Java è un'API di manipolazione HTML avanzata creata per eseguire un'ampia gamma di attività di manipolazione HTML direttamente all'interno delle applicazioni Java. L'API fornisce anche un motore di rendering ad alta fedeltà per formati a layout fisso come PDF e XPS e una serie di formati di immagini raster.
