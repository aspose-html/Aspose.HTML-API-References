## Arc(double, double, double, double, double) {#arc}

Aggiunge un arco al percorso centrato nella posizione (x, y) con raggio r che inizia a startAngle e termina a endAngle andando nella direzione data in senso antiorario (predefinito in senso orario).

```csharp
public void Arc(double x, double y, double radius, double startAngle, double endAngle)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Double | La coordinata x del centro dell'arco. |
| y | Double | La coordinata y del centro dell'arco. |
| radius | Double | Il raggio dell'arco. |
| startAngle | Double | L'angolo di inizio dell'arco, misurato in senso orario dall'asse x positivo ed espresso in radianti. |
| endAngle | Double | L'angolo in cui termina l'arco, misurato in senso orario dall'asse x positivo ed espresso in radianti. |

## Arc(double, double, double, double, double, bool) {#arc_1}

Aggiunge un arco al percorso centrato nella posizione (x, y) con raggio r che inizia a startAngle e termina a endAngle andando nella direzione data in senso antiorario (predefinito in senso orario).

```csharp
public void Arc(double x, double y, double radius, double startAngle, double endAngle, 
    bool counterclockwise)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| x | Double | La coordinata x del centro dell'arco. |
| y | Double | La coordinata y del centro dell'arco. |
| radius | Double | Il raggio dell'arco. |
| startAngle | Double | L'angolo di inizio dell'arco, misurato in senso orario dall'asse x positivo ed espresso in radianti. |
| endAngle | Double | L'angolo in cui termina l'arco, misurato in senso orario dall'asse x positivo ed espresso in radianti. |
| counterclockwise | Boolean | Fa in modo che l'arco venga tracciato in senso antiorario tra i due angoli. Di default è disegnato in senso orario. |
