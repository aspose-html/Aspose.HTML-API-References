## ICanvasPathMethods.BezierCurveTo method

Aggiunge una curva di Bézier cubica al tracciato. Richiede tre punti. I primi due punti sono punti di controllo e il terzo è il punto finale. Il punto di partenza è l'ultimo punto nel percorso corrente, che può essere modificato utilizzando moveTo() prima di creare la curva di Bézier.

```csharp
public void BezierCurveTo(double cp1x, double cp1y, double cp2x, double cp2y, double x, double y)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| cp1x | Double | L'asse x della coordinata per il primo punto di controllo. |
| cp1y | Double | L'asse y della coordinata per il primo punto di controllo. |
| cp2x | Double | L'asse x della coordinata per il secondo punto di controllo. |
| cp2y | Double | L'asse y della coordinata per il secondo punto di controllo. |
| x | Double | L'asse x della coordinata per il punto finale. |
| y | Double | L'asse y della coordinata per il punto finale. |
