## IImageData interface

Crea un oggetto ImageData da un dato Uint8ClampedArray e dalla dimensione dell'immagine che contiene. Se non viene fornito alcun array, crea un'immagine di un rettangolo nero.

```csharp
public interface IImageData
```

## Proprietà

| Nome | Descrizione |
| --- | --- |
| [Data](../../aspose.html.dom.canvas/iimagedata/data/) { get; } | È un Uint8ClampedArray che rappresenta un array unidimensionale contenente i dati nell'ordine RGBA, con valori interi compresi tra 0 e 255 (inclusi). |
| [Height](../../aspose.html.dom.canvas/iimagedata/height/) { get; } | È un long senza segno che rappresenta l'altezza effettiva, in pixel, di ImageData. |
| [Width](../../aspose.html.dom.canvas/iimagedata/width/) { get; } | È un long senza segno che rappresenta la larghezza effettiva, in pixel, di ImageData. |
