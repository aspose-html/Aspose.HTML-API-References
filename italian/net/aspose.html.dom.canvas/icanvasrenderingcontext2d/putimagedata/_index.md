## PutImageData(IImageData, double, double) {#putimagedata}

Dipinge i dati dall'oggetto ImageData specificato sulla bitmap. Se viene fornito un rettangolo sporco, vengono dipinti solo i pixel di quel rettangolo. Questo metodo non è influenzato dalla matrice di trasformazione della tela.

```csharp
public void PutImageData(IImageData imagedata, double dx, double dy)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| imagedata | IImageData | Un oggetto ImageData contenente l'array di valori in pixel. |
| dx | Double | Posizione orizzontale (coordinata x) in cui posizionare i dati dell'immagine nell'area di destinazione. |
| dy | Double | Posizione verticale (coordinata y) in cui posizionare i dati dell'immagine nell'area di destinazione. |

## PutImageData(IImageData, double, double, double, double, double, double) {#putimagedata_1}

Dipinge i dati dall'oggetto ImageData specificato sulla bitmap. Se viene fornito un rettangolo sporco, vengono dipinti solo i pixel di quel rettangolo. Questo metodo non è influenzato dalla matrice di trasformazione della tela.

```csharp
public void PutImageData(IImageData imagedata, double dx, double dy, double dirtyX, double dirtyY, 
    double dirtyWidth, double dirtyHeight)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| imagedata | IImageData | Un oggetto ImageData contenente l'array di valori in pixel. |
| dx | Double | Posizione orizzontale (coordinata x) in cui posizionare i dati dell'immagine nell'area di destinazione. |
| dy | Double | Posizione verticale (coordinata y) in cui posizionare i dati dell'immagine nell'area di destinazione. |
| dirtyX | Double | Posizione orizzontale (coordinata x). La coordinata x dell'angolo in alto a sinistra dei dati dell'immagine. Il valore predefinito è 0. |
| dirtyY | Double | Posizione verticale (coordinata y). La coordinata y dell'angolo in alto a sinistra dei dati dell'immagine. Il valore predefinito è 0. |
| dirtyWidth | Double | Larghezza del rettangolo da dipingere. L'impostazione predefinita è la larghezza dei dati dell'immagine. |
| dirtyHeight | Double | Altezza del rettangolo da dipingere. L'impostazione predefinita è l'altezza dei dati dell'immagine. |
