```csharp
public void StrokeText(string text, double x, double y, double? maxWidth)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| text | String | Il testo da disegnare utilizzando i valori correnti di font, textAlign, textBaseline e direction. |
| x | Double | L'asse x della coordinata per il punto iniziale del testo. |
| y | Double | L'asse y della coordinata per il punto iniziale del testo. |
| maxWidth | Nullable`1 | La larghezza massima da disegnare. Se specificato, e la stringa viene calcolata per essere più larga di questa larghezza, il carattere viene regolato per utilizzare un carattere più condensato orizzontalmente (se disponibile o se uno ragionevolmente leggibile può essere sintetizzato ridimensionando orizzontalmente il carattere corrente) o un carattere più piccolo font. |
