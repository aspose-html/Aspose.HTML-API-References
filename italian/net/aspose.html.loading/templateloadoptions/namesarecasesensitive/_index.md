## TemplateLoadOptions.NamesAreCaseSensitive property

Proprietà che specifica se i nomi del modello e dell'elemento dati verranno abbinati indipendentemente dal carattere maiuscolo o meno. Se vero, la corrispondenza fa distinzione tra maiuscole e minuscole; se false, le maiuscole e minuscole verranno ignorate. Il valore predefinito è true.

```csharp
public bool NamesAreCaseSensitive { get; set; }
```
