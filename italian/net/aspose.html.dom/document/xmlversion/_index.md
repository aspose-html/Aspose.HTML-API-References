## Document.XmlVersion property

Un attributo che specifica, come parte della dichiarazione XML, il numero di versione di questo documento. Se non c'è alcuna dichiarazione e se questo documento supporta la funzione "XML", il valore è "1.0". Se questo documento non supporta la funzionalità "XML", il valore è sempre null.

```csharp
public string XmlVersion { get; set; }
```

### Valore della proprietà

La versione XML.
