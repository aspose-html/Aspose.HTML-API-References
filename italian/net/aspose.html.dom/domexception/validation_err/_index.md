## DOMException.VALIDATION_ERR field

Se una chiamata a un metodo come insertBefore o removeChild rendesse il Node non valido rispetto alla "validità parziale", questa eccezione verrebbe sollevata e l'operazione non verrebbe eseguita. Questo codice è utilizzato in [Convalida DOM Livello 3]. Fare riferimento a questa specifica per ulteriori informazioni.

```csharp
public const ushort VALIDATION_ERR;
```
