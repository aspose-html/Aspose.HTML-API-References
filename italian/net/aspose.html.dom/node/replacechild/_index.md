## Node.ReplaceChild method

Sostituisce il nodo figlio oldChild con newChild nell'elenco dei figli e restituisce il nodo oldChild. Se newChild è un oggetto DocumentFragment, oldChild viene sostituito da tutti i figli DocumentFragment, che vengono inseriti nello stesso ordine. Se il newChild è già nell'albero, viene prima rimosso.

```csharp
public Node ReplaceChild(Node node, Node child)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| node | Node | Il nuovo nodo. |
| child | Node | Il vecchio bambino. |

### Valore di ritorno

Restituisce il nodo
