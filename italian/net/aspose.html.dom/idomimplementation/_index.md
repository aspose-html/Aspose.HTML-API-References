## IDOMImplementation interface

L'interfaccia DOMImplementation fornisce una serie di metodi per eseguire operazioni che sono indipendenti da qualsiasi istanza particolare del modello a oggetti del documento.

```csharp
public interface IDOMImplementation
```

## Metodi

| Nome | Descrizione |
| --- | --- |
| [CreateDocument](../../aspose.html.dom/idomimplementation/createdocument/)(string, string, DocumentType) | Crea un oggetto Documento DOM del tipo specificato con il suo elemento documento. |
| [CreateDocumentType](../../aspose.html.dom/idomimplementation/createdocumenttype/)(string, string, string) | Crea un nodo DocumentType vuoto. Le dichiarazioni di entità e le notazioni non sono rese disponibili. Le espansioni dei riferimenti alle entità e le aggiunte agli attributi predefiniti non si verificano. |
| [CreateHTMLDocument](../../aspose.html.dom/idomimplementation/createhtmldocument/)(string) | Restituisce un documento, con un albero base già costruito che include un elemento title, a meno che l'argomento title non sia omesso. |
| [HasFeature](../../aspose.html.dom/idomimplementation/hasfeature/)() | Verifica se l'implementazione DOM implementa una funzionalità e una versione specifiche, come specificato in Funzionalità DOM. |
