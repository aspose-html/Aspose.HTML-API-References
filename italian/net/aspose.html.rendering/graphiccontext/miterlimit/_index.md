## GraphicContext.MiterLimit property

Imposta o ottiene la lunghezza massima delle giunzioni delle linee squadrate per i percorsi tracciati. Questo parametro limita la lunghezza delle "punte" prodotte quando i segmenti di linea si uniscono ad angoli acuti.

```csharp
public virtual float MiterLimit { get; set; }
```
