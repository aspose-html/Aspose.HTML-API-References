Lo spazio dei nomi fornisce l'accesso a numerosi oggetti (elementi) che sono responsabili dell'interattività dell'utente all'interno del modulo come componente della pagina web.

## Classi

| Classe | Descrizione |
| --- | --- |
| [ButtonElement](./buttonelement/) | ButtonElement rappresenta un wrapper associato a HTMLButtonElement. |
| [DataListElement](./datalistelement/) | DataListElement rappresenta un wrapper associato a HTMLDataListElement |
| [FormEditor](./formeditor/) | Questa classe rappresenta l'editor rispetto a[`HTMLFormElement`](../aspose.html/htmlformelement/) che crea un modo più semplice per gli sviluppatori .net di modificare i moduli html. |
| [FormElement](./formelement/) | Rappresenta la classe base per gli elementi del modulo. |
| [FormElement&lt;T&gt;](./formelement-1/) | Il[`FormElement`](../aspose.html.forms/formelement/) rappresentano il campo generico |
| [FormSubmitter](./formsubmitter/) | Questa classe permette di preparare specificato[`HTMLFormElement`](../aspose.html/htmlformelement/) , raccoglie i valori dall'elemento del modulo, li invia al server remoto e riceve una risposta. |
| [InputElement](./inputelement/) | InputElement rappresenta un wrapper associato a HTMLInputElement. |
| [OptionCollection](./optioncollection/) | OptionElements rappresenta un wrapper associato a IHTMLOptionsCollection |
| [OptionElement](./optionelement/) | OptionElement rappresenta un wrapper associato a HTMLOptionElement |
| [SelectElement](./selectelement/) | SelectElement rappresenta un wrapper associato a HTMLSelectElement |
| [SubmissionResult](./submissionresult/) | Questa classe rappresenta il risultato dell'invio dei dati del modulo al server. |
| [TextAreaElement](./textareaelement/) | TextAreaElement rappresenta un wrapper associato a HTMLTextAreaElement |
## Enumerazione

| Enumerazione | Descrizione |
| --- | --- |
| [ButtonType](./buttontype/) | Questa enumerazione rappresenta gli stati del pulsante. |
| [FormElementType](./formelementtype/) | Rappresenta un'enumerazione dei tipi di elementi del modulo e il loro corrispondente agli elementi HTML |
| [InputElementType](./inputelementtype/) | Rappresenta gli stati del campo di input. |
