## OptionElement.Selected property

Rappresenta lo stato corrente del controllo del modulo corrispondente, in un agente utente interattivo. La modifica di questo attributo modifica lo stato del controllo del modulo, ma non modifica il valore dell'attributo HTML selezionato dell'elemento.

```csharp
public bool Selected { get; set; }
```
