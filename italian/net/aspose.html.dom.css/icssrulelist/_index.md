## ICSSRuleList interface

L'interfaccia CSSRuleList fornisce l'astrazione di una raccolta ordinata di regole CSS.

```csharp
public interface ICSSRuleList : IEnumerable<ICSSRule>
```

## Proprietà

| Nome | Descrizione |
| --- | --- |
| [Item](../../aspose.html.dom.css/icssrulelist/item/) { get; } | Utilizzato per recuperare una regola CSS tramite il metodo item() (http://www.w3.org/TR/DOM-Level-2-Style/css.html#CSS-CSSRuleList). L'ordine in questa raccolta rappresenta l'ordine delle regole nel foglio di stile CSS. Se index è maggiore o uguale al numero di regole nell'elenco, restituisce null. |
| [Length](../../aspose.html.dom.css/icssrulelist/length/) { get; } | Il numero di CSSRules nell'elenco. L'intervallo di indici di regole figlie valide va da 0 a length-1 incluso. |
