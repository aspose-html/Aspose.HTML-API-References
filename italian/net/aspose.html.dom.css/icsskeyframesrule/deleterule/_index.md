## ICSSKeyframesRule.DeleteRule method

Il metodo deleteRule elimina CSSKeyframeRule con la chiave passata. Se una regola con questa chiave non esiste, il metodo non fa nulla

```csharp
public void DeleteRule(string key)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| key | String | La chiave che descrive la regola da eliminare. La chiave deve risolversi in un numero compreso tra 0 e 1, altrimenti la regola viene ignorata |
