## ICSSStyleDeclaration.CSSText property

La rappresentazione testuale analizzabile del blocco di dichiarazione (escluse le parentesi graffe circostanti). L'impostazione di questo attributo comporterà l'analisi del nuovo valore e il ripristino di tutte le proprietà nel blocco di dichiarazione, inclusa la rimozione o l'aggiunta di proprietà.

```csharp
public string CSSText { get; set; }
```

### Valore della proprietà

Il testo CSS.
