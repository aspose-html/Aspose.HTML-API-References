## IDocumentCSS.GetOverrideStyle method

Questo metodo viene utilizzato per recuperare la dichiarazione di stile override per un elemento specificato e uno pseudo-elemento specificato.

```csharp
public ICSSStyleDeclaration GetOverrideStyle(Element elt, string pseudoElt)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| elt | Element | L'elemento il cui stile deve essere modificato. Questo parametro non può essere nullo. |
| pseudoElt | String | Lo pseudo-elemento o null se nessuno. |

### Valore di ritorno

La dichiarazione di stile override
