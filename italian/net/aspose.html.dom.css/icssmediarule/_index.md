## ICSSMediaRule interface

L'interfaccia CSSMediaRule rappresenta una regola @media in un foglio di stile CSS. Una regola @media può essere utilizzata per delimitare le regole di stile per tipi di media specifici.

```csharp
public interface ICSSMediaRule : ICSSRule
```

## Proprietà

| Nome | Descrizione |
| --- | --- |
| [CSSRules](../../aspose.html.dom.css/icssmediarule/cssrules/) { get; } | Un elenco di tutte le regole CSS contenute nel blocco multimediale. |
| [Media](../../aspose.html.dom.css/icssmediarule/media/) { get; } | Un elenco di tipi di media per questa regola. |
