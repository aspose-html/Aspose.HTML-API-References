## SVGUnitTypes enumeration

L'enumerazione SVGUnitTypes definisce un set di costanti di uso comune ed è un'interfaccia di base utilizzata da SVGGradientElement, SVGPatternElement, SVGClipPathElement, SVGMaskElement e SVGFilterElement.

```csharp
public enum SVGUnitTypes
```

| Nome | Valore | Descrizione |
| --- | --- | --- |
| SVG_UNIT_TYPE_UNKNOWN | `0` | Il tipo non è uno dei tipi predefiniti. Non è valido tentare di definire un nuovo valore di questo tipo o tentare di convertire un valore esistente in questo tipo. |
| SVG_UNIT_TYPE_USERSPACEONUSE | `1` | Corrisponde al valore 'userSpaceOnUse'. |
| SVG_UNIT_TYPE_OBJECTBOUNDINGBOX | `2` | Corrisponde al valore 'objectBoundingBox'. |
