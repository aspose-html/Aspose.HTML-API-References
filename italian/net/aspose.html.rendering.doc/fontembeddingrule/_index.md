## FontEmbeddingRule enumeration

Rappresenta le regole di incorporamento dei caratteri.

```csharp
public enum FontEmbeddingRule
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Full | `1` | Tutti i caratteri utilizzati verranno incorporati così come sono. Tieni presente che alcuni font potrebbero essere concessi in licenza, questo renderà il documento risultante non modificabile. Anche i file dei caratteri incorporati potrebbero aumentare notevolmente le dimensioni del documento risultante. |
| None | `2` | I caratteri non verranno incorporati. |
