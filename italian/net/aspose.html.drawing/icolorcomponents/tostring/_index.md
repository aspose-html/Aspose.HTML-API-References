## IColorComponents.ToString method

Converte i componenti del colore in una rappresentazione di stringa.

```csharp
public string ToString(bool includeAlpha, bool asDecimal, int digits)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| includeAlpha | Boolean | Specifica se aggiungere il componente Alpha. |
| asDecimal | Boolean | Specifica se i componenti del colore vengono mantenuti come numero decimale o come percentuale. |
| digits | Int32 | Imposta la precisione di arrotondamento per i componenti colore. |

### Valore di ritorno

Rappresentazione in forma di stringa dei componenti del colore.
