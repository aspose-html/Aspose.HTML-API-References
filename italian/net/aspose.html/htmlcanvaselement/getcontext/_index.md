## HTMLCanvasElement.GetContext method

Restituisce un contesto di disegno nell'area di disegno o null se l'ID contesto non è supportato. Un contesto di disegno ti consente di disegnare sulla tela.

```csharp
public object GetContext(string contextId)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| contextId | String | È una stringa contenente l'identificatore di contesto che definisce il contesto del disegno associato all'area di disegno |

### Valore di ritorno

Un contesto di rendering
