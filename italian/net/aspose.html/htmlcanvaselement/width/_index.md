## HTMLCanvasElement.Width property

È un numero intero positivo che riflette l'attributo HTML width dell'elemento canvas interpretato in pixel CSS. Quando l'attributo non è specificato o se è impostato su un valore non valido, come un valore negativo, viene utilizzato il valore predefinito 300.

```csharp
public ulong Width { get; set; }
```
