## ToDataURL() {#todataurl}

restituisce un URI di dati contenente una rappresentazione dell'immagine nel formato specificato dal parametro type (il valore predefinito è PNG). L'immagine restituita ha una risoluzione di 96 dpi.

```csharp
public string ToDataURL()
```

### Valore di ritorno

Una DOMString contenente l'URI dei dati richiesti.

## ToDataURL(string, double) {#todataurl_2}

restituisce un URI di dati contenente una rappresentazione dell'immagine nel formato specificato dal parametro type (il valore predefinito è PNG). L'immagine restituita ha una risoluzione di 96 dpi.

```csharp
public string ToDataURL(string type, double encoderOptions)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| type | String | Una stringa che indica il formato dell'immagine. Il tipo di formato predefinito è image/png. |
| encoderOptions | Double | Un numero compreso tra 0 e 1 che indica la qualità dell'immagine se il tipo richiesto è image/jpeg o image/webp. |

Una DOMString contenente l'URI dei dati richiesti.
