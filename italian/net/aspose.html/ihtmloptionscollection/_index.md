## IHTMLOptionsCollection interface

An`HTMLOptionsCollection` è un elenco di nodi che rappresentano l'elemento opzione HTML . È possibile accedere a un singolo nodo tramite l'indice ordinale o quello del nodo`nome` O`id` attributi. Si presuppone che le raccolte nel DOM HTML siano attive, il che significa che vengono aggiornate automaticamente quando il documento sottostante viene modificato.

Vedi anche il[Document object Model (DOM) Specifica HTML di livello 2](http://www.w3.org/TR/2003/REC-DOM-Level-2-HTML-20030109). @since DOM Level 2
