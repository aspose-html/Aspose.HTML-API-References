## HTMLDocument.Anchors property

Una raccolta di tutte le ancore (`UN` ) elementi in un documento con un valore per the`nome`attributo. Per motivi di compatibilità con le versioni precedenti, il set di ancore restituito contiene solo quelle ancore create con il`nome` attributo, non quelli creati con l'attributo`id` attributo. Si noti che in [[XHTML 1.0](http://www.w3.org/TR/2002/REC-xhtml1-20020801) ], il `nome` attributo (vedere la sezione 4.10) non ha semantica e è presente solo per i programmi utente legacy: il`id` viene invece utilizzato l'attributo . Gli utenti dovrebbero preferire i meccanismi iteratori forniti da [[DOM Livello 2 Attraversamento](http://www.w3.org/TR/2000/REC-DOM-Level-2-Traversal-Range-20001113) ] Invece.

public HTMLCollection Anchors { get; }
