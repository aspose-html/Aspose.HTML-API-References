## HTMLDocument.Referrer property

Restituisce l'URI [[RFC 2396 dell'IETF](http://www.ietf.org/rfc/rfc2396.txt) della pagina collegata a questa pagina. Il valore è una stringa vuota se l'utente è passato direttamente alla pagina (non tramite un collegamento, ma, ad esempio, tramite un segnalibro).

```csharp
public string Referrer { get; }
```
