## HTMLTextAreaElement.Value property

Rappresenta il contenuto corrente del controllo del modulo corrispondente, in un agente utente interattivo. La modifica di questo attributo modifica il contenuto di del controllo del modulo, ma non modifica il contenuto dell'elemento . Se la totalità dei dati non può rientrare in un singolo `DOMStringa` , l'implementazione potrebbe troncare i dati.

```csharp
public string Value { get; set; }
```
