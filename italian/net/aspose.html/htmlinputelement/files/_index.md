## HTMLInputElement.Files property

L'attributo files IDL consente agli script di accedere ai file selezionati dell'elemento. Al momento dell'acquisizione, se l'attributo IDL si applica, deve restituire un oggetto FileList che rappresenta i file attualmente selezionati. Lo stesso oggetto deve essere restituito finché l'elenco dei file selezionati non cambia. Se l'attributo IDL non si applica, deve invece restituire null. [FILEAPI]

```csharp
public IFileList Files { get; }
```
