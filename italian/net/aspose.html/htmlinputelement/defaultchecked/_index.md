## HTMLInputElement.DefaultChecked property

Quando`tipo` ha il valore "radio" o "checkbox", questo rappresenta l'attributo HTML verificato dell'elemento. Il valore di questo attributo non cambia se lo stato del corrispondente modulo control, in un agente utente interattivo, cambia. Vedi la definizione dell'attributo verificato in HTML 4.01.

```csharp
public bool DefaultChecked { get; set; }
```
