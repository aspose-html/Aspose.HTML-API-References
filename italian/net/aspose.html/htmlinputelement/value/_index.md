## HTMLInputElement.Value property

Quando il`tipo` l'attributo dell'elemento ha il valore "testo", "file" o "password", questo rappresenta il contenuto corrente di il controllo del modulo corrispondente, in un programma utente interattivo. La modifica di questo attributo modifica il contenuto del controllo del modulo, ma non modifica il valore dell'attributo HTML value dell'elemento. Quando il`tipo`l'attributo dell'elemento ha il valore "button", "hidden", "submit", "reset", "image", "checkbox" o "radio", questo rappresenta l'attributo del valore HTML dell'elemento. Vedi la definizione dell'attributo value in HTML 4.01.

```csharp
public string Value { get; set; }
```
