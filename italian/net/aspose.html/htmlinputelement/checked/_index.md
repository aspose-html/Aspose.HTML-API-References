## HTMLInputElement.Checked property

Quando il`tipo`l'attributo dell'elemento ha il valore "radio" o "checkbox", questo rappresenta lo stato corrente del modulo control, in un programma utente interattivo. Le modifiche a questo attributo modificano lo stato del controllo del modulo, ma non modificano il valore di l'attributo HTML verificato dell'elemento INPUT. Durante la gestione di un evento clic su un elemento di input con un attributo di tipo che ha il value "radio" o "checkbox", alcune implementazioni potrebbero modificare il valore di questa proprietà prima che l'evento venga inviato nel documento . Se l'azione predefinita dell'evento viene annullata, il valore della proprietà può essere riportato al suo valore originale. Ciò significa che il valore di questa proprietà durante la gestione degli eventi clic dipende dall'implementazione.

public bool Checked { get; set; }
