## InputEvent.Data property

I dati contengono il valore dei caratteri generati da un metodo di input. Questo POTREBBE essere un singolo carattere Unicode o una sequenza non vuota di caratteri Unicode [Unicode]. I caratteri DOVREBBERO essere normalizzati come definito dalla forma di normalizzazione Unicode NFC, definita in [UAX15]. Questo attributo PUÒ contenere la stringa vuota.

```csharp
public string Data { get; }
```
