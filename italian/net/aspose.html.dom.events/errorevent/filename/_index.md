## ErrorEvent.FileName property

L'attributo filename deve restituire il valore con cui è stato inizializzato. Quando l'oggetto viene creato, questo attributo deve essere inizializzato sulla stringa vuota. Rappresenta l'URL assoluto dello script in cui si è verificato originariamente l'errore.

```csharp
public string FileName { get; }
```

### Valore della proprietà

Il nome del file.
