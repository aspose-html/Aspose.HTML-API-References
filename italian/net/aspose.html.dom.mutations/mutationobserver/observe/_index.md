## Observe(Node) {#observe}

Indica all'agente utente di osservare un determinato target (un nodo) e di segnalare eventuali mutazioni in base ai criteri forniti dalle opzioni (un oggetto). L'argomento options consente di impostare le opzioni di osservazione delle mutazioni tramite i membri dell'oggetto.

```csharp
public void Observe(Node target)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| target | Node | L'obiettivo per osservare. |
