## SVGAngle.ValueInSpecifiedUnits property

Il valore dell'angolo come valore in virgola mobile, nelle unità espresse da unitType. L'impostazione di questo attributo causerà l'aggiornamento automatico di value e valueAsString per riflettere questa impostazione.

```csharp
public float ValueInSpecifiedUnits { get; set; }
```

### Valore della proprietà

Il valore nelle unità specificate.
