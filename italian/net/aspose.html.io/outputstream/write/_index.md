## OutputStream.Write method

Scrive una sequenza di byte nel flusso output avvolto e fa avanzare la posizione corrente all'interno di questo flusso del numero di byte scritti.

```csharp
public override void Write(byte[] buffer, int offset, int count)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| buffer | Byte[] | Una matrice di byte. Questo metodo copia count byte dal buffer al flusso output avvolto. |
| offset | Int32 | L'offset di byte in base zero nel buffer in corrispondenza del quale iniziare a copiare i byte nel flusso output sottoposto a wrapping. |
| count | Int32 | Il numero di byte da scrivere nel flusso di output di cui è stato eseguito il wrapping. |
