## IRuntimeService.JavaScriptTimeout property

Ottiene o impostaTimeSpanche limita il tempo di esecuzione di JavaScript. Se lo script viene eseguito più a lungo di quanto previstoTimeSpan , verrà annullato. È possibile specificare un timeout infinito impostandoTimeSpan pari a -1 millisecondo. Il valore predefinito è 1 minuto.

```csharp
public TimeSpan JavaScriptTimeout { get; set; }
```
