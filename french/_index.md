## Aspose.HTML pour .NET

{{% alert color="primary" %}} 

![Logo du produit Aspose.HTML pour .NET](../home_1.png)


Aspose.HTML pour .NET est une bibliothèque de classes multiplateforme qui permet à vos applications d'effectuer un large éventail de tâches de manipulation HTML. Étant donné que HTML est généralement utilisé en combinaison avec d'autres technologies, Aspose.HTML prend en charge non seulement HTML5, mais également les spécifications CSS3, SVG et HTML Canvas. À l'aide d'Aspose.HTML, vous pouvez charger des pages Web, analyser et modifier le contenu, enregistrer et convertir des pages vers et depuis les formats suivants : HTML, XHTML, MHTML, SVG, Markdown, PDF, XPS,
EPUB, PNG, TIFF, JPEG, BMP et autres formats.

## Aspose.HTML pour Java

{{% alert color="primary" %}} 

![Logo du produit Aspose.HTML pour Java](../home_2.png)


Aspose.HTML pour Java est une API de manipulation HTML avancée conçue pour effectuer un large éventail de tâches de manipulation HTML directement dans les applications Java. L'API fournit également un moteur de rendu haute fidélité pour les formats à mise en page fixe tels que PDF et XPS, et un certain nombre de formats d'image raster.
