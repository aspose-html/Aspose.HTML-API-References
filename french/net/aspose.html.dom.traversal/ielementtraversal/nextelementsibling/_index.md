## IElementTraversal.NextElementSibling property

Renvoie le prochain nœud d'élément frère de cet élément. null si cet élément n'a aucun nœud frère d'élément qui vient après celui-ci dans l'arborescence du document.

```csharp
public Element NextElementSibling { get; }
```

### Valeur de la propriété
