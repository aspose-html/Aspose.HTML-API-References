## FontEmbeddingRule enumeration

Représente les règles d'incorporation des polices.

```csharp
public enum FontEmbeddingRule
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Full | `1` | Toutes les polices utilisées seront intégrées telles quelles. Sachez que certaines polices peuvent être sous licence, cela rendra le document résultant non modifiable. De plus, les fichiers de polices intégrés pourraient augmenter considérablement la taille du document résultant. |
| None | `2` | Les polices ne seront pas intégrées. |
