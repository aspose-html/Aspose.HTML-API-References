## HTMLCanvasElement.Width property

Est un entier positif reflétant l'attribut HTML width de l'élément canvas interprété en pixels CSS. Lorsque l'attribut n'est pas spécifié, ou s'il est défini sur une valeur non valide, comme un négatif, la valeur par défaut de 300 est utilisée.
