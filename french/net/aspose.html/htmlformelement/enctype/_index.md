## HTMLFormElement.Enctype property

Le type de contenu du formulaire soumis, généralement "application/x-www-form-urlencoded". Voir la définition de l'attribut enctype dans HTML 4.01. Il n'est pas garanti que le gestionnaire onsubmit even soit déclenché lors de l'appel de cette méthode. Le comportement est incohérent pour des raisons historiques et les auteurs ne doivent pas s'appuyer sur un particulier.

public string Enctype { get; set; }
