## HTMLInputElement.Files property

L'attribut IDL des fichiers permet aux scripts d'accéder aux fichiers sélectionnés de l'élément. Lors de l'obtention, si l'attribut IDL s'applique, il doit renvoyer un objet FileList qui représente les fichiers actuellement sélectionnés. Le même objet doit être renvoyé jusqu'à ce que la liste des fichiers sélectionnés change. Si l'attribut IDL ne s'applique pas, il doit à la place renvoyer null. [FILEAPI]

```csharp
public IFileList Files { get; }
```
