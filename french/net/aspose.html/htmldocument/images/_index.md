## HTMLDocument.Images property

Une collection de tous les`IMG` éléments dans un document. Le comportement est limité à`IMG` éléments pour la rétrocompatibilité . Comme suggéré par [[HTML 4.01](http://www.w3.org/TR/1999/REC-html401-19991224)], pour inclure des images, les auteurs peuvent utiliser le`OBJET` élément ou le`IMG` élément. Par conséquent, il est recommandé de ne pas utiliser cet attribut pour rechercher les images dans le document mais`getElementsByTagName` avec HTML 4.01 ou`getElementsByTagNameNS` avec XHTML 1.0.

```csharp
public HTMLCollection Images { get; }
```
