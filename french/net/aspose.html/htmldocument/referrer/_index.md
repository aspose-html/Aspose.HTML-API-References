## HTMLDocument.Referrer property

Renvoie l'URI [[IETF RFC 2396](http://www.ietf.org/rfc/rfc2396.txt) de la page liée à cette page. La valeur est une chaîne vide si l'utilisateur a accédé directement à la page (pas via un lien, mais, par exemple, via un signet).

public string Referrer { get; }
