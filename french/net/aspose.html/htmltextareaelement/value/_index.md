## HTMLTextAreaElement.Value property

Représente le contenu actuel du contrôle de formulaire correspondant, dans un agent utilisateur interactif. La modification de cet attribut modifie le contenu du contrôle de formulaire, mais ne modifie pas le contenu de l'élément . Si l'intégralité des données ne peut pas tenir dans un seul `Chaîne DOM` , l'implémentation peut tronquer les données.

```csharp
public string Value { get; set; }
```
