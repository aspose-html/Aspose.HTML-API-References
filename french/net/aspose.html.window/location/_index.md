## Location class

Les objets Location fournissent une représentation de l'adresse du document actif du contexte de navigation de leur Document, et permettent de modifier l'entrée courante de l'historique de session du contexte de navigation, en ajoutant ou remplaçant des entrées dans l'objet historique.

```csharp
public sealed class Location : Url
```
