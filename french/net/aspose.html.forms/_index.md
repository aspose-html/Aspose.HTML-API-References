L'espace de noms donne accès à de nombreux objets (éléments) qui sont responsables de l'interactivité de l'utilisateur dans le formulaire en tant que composant de page Web.

## Des classes

| Classer | La description |
| --- | --- |
| [ButtonElement](./buttonelement/) | Le ButtonElement représente un wrapper associé au HTMLButtonElement. |
| [DataListElement](./datalistelement/) | Le DataListElement représente un wrapper associé au HTMLDataListElement |
| [FormEditor](./formeditor/) | Cette classe représente l'éditeur sur le[`HTMLFormElement`](../aspose.html/htmlformelement/) qui crée un moyen plus simple pour les développeurs .net de modifier les formulaires html. |
| [FormElement](./formelement/) | Représente la classe de base pour les éléments de formulaire. |
| [FormElement&lt;T&gt;](./formelement-1/) | Le[`FormElement`](../aspose.html.forms/formelement/) représentent le champ générique |
| [FormSubmitter](./formsubmitter/) | Cette classe permet de préparer des[`HTMLFormElement`](../aspose.html/htmlformelement/) , collecte les valeurs de l'élément de formulaire, les soumet au serveur distant et reçoit une réponse. |
| [InputElement](./inputelement/) | L'InputElement représente un wrapper associé à l'HTMLInputElement. |
| [OptionCollection](./optioncollection/) | Les OptionElements représentent un wrapper associé à IHTMLOptionsCollection |
| [OptionElement](./optionelement/) | L'OptionElement représente un wrapper associé à l'HTMLOptionElement |
| [SelectElement](./selectelement/) | Le SelectElement représente un wrapper associé au HTMLSelectElement |
| [SubmissionResult](./submissionresult/) | Cette classe représente le résultat de la soumission des données du formulaire au serveur. |
| [TextAreaElement](./textareaelement/) | Le TextAreaElement représente un wrapper associé au HTMLTextAreaElement |
## Énumération

| [InputElementType](./inputelementtype/) | Représente les états du champ de saisie. |
