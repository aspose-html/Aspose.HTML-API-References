## Ellipse(double, double, double, double, double, double, double) {#ellipse}

Ajoute une ellipse au chemin qui est centrée à la position (x, y) avec les rayons radiusX et radiusY commençant à startAngle et se terminant à endAngle allant dans la direction donnée dans le sens inverse des aiguilles d'une montre (par défaut dans le sens des aiguilles d'une montre).

```csharp
public void Ellipse(double x, double y, double radiusX, double radiusY, double rotation, 
    double startAngle, double endAngle)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| x | Double | L'axe x de la coordonnée du centre de l'ellipse. |
| y | Double | L'axe y de la coordonnée du centre de l'ellipse. |
| radiusX | Double | Le rayon du grand axe de l'ellipse. |
| radiusY | Double | Le rayon du petit axe de l'ellipse. |
| rotation | Double | La rotation de cette ellipse, exprimée en radians. |
| startAngle | Double | Le point de départ, mesuré à partir de l'axe x, à partir duquel il sera tiré, exprimé en radians. |
| endAngle | Double | L'angle de l'ellipse finale vers lequel elle sera dessinée, exprimé en radians. |

## Ellipse(double, double, double, double, double, double, double, bool) {#ellipse_1}

Ajoute une ellipse au chemin qui est centrée à la position (x, y) avec les rayons radiusX et radiusY commençant à startAngle et se terminant à endAngle allant dans la direction donnée dans le sens inverse des aiguilles d'une montre (par défaut dans le sens des aiguilles d'une montre).

```csharp
public void Ellipse(double x, double y, double radiusX, double radiusY, double rotation, 
    double startAngle, double endAngle, bool anticlockwise)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| x | Double | L'axe x de la coordonnée du centre de l'ellipse. |
| y | Double | L'axe y de la coordonnée du centre de l'ellipse. |

| radiusY | Double | Le rayon du petit axe de l'ellipse. |
| rotation | Double | La rotation de cette ellipse, exprimée en radians. |
