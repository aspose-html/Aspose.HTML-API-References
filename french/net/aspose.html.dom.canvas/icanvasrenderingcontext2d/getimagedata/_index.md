## ICanvasRenderingContext2D.GetImageData method

Renvoie un objet ImageData représentant les données de pixel sous-jacentes pour la zone du canevas désignée par le rectangle qui commence à (sx, sy) et a une largeur sw et une hauteur sh. Cette méthode n'est pas affectée par la matrice de transformation du canevas.

```csharp
public IImageData GetImageData(double sx, double sy, double sw, double sh)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| sx | Double | Coordonnée x du coin supérieur gauche du rectangle à partir duquel ImageData sera extrait. |
| sy | Double | La coordonnée y du coin supérieur gauche du rectangle à partir duquel ImageData sera extrait. |
| sw | Double | La largeur du rectangle à partir duquel ImageData sera extrait. |
| sh | Double | La hauteur du rectangle à partir duquel ImageData sera extrait. |

### Return_Value

Un objet ImageData contenant les données d'image pour le rectangle donné du canevas.
