## ICanvasRenderingContext2D.CreateRadialGradient method

Crée un gradient radial donné par les coordonnées des deux cercles représentés par les paramètres.

```csharp
public ICanvasGradient CreateRadialGradient(double x0, double y0, double r0, double x1, double y1, 
    double r1)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| x0 | Double | L'axe x de la coordonnée du cercle de départ. |
| y0 | Double | L'axe y de la coordonnée du cercle de départ |
| r0 | Double | Le rayon du cercle de départ. |
| x1 | Double | L'axe x de la coordonnée du cercle de fin. |
| y1 | Double | L'axe y de la coordonnée du cercle final. |
| r1 | Double | Le rayon du cercle final. |

### Return_Value

Un CanvasGradient radial initialisé avec les deux cercles spécifiés.
