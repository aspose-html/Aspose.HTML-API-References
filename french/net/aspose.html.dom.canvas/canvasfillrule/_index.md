## CanvasFillRule enumeration

Cette énumération est utilisée pour sélectionner l'algorithme de règle de remplissage permettant de déterminer si un point se trouve à l'intérieur ou à l'extérieur d'un chemin.

```csharp
public enum CanvasFillRule
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| NonZero | `0` | La valeur "non nulle" indique la règle d'enroulement non nulle, dans laquelle un point est considéré comme étant à l'extérieur d'une forme si le nombre de fois qu'une ligne droite semi-infinie tracée à partir de ce point croise le chemin de la forme dans une direction est égal au nombre de fois qu'il croise le chemin allant dans l'autre sens. |
| EvenOdd | `1` | La valeur "evenodd" indique la règle paire-impaire, dans laquelle un point est considéré comme étant à l'extérieur d'une forme si le nombre de fois qu'une ligne droite semi-infinie tracée à partir de ce point croise le chemin de la forme est pair. |
