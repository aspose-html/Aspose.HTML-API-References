## IDocumentStyle interface

L'interface DocumentStyle fournit un mécanisme par lequel les feuilles de style incorporées dans un document peuvent être récupérées. L'attente est qu'une instance de l'interface DocumentStyle peut être obtenue en utilisant des méthodes de transtypage spécifiques à la liaison sur une instance de l'interface Document.
