## Rect class

L'interface Rect est utilisée pour représenter n'importe quelle valeur rect. Cette interface reflète les valeurs de la propriété de style sous-jacente. Par conséquent, les modifications apportées aux objets CSSPrimitiveValue modifient la propriété de style.
