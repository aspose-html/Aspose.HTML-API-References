## ICSSMediaRule interface

L'interface CSSMediaRule représente une règle @media dans une feuille de style CSS. Une règle @media peut être utilisée pour délimiter les règles de style pour des types de médias spécifiques.

```csharp
public interface ICSSMediaRule : ICSSRule
```

| [Media](../../aspose.html.dom.css/icssmediarule/media/) { get; } | Une liste des types de médias pour cette règle. |
