## PageSetup.AdjustToWidestPage property

Obtient ou définit l'indicateur qui détermine le cas où la taille de la page sera ajustée à la page la plus large du document. Cette option prend du temps, de sorte que le temps de traitement du document peut être multiplié par deux. L'ajustement n'aura lieu que si la page la plus large du document est plus large que la taille de page déterminée dans[`PageSetup`](../) . La taille de page ajustée sera utilisée pour toutes les pages du document.

```csharp
public bool AdjustToWidestPage { get; set; }
```
