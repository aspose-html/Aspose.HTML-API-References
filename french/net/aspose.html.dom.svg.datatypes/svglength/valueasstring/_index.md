## SVGLength.ValueAsString property

La valeur sous forme de chaîne, dans les unités exprimées par unitType. La définition de cet attribut entraînera la mise à jour automatique de value, valueInSpecifiedUnits et unitType pour refléter ce paramètre.

```csharp
public string ValueAsString { get; set; }
```

### Valeur de la propriété

La valeur sous forme de chaîne.
