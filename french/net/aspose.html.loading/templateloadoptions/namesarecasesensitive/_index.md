## TemplateLoadOptions.NamesAreCaseSensitive property

Propriété qui spécifie si le modèle et les noms des éléments de données seront mis en correspondance indépendamment de la casse des caractères ou non. Si vrai, la correspondance est sensible à la casse ; si false, la casse des caractères sera ignorée. La valeur par défaut est true.

public bool NamesAreCaseSensitive { get; set; }
