## InputEvent.Data property

Veriler, bir giriş yöntemi tarafından oluşturulan karakterlerin değerini tutar. Bu, tek bir Unicode karakter veya boş olmayan bir Unicode karakter dizisi [Unicode] OLABİLİR. Karakterler, [UAX15]'te tanımlanan NFC Unicode normalleştirme formu tarafından tanımlandığı şekilde normalleştirilmelidir *ÖNERİ*. Bu öznitelik boş dizeyi İÇEREBİLİR.

```csharp
public string Data { get; }
```
