## SVGTransform.SetRotate method

```csharp
public void SetRotate(float angle, float cx, float cy)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| angle | Single | Dönme açısı. |
| cx | Single | Dönme merkezinin x koordinatı. |
| cy | Single | Dönme merkezinin y koordinatı. |
