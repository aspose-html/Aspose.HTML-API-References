## HTMLInputElement.Files property

files IDL özniteliği, betiklerin öğenin seçili dosyalarına erişmesine izin verir. Alırken, IDL özniteliği geçerliyse, geçerli seçili dosyaları temsil eden bir FileList nesnesi döndürmesi gerekir. Seçilen dosyaların listesi değişene kadar aynı nesne döndürülmelidir. IDL özniteliği geçerli değilse, bunun yerine boş değer döndürmesi gerekir. [FILEAPI]

```csharp
public IFileList Files { get; }
```
