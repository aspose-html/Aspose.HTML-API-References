## IBlob.Type property

Blob'un ortam türünü temsil eden küçük harfli ASCII kodlu dize. Alınırken, kullanıcı aracıları Blob türünü küçük harfli ASCII kodlu dize olarak döndürmelidir, öyle ki bir bayta dönüştürüldüğünde dizi, ayrıştırılabilir bir MIME türüdür, veya tür belirlenemezse boş dize – 0 bayt – olur.

public string Type { get; }
