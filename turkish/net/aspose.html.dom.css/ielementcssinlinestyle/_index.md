## IElementCSSInlineStyle interface

Öğelere iliştirilen satır içi stil bilgileri, style özniteliği aracılığıyla gösterilir. Bu, HTML öğeleri (veya STYLE özniteliğini aynı şekilde kullanan diğer şemalardaki veya DTD'lerdeki öğeler) için STYLE özniteliğinin içeriğini temsil eder.
