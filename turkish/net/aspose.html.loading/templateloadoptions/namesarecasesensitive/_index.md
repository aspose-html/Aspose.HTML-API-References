## TemplateLoadOptions.NamesAreCaseSensitive property

Karakter büyük/küçük durumuna bakılmaksızın şablon ve veri öğesi adlarının eşleştirilip eşleştirilmeyeceğini belirten özellik. True ise, eşleştirme büyük/küçük harfe duyarlıdır; false ise, büyük/küçük harf dikkate alınmaz. Varsayılan değer true'dur.

```csharp
public bool NamesAreCaseSensitive { get; set; }
```
