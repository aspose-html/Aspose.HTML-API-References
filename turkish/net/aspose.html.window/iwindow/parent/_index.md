## IWindow.Parent property

Gözatma bağlamında b bir Belgenin Window nesnesindeki üst IDL özniteliği, varsa üst gözatma bağlamının WindowProxy nesnesini (yani, b bir alt gözatma bağlamıysa) veya gözatmanın WindowProxy nesnesini döndürmelidir. bağlam b'nin kendisi, aksi takdirde (yani, üst düzey bir göz atma bağlamı veya ayrılmış, iç içe geçmiş bir göz atma bağlamıysa).

public IWindow Parent { get; }
