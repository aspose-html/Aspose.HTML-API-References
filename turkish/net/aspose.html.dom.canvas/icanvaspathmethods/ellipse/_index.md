## Ellipse(double, double, double, double, double, double, double) {#ellipse}

StartAngle 'de başlayıp endAngle'da biten, yarıçapX ve yarıçapı ile (x, y) konumunda ortalanmış yola bir elips ekler (varsayılan olarak saat yönündedir).

```csharp
public void Ellipse(double x, double y, double radiusX, double radiusY, double rotation, 
    double startAngle, double endAngle)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| x | Double | Elips merkezinin koordinatının x ekseni. |
| y | Double | Elips merkezinin koordinatının y ekseni. |
| radiusX | Double | Elipsin ana eksen yarıçapı. |
| radiusY | Double | Elipsin küçük eksen yarıçapı. |
| rotation | Double | Bu elipsin dönüşü, radyan cinsinden ifade edilir. |
| startAngle | Double | Radyan olarak ifade edilen, çizileceği x ekseninden ölçülen başlangıç noktası. |
| endAngle | Double | Uç elipsin çizileceği açı, radyan olarak ifade edilir. |
