## CreatePattern(HTMLImageElement, string) {#createpattern_1}

Belirtilen görüntüyü (bir CanvasImageSource) kullanarak bir desen oluşturur. Tekrarlama bağımsız değişkeni tarafından belirtilen yönlerde kaynağı tekrarlar.

```csharp
public ICanvasPattern CreatePattern(HTMLImageElement image, string repetition)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| image | HTMLImageElement | Yinelenecek görüntü olarak kullanılacak bir HTMLImageElement |
| repetition | String | Görüntünün nasıl tekrarlanacağını gösteren bir dize. |
