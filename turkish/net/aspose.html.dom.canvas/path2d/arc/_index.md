## Arc(double, double, double, double, double) {#arc}

StartAngle'da başlayıp endAngle'da biten r yarıçaplı (x, y) konumunda ortalanmış yola, saat yönünün tersine (varsayılanı saat yönündedir) giden bir yay ekler.

```csharp
public void Arc(double x, double y, double radius, double startAngle, double endAngle)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| x | Double | Yayın merkezinin x koordinatı. |
| y | Double | Yayın merkezinin y koordinatı. |
| radius | Double | Yayın yarıçapı. |
| startAngle | Double | Yayın başladığı açı, pozitif x ekseninden saat yönünde ölçülür ve radyan olarak ifade edilir. |
| endAngle | Double | Yayın bittiği açı, pozitif x ekseninden saat yönünde ölçülür ve radyan olarak ifade edilir. |
