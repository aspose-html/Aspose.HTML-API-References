## .NET için Aspose.HTML

{{% alert color="primary" %}} 

![Aspose.HTML for .NET Ürün Logosu](../home_1.png)


Aspose.HTML for .NET, uygulamalarınızın çok çeşitli HTML işleme görevlerini gerçekleştirmesini sağlayan platformlar arası bir sınıf kitaplığıdır. HTML genellikle diğer teknolojilerle birlikte kullanıldığından, Aspose.HTML yalnızca HTML5'i değil, CSS3, SVG ve HTML Canvas özelliklerini de destekler. Aspose.HTML'yi kullanarak web sayfalarını yükleyebilir, içeriği analiz edebilir ve düzenleyebilir, sayfaları kaydedebilir ve şu biçimlere dönüştürebilirsiniz: HTML, XHTML, MHTML, SVG, Markdown, PDF, XPS,
EPUB, PNG, TIFF, JPEG, BMP ve diğer formatlar.

{{% /alert %}}

- [Aspose.HTML for .NET API Referansı](/html/tr/net/)


## Java için Aspose.HTML

{{% alert color="primary" %}} 

![Aspose.HTML for Java Ürün Logosu](../home_2.png)


Aspose.HTML for Java, doğrudan Java uygulamaları içinde çok çeşitli HTML işleme görevlerini gerçekleştirmek için oluşturulmuş gelişmiş bir HTML işleme API'sidir. API ayrıca, PDF ve XPS gibi sabit mizanpajlı formatlar ve bir dizi raster görüntü formatı için yüksek doğrulukta bir işleme motoru sağlar.
