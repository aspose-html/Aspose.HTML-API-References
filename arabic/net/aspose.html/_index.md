ملف **Aspose.Html** تحتوي مساحة الاسم على فئات وطرق من أجل معالجة مستندات HTML ، سواء كانت بسيطة أو معقدة وسريعة. **Aspose.HTML**يسمح لـ .NET للمطورين بإدراج ، إزالة ، استبدال عقد HTML ، واستخراج معلومات نمط CSS ، التنقل عبر مستند HTML إما عن طريق NodeIterator ، TreeWalker التي يتم توفيرها بواسطة مواصفات اجتياز ، XPath أو استعلامات محدد CSS. كما أنه يوفر البرمجة النصية التي تسمح لـ بمعالجة HTML DOM عبر JavaScript. بالإضافة إلى HTML ، توفر واجهة برمجة التطبيقات هذه أيضًا الإمكانيات لتحميل EPUB و MHTML. تشتهر واجهات برمجة التطبيقات Aspose API بميزات تحويل تنسيق الملفات بين وتوفر واجهة برمجة التطبيقات هذه أيضًا إمكانيات تحميل ملف HTML وتقديم الإخراج في تنسيقات PDF و XPS وتنسيقات الصور النقطية بما في ذلك JPEG و PNG و BMP و TIFF.
