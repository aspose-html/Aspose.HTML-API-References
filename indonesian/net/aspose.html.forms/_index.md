Namespace menyediakan akses ke banyak objek (elemen) yang bertanggung jawab untuk interaktivitas pengguna dalam form sebagai komponen halaman web.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [ButtonElement](./buttonelement/) | ButtonElement mewakili pembungkus yang terkait dengan HTMLButtonElement. |
| [DataListElement](./datalistelement/) | DataListElement mewakili pembungkus yang terkait dengan HTMLDataListElement |
| [FormEditor](./formeditor/) | Kelas ini mewakili editor di atas[`HTMLFormElement`](../aspose.html/htmlformelement/) yang menciptakan cara yang lebih mudah bagi pengembang .net untuk mengedit formulir html. |
| [FormElement](./formelement/) | Mewakili kelas dasar untuk elemen formulir. |
| [FormElement&lt;T&gt;](./formelement-1/) | Itu[`FormElement`](../aspose.html.forms/formelement/) mewakili bidang umum |
| [FormSubmitter](./formsubmitter/) | Kelas ini memungkinkan untuk mempersiapkan ditentukan[`HTMLFormElement`](../aspose.html/htmlformelement/) , mengumpulkan nilai dari elemen formulir, mengirimkannya ke server jarak jauh dan menerima respons. |
| [InputElement](./inputelement/) | InputElement mewakili pembungkus yang diasosiasikan dengan HTMLInputElement. |
| [OptionCollection](./optioncollection/) | OptionElements mewakili pembungkus yang terkait dengan IHTMLOptionsCollection |
| [OptionElement](./optionelement/) | OptionElement mewakili pembungkus yang terkait dengan HTMLOptionElement |
| [SelectElement](./selectelement/) | SelectElement mewakili pembungkus yang terkait dengan HTMLSelectElement |
| [SubmissionResult](./submissionresult/) | Class ini merupakan hasil dari pengiriman form data ke server. |
| [TextAreaElement](./textareaelement/) | TextAreaElement mewakili pembungkus yang diasosiasikan dengan HTMLTextAreaElement |
## Pencacahan

| Pencacahan | Keterangan |
| --- | --- |
| [ButtonType](./buttontype/) | Pencacahan ini mewakili keadaan tombol. |
| [FormElementType](./formelementtype/) | Mewakili pencacahan jenis Elemen Formulir dan yang sesuai dengan Elemen HTML |
| [InputElementType](./inputelementtype/) | Mewakili status kolom input. |
