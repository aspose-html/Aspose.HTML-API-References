## IFile interface

Objek File adalah objek Blob dengan atribut nama, yaitu string; itu dapat dibuat dalam aplikasi web melalui konstruktor, atau merupakan referensi ke urutan byte dari file dari sistem file (OS) yang mendasarinya.

```csharp
public interface IFile : IBlob
```

## Properti

| Nama | Keterangan |
| --- | --- |
| [LastModified](../../aspose.html.io/ifile/lastmodified/) { get; } | Tanggal file terakhir diubah. Saat mendapatkan, jika agen pengguna dapat membuat informasi ini tersedia, ini harus mengembalikan set lama ke waktu file terakhir diubah sebagai jumlah milidetik sejak Unix Epoch. |
| [Name](../../aspose.html.io/ifile/name/) { get; } | Nama file. Saat mendapatkannya, ini harus mengembalikan nama file sebagai string. |
