Itu **Aspose.Html.Menyimpan** namespace disajikan oleh kelas data untuk deskripsi opsi penyimpanan tertentu di proses konversi/penyimpanan.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [DocSaveOptions](./docsaveoptions/) | Kelas data opsi khusus. |
| [HTMLSaveOptions](./htmlsaveoptions/) | Merupakan opsi penyimpanan HTML. |
| [ImageSaveOptions](./imagesaveoptions/) | Kelas data opsi khusus. |
| [MarkdownSaveOptions](./markdownsaveoptions/) | Mewakili opsi penyimpanan Markdown. |
| [MHTMLSaveOptions](./mhtmlsaveoptions/) | Merupakan opsi penyimpanan MHTML. |
| [PdfSaveOptions](./pdfsaveoptions/) | Kelas data opsi khusus. |
| [ResourceHandlingOptions](./resourcehandlingoptions/) | Mewakili opsi penanganan sumber daya. |
| [SaveOptions](./saveoptions/) | Ini adalah kelas dasar abstrak untuk kelas yang memungkinkan pengguna menentukan opsi tambahan saat menyimpan dokumen ke dalam format tertentu. |
| [XpsSaveOptions](./xpssaveoptions/) | Kelas data opsi khusus. |
## Pencacahan

| Pencacahan | Keterangan |
| --- | --- |
| [HTMLSaveFormat](./htmlsaveformat/) | Menentukan format penyimpanan dokumen. |
| [MarkdownFeatures](./markdownfeatures/) | A[`MarkdownFeatures`](../aspose.html.saving/markdownfeatures/) set flag adalah sekumpulan nol atau lebih dari flag berikut, yang digunakan untuk memilih elemen yang dikonversi menjadi markdown. |
| [MarkdownFormatter](./markdownformatter/) | Menentukan cara output akan diformat. |
| [ResourceHandling](./resourcehandling/) | Enum ini mewakili opsi penanganan sumber daya. |
| [UrlRestriction](./urlrestriction/) | Enum ini menunjukkan batasan yang diterapkan pada URL sumber daya yang diproses. |
