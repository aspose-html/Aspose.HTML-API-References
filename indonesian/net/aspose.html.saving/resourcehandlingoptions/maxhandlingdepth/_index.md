## ResourceHandlingOptions.MaxHandlingDepth property

Mendapat atau menyetel kedalaman maksimum halaman yang akan ditangani. Kedalaman 1 berarti hanya halaman yang direferensikan langsung dari dokumen tersimpan yang akan ditangani. Mengatur properti ini ke -1 akan menyebabkan penanganan semua halaman. Nilai default adalah 0.

```csharp
public int MaxHandlingDepth { get; set; }
```
