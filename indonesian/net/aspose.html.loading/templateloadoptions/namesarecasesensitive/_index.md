## TemplateLoadOptions.NamesAreCaseSensitive property

Properti yang menentukan apakah templat dan nama elemen data akan cocok terlepas dari huruf besar kecil atau tidak. Jika benar, pencocokan peka huruf besar-kecil; jika salah, huruf kapital akan diabaikan. Nilai default adalah benar.

```csharp
public bool NamesAreCaseSensitive { get; set; }
```
