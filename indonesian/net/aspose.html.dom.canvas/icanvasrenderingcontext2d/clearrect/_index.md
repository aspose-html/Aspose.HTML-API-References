## ICanvasRenderingContext2D.ClearRect method

Menyetel semua piksel dalam persegi panjang yang ditentukan oleh titik awal (x, y) dan ukuran (lebar, tinggi) menjadi hitam transparan, menghapus konten yang digambar sebelumnya.

```csharp
public void ClearRect(double x, double y, double w, double h)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| x | Double | Sumbu x koordinat untuk titik awal persegi panjang. |
| y | Double | Sumbu y koordinat untuk titik awal persegi panjang. |
| w | Double | Lebar persegi panjang. |
| h | Double | Tinggi persegi panjang. |
