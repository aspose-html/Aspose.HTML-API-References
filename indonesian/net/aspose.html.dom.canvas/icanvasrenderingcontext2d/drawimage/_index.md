| dx | Double | Koordinat X di kanvas tujuan untuk menempatkan sudut kiri atas gambar sumber. |
| dy | Double | Koordinat Y di kanvas tujuan untuk menempatkan sudut kiri atas gambar sumber. |
| dw | Double | Lebar untuk menggambar gambar di kanvas tujuan. Ini memungkinkan penskalaan gambar yang digambar. Jika tidak ditentukan, lebar gambar tidak diskalakan saat digambar. |
| dh | Double | Ketinggian untuk menggambar gambar di kanvas tujuan. Ini memungkinkan penskalaan gambar yang digambar. Jika tidak ditentukan, tinggi gambar tidak diskalakan saat digambar. |
