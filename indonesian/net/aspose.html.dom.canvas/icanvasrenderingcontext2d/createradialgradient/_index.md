## ICanvasRenderingContext2D.CreateRadialGradient method

Membuat gradien radial yang diberikan oleh koordinat dua lingkaran yang diwakili oleh parameter.

```csharp
public ICanvasGradient CreateRadialGradient(double x0, double y0, double r0, double x1, double y1, 
    double r1)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| x0 | Double | Sumbu x koordinat lingkaran awal. |
| y0 | Double | Sumbu y dari koordinat lingkaran awal |
| r0 | Double | Jari-jari lingkaran awal. |
| x1 | Double | Sumbu x koordinat lingkaran ujung. |
| y1 | Double | Sumbu y dari koordinat lingkaran ujung. |
| r1 | Double | Jari-jari lingkaran ujung. |

### Nilai Pengembalian

CanvasGradient radial diinisialisasi dengan dua lingkaran yang ditentukan.
