```csharp
public void StrokeText(string text, double x, double y, double? maxWidth)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| text | String | Teks yang akan digambar menggunakan font saat ini, textAlign, textBaseline, dan nilai arah. |

| y | Double | Sumbu y koordinat untuk titik awal teks. |
