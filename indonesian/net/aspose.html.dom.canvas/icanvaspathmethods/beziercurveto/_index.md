## ICanvasPathMethods.BezierCurveTo method

Menambahkan kurva Bézier kubik ke jalur. Itu membutuhkan tiga poin. Dua titik pertama adalah titik kontrol dan titik ketiga adalah titik akhir. Titik awal adalah titik terakhir di jalur saat ini, yang dapat diubah menggunakan moveTo() sebelum membuat kurva Bézier.

```csharp
public void BezierCurveTo(double cp1x, double cp1y, double cp2x, double cp2y, double x, double y)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| cp1x | Double | Sumbu x koordinat untuk titik kontrol pertama. |
| cp1y | Double | Sumbu y koordinat untuk titik kontrol pertama. |
| cp2x | Double | Sumbu x koordinat untuk titik kontrol kedua. |
| cp2y | Double | Sumbu y koordinat untuk titik kontrol kedua. |
| x | Double | Sumbu x koordinat titik akhir. |
| y | Double | Sumbu y koordinat titik akhir. |
