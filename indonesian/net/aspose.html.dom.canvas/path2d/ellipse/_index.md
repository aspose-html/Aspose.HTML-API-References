## Ellipse(double, double, double, double, double, double, double) {#ellipse}

Menambahkan elips ke jalur yang berpusat di posisi (x, y) dengan jari-jari radiusX dan radiusY mulai dari startAngle dan berakhir di endAngle menuju arah yang diberikan berlawanan arah jarum jam (defaultnya searah jarum jam).

```csharp
public void Ellipse(double x, double y, double radiusX, double radiusY, double rotation, 
    double startAngle, double endAngle)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| x | Double | Sumbu x koordinat pusat elips. |
| y | Double | Sumbu y koordinat pusat elips. |
| radiusX | Double | Jari-jari sumbu utama elips. |
| radiusY | Double | Jari-jari sumbu minor elips. |
| rotation | Double | Rotasi untuk elips ini, dinyatakan dalam radian. |
| startAngle | Double | Titik awal, diukur dari sumbu x, dari mana ia akan ditarik, dinyatakan dalam radian. |
| endAngle | Double | Sudut akhir elips yang akan digambar, dinyatakan dalam radian. |
