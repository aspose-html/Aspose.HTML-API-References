## Node.ReplaceChild method

Mengganti node anak oldChild dengan newChild dalam daftar anak, dan mengembalikan node oldChild. Jika newChild adalah objek DocumentFragment, oldChild digantikan oleh semua turunan DocumentFragment, yang disisipkan dalam urutan yang sama. Jika newChild sudah ada di pohon, pertama dihapus.

```csharp
public Node ReplaceChild(Node node, Node child)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| node | Node | Node baru. |
| child | Node | Anak tua. |
