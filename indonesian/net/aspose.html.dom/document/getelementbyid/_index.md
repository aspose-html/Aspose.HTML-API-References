## Document.GetElementById method

Mengembalikan Elemen yang memiliki atribut ID dengan nilai yang diberikan. Jika tidak ada elemen seperti itu, ini mengembalikan nol. Jika lebih dari satu elemen memiliki atribut ID dengan nilai tersebut, yang dikembalikan adalah undefined.

```csharp
public Element GetElementById(string elementId)
```
