## INodeIterator.Detach method

Melepaskan NodeIterator dari set yang diulangi , melepaskan semua sumber daya komputasi dan menempatkan iterator dalam status INVALID. Setelah pelepasan dipanggil, panggilan ke Node berikutnya atau Node sebelumnya akan memunculkan pengecualian INVALID_STATE_ERR.
