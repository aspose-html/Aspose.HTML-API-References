## INodeIterator.NextNode method

Mengembalikan node berikutnya di set dan memajukan posisi iterator di set. Setelah NodeIterator dibuat, panggilan pertama ke nextNode() mengembalikan node pertama di set.

```csharp
public Node NextNode()
```

### Nilai Pengembalian

Node berikutnya dalam himpunan sedang diulangi, atau null jika tidak ada lagi anggota dalam himpunan tersebut.
