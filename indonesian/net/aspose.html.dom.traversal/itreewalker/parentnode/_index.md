## ITreeWalker.ParentNode method

Berpindah ke dan mengembalikan node leluhur terdekat yang terlihat dari node current . Jika pencarian parentNode mencoba step ke atas dari node root TreeWalker, atau jika gagal menemukan node ancestor yang terlihat, metode ini mempertahankan posisi saat ini dan mengembalikan null.

```csharp
public Node ParentNode()
```

### Nilai Pengembalian

Node induk baru, atau null jika node saat ini tidak memiliki induk dalam tampilan logis TreeWalker.
