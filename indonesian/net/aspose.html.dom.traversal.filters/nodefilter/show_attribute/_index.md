## NodeFilter.SHOW_ATTRIBUTE field

Tampilkan node Attr. Ini berarti hanya saat membuat iterator atau tree-walker dengan node atribut sebagai root ; dalam hal ini berarti node atribut akan muncul di posisi pertama iterasi atau traversal. Karena atribut tidak pernah menjadi anak dari node lain, mereka tidak muncul saat melintasi pohon dokumen.

```csharp
public const long SHOW_ATTRIBUTE;
```
