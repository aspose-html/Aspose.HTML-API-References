Itu **Aspose.Html.Collections** namespace terdiri dari kelas untuk mewakili, menyimpan dan memanipulasi node dan elemen.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [DOMTokenList](./domtokenlist/) | Kelas DOMTokenList mewakili satu set token yang dipisahkan oleh ruang. Itu diindeks dimulai dengan 0 seperti objek JavaScript Array. DOMTokenList selalu peka huruf besar/kecil. |
| [HTMLCollection](./htmlcollection/) | Itu[`HTMLCollection`](../aspose.html.collections/htmlcollection/) mewakili koleksi umum dari[`Element`](../aspose.html.dom/element/) . |
| [NamedNodeMap](./namednodemap/) | Mewakili kumpulan atribut yang dapat diakses dengan nama. |
| [NodeList](./nodelist/) | NodeList menyediakan abstraksi kumpulan node yang diurutkan, tanpa menentukan atau membatasi bagaimana kumpulan ini diimplementasikan. |
