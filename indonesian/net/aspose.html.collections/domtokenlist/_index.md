## DOMTokenList class

Kelas DOMTokenList mewakili satu set token yang dipisahkan oleh ruang. Itu diindeks dimulai dengan 0 seperti objek JavaScript Array. DOMTokenList selalu peka huruf besar/kecil.

public class DOMTokenList : DOMObject, IEnumerable<string>
