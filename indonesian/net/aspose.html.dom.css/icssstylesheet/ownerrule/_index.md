## ICSSStyleSheet.OwnerRule property

Jika style sheet ini berasal dari aturan @import, atribut ownerRule akan berisi CSSImportRule. Dalam hal ini, atribut ownerNode di antarmuka StyleSheet akan menjadi nol. Jika style sheet berasal dari elemen atau instruksi pemrosesan, atribut ownerRule akan menjadi null dan atribut ownerNode akan berisi Node.

```csharp
public ICSSRule OwnerRule { get; }
```
