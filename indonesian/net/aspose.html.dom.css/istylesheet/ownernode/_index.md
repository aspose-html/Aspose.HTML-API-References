## IStyleSheet.OwnerNode property

Node yang menghubungkan style sheet ini dengan dokumen. Untuk HTML, ini mungkin elemen LINK atau GAYA yang sesuai. Untuk XML, ini mungkin merupakan instruksi pemrosesan penautan. Untuk style sheet yang disertakan oleh style sheet lain, nilai atribut ini adalah null.
