## ILinkStyle interface

Antarmuka LinkStyle menyediakan mekanisme di mana lembar gaya dapat diambil dari node yang bertanggung jawab untuk menautkannya ke dalam dokumen. Instance antarmuka LinkStyle dapat diperoleh dengan menggunakan metode pengecoran khusus yang mengikat pada instance node penautan (HTMLLinkElement, HTMLStyleElement, atau ProcessingInstruction di DOM Level 2).
