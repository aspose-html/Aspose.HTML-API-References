## IXPathNSResolver interface

Itu`XPathNSResolver` izin antarmuka`awalan` string dalam ekspresi yang akan diikat dengan benar`namespaceURI` string. [`IXPathEvaluator`](../ixpathevaluator/) dapat membuat implementasi `IXPathNSResolver` dari sebuah node, atau antarmuka dapat diimplementasikan oleh aplikasi apa pun.
