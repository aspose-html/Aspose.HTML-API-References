Ruang nama berisi metode untuk menavigasi elemen dan atribut dalam dokumen XML.

## Antarmuka

| Antarmuka | Keterangan |
| --- | --- |
| [IXPathEvaluator](./ixpathevaluator/) | Evaluasi ekspresi XPath disediakan oleh[`IXPathEvaluator`](../aspose.html.dom.xpath/ixpathevaluator/) . |
| [IXPathExpression](./ixpathexpression/) | Itu`XPathExpression`antarmuka mewakili ekspresi XPath yang diuraikan dan diselesaikan. |
| [IXPathNamespace](./ixpathnamespace/) | Antarmuka XPathNamespace dikembalikan oleh antarmuka XPathResult untuk mewakili tipe node namespace XPath yang tidak dimiliki DOM. |
| [IXPathNSResolver](./ixpathnsresolver/) | Itu`XPathNSResolver` izin antarmuka`awalan` string dalam ekspresi yang akan diikat dengan benar`namespaceURI` string. [`IXPathEvaluator`](../aspose.html.dom.xpath/ixpathevaluator/) dapat membuat implementasi [`IXPathNSResolver`](../aspose.html.dom.xpath/ixpathnsresolver/) dari sebuah node, atau antarmuka dapat diimplementasikan oleh aplikasi apa pun. |
| [IXPathResult](./ixpathresult/) | Itu`XPathResult` antarmuka mewakili hasil evaluasi ekspresi XPath 1.0 dalam konteks node tertentu. Karena evaluasi dari ekspresi XPath dapat menghasilkan berbagai jenis hasil, objek ini memungkinkan untuk menemukan dan memanipulasi jenis dan nilai hasil. |
## Pencacahan

| Pencacahan | Keterangan |
| --- | --- |
| [XPathResultType](./xpathresulttype/) | Short unsigned yang menunjukkan jenis hasil apa ini. Jika tertentu`jenis`ditentukan, maka hasilnya akan dikembalikan sebagai tipe yang sesuai, menggunakan konversi tipe XPath jika diperlukan dan memungkinkan. |
