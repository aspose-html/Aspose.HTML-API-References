## HTMLCanvasElement.Width property

Adalah bilangan bulat positif yang mencerminkan atribut HTML lebar dari elemen kanvas yang ditafsirkan dalam piksel CSS. Saat atribut tidak ditentukan, atau jika disetel ke nilai yang tidak valid, seperti negatif, nilai default 300 digunakan.

```csharp
public ulong Width { get; set; }
```
