## HTMLCanvasElement.Height property

Adalah bilangan bulat positif yang mencerminkan atribut HTML tinggi dari elemen kanvas yang ditafsirkan dalam piksel CSS. Saat atribut tidak ditentukan, atau jika disetel ke nilai yang tidak valid, seperti negatif, nilai default 150 digunakan.

```csharp
public ulong Height { get; set; }
```
