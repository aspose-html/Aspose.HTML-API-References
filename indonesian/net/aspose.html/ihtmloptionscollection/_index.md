## IHTMLOptionsCollection interface

An`Koleksi HTMLOptions` adalah daftar node yang mewakili elemen opsi HTML. Sebuah node individual dapat diakses dengan indeks ordinal atau milik node`nama` atau`pengenal` atribut. Koleksi di DOM HTML dianggap hidup yang berarti bahwa mereka secara otomatis diperbarui ketika dokumen yang mendasarinya diubah.

Lihat juga[Document object Model (DOM) Spesifikasi HTML Level 2](http://www.w3.org/TR/2003/REC-DOM-Level-2-HTML-20030109). @sejak DOM Tingkat 2
