## HTMLInputElement.DefaultValue property

Ketika`jenis`atribut elemen memiliki nilai "text", "file" atau "password", ini mewakili nilai HTML atribut elemen. Nilai atribut ini tidak berubah jika konten kontrol formulir yang sesuai, dalam agen pengguna interaktif, berubah. Lihat definisi atribut nilai dalam HTML 4.01.

```csharp
public string DefaultValue { get; set; }
```
