## HTMLInputElement.Files property

Atribut IDL file memungkinkan skrip untuk mengakses file elemen yang dipilih. Saat menerima, jika atribut IDL berlaku, ia harus mengembalikan objek FileList yang mewakili file yang dipilih saat ini. Objek yang sama harus dikembalikan hingga daftar file yang dipilih berubah. Jika atribut IDL tidak berlaku, maka atribut tersebut harus mengembalikan nol. [FILEAPI]

```csharp
public IFileList Files { get; }
```
