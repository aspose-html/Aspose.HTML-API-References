## HTMLDocument.Images property

Koleksi dari semua`IMG` elemen dalam sebuah dokumen. Perilaku terbatas pada`IMG` elemen untuk kompatibilitas mundur. Seperti yang disarankan oleh [[HTML4.01](http://www.w3.org/TR/1999/REC-html401-19991224)], untuk menyertakan gambar, penulis dapat menggunakan `OBYEK` elemen atau`IMG` elemen. Oleh karena itu, disarankan untuk tidak menggunakan atribut ini untuk menemukan gambar dalam dokumen tetapi`getElementsByTagName` dengan HTML 4.01 atau`getElementsByTagNameNS` dengan XHTML 1.0.

```csharp
public HTMLCollection Images { get; }
```
