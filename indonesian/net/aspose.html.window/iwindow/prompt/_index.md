## IWindow.Prompt method

Menampilkan prompt bidang teks modal dengan pesan yang diberikan, menunggu pengguna menutupnya, dan mengembalikan nilai yang dimasukkan pengguna. Jika pengguna membatalkan prompt, maka kembalikan null sebagai gantinya. Jika argumen kedua ada, maka nilai yang diberikan digunakan sebagai default.

```csharp
public string Prompt(string message, string @default)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| message | String | Pesan. |
| default | String | Default. |

### Nilai Pengembalian
