## SVGPoint class

Banyak antarmuka DOM SVG mengacu pada objek kelas SVGPoint. SVGPoint adalah pasangan koordinat (x, y). Saat digunakan dalam operasi matriks, SVGPoint diperlakukan sebagai vektor dalam bentuk: [x] [y] [1] Jika objek SVGRect ditetapkan sebagai hanya baca, maka mencoba menetapkan ke salah satu atributnya akan menghasilkan pengecualian yang dilemparkan.

```csharp
public class SVGPoint : SVGValueType
```
