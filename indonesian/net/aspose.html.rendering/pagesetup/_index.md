## PageSetup class

Merupakan objek penyiapan halaman yang digunakan untuk set-halaman keluaran konfigurasi.

```csharp
public class PageSetup
```

## Properti

| Nama | Keterangan |
| --- | --- |
| [AdjustToWidestPage](../../aspose.html.rendering/pagesetup/adjusttowidestpage/) { get; set; } | Mendapat atau menyetel flag yang menentukan kasus kapan ukuran halaman akan disesuaikan dengan halaman terluas dalam dokumen. Opsi ini memakan waktu sehingga waktu pemrosesan dokumen dapat ditingkatkan dua kali lipat. Penyesuaian akan terjadi hanya jika halaman terluas dalam dokumen lebih lebar dari ukuran halaman yang ditentukan`PageSetup` . Ukuran halaman yang disesuaikan akan digunakan untuk semua halaman dalam dokumen. |
| [AnyPage](../../aspose.html.rendering/pagesetup/anypage/) { get; set; } | Mendapat atau menyetel semua konfigurasi halaman dalam urutan halaman. |
| [AtPagePriority](../../aspose.html.rendering/pagesetup/atpagepriority/) { get; set; } | Mendapat atau set[`AtPagePriority`](../atpagepriority/) yang akan menentukan urutan penerapan deklarasi ukuran halaman. Secara default opsi akan menimpa css`@halaman` aturan . |
| [FirstPage](../../aspose.html.rendering/pagesetup/firstpage/) { get; set; } | Mendapat atau menyetel konfigurasi halaman pertama. |
| [LeftPage](../../aspose.html.rendering/pagesetup/leftpage/) { get; } | Mendapatkan konfigurasi Halaman Ganjil. |
| [PageLayoutOptions](../../aspose.html.rendering/pagesetup/pagelayoutoptions/) { get; set; } | Mendapat atau menyetel[`PageLayoutOptions`](../pagelayoutoptions/) . Nilai default adalahNone , nilai lain apa pun akan menimpa[`AdjustToWidestPage`](./adjusttowidestpage/) perilaku. Hanya berfungsi dengan dokumen HTML. |
| [RightPage](../../aspose.html.rendering/pagesetup/rightpage/) { get; } | Mendapatkan konfigurasi Halaman Genap. |
