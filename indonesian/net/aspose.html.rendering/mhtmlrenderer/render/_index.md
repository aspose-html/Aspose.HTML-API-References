## Render(IDevice, TimeSpan, params Stream[]) {#render_9}

Membuat beberapa dokumen MHTML menjadi ditentukan[`IDevice`](../../idevice/) . Rendering akan dilakukan setelah tidak ada operasi jaringan apa pun untuk memuat sumber daya, timer aktif, tugas animasi, atau batas waktu tertentu telah berlalu.

```csharp
public override void Render(IDevice device, TimeSpan timeout, params Stream[] documents)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| device | IDevice | Perangkat. |
| timeout | TimeSpan | ATimeSpan yang mewakili jumlah milidetik untuk menunggu, atau aTimeSpan yang mewakili -1 milidetik untuk menunggu tanpa batas. |
| documents | Stream[] | Dokumen yang akan dirender. |
