Itu **Aspose.Html.Rendering** namespace terdiri dari banyak objek penyaji serta kelas opsi tingkat rendah yang sesuai yang bertanggung jawab untuk merender dokumen/file ke implementasi IDevice.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [CssOptions](./cssoptions/) | Mewakili opsi rendering css. |
| [Device&lt;TGraphicContext,TRenderingOptions&gt;](./device-2/) | Merupakan kelas dasar untuk implementasi perangkat rendering tertentu. |
| [EpubRenderer](./epubrenderer/) | Merupakan perender dokumen EPub. |
| [GraphicContext](./graphiccontext/) | Menyimpan parameter kontrol grafis saat ini. Parameter ini menentukan kerangka kerja global tempat operator grafis mengeksekusi. |
| [HtmlRenderer](./htmlrenderer/) | Merupakan penyaji dokumen HTML. |
| [MhtmlRenderer](./mhtmlrenderer/) | Merupakan penyaji dokumen MHTML. |
| [PageSetup](./pagesetup/) | Merupakan objek penyiapan halaman yang digunakan untuk set-halaman keluaran konfigurasi. |
| [Renderer](./renderer/) | Mewakili kelas dasar untuk semua penyaji dan mengimplementasikan antarmuka IDisposable. |
| [Renderer&lt;TDocument&gt;](./renderer-1/) | Merupakan kelas abstrak untuk semua penyaji. |
| [RenderingOptions](./renderingoptions/) | Mewakili opsi rendering. |
| [SvgRenderer](./svgrenderer/) | Merupakan perender dokumen SVG. |
| [TextInfo](./textinfo/) | Berisi informasi tentang teks yang dirender. |
## Struktur

| Struktur | Keterangan |

| [GlyphInfo](./glyphinfo/) | Berisi informasi terkait mesin terbang. |

| Antarmuka | Keterangan |
