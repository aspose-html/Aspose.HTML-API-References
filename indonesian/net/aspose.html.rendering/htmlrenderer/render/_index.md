## HtmlRenderer.Render method

Menentukan metode untuk merender banyak[`Document`](../../../aspose.html.dom/document/) menjadi spesifik[`IDevice`](../../idevice/) . Rendering akan dilakukan setelah tidak ada operasi jaringan apa pun untuk memuat sumber daya, timer aktif, tugas animasi, atau batas waktu tertentu telah berlalu.

```csharp
public override void Render(IDevice device, TimeSpan timeout, params Document[] documents)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| device | IDevice | Perangkat keluaran. |
| timeout | TimeSpan | ATimeSpan yang mewakili jumlah milidetik untuk menunggu, atau aTimeSpan yang mewakili -1 milidetik untuk menunggu tanpa batas. |
| documents | Document[] | Dokumen yang akan dirender. |
