## ImageRenderingOptions.HorizontalResolution property

设置或获取输出和内部（在过滤器处理期间使用）图像的水平分辨率，以像素/英寸为单位。默认情况下，此属性为 300 dpi.

```csharp
public override Resolution HorizontalResolution { get; set; }
```
