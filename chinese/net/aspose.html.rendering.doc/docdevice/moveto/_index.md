## DocDevice.MoveTo method

通过将当前点移动到参数 pt 的坐标开始新的子路径，省略任何连接线段。 如果当前路径中之前的路径构造方法也是“MoveTo”，则新的“MoveTo”覆盖它； 路径中没有先前“MoveTo”操作的痕迹。

```csharp
public override void MoveTo(PointF pt)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| pt | PointF | 将路径移动到的位置。 |
