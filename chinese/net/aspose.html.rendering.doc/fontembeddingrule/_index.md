## FontEmbeddingRule enumeration

表示字体嵌入规则。

```csharp
public enum FontEmbeddingRule
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Full | `1` | 所有使用的字体都将按原样嵌入。请注意，某些字体可能已获得许可，这将使生成的文档无法编辑。此外，嵌入的字体文件可能会显着增加生成的文档的大小。 |
| None | `2` | 字体不会被嵌入。 |
