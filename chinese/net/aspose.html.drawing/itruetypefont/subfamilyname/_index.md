## ITrueTypeFont.SubFamilyName property

字体子系列名称用于区分具有相同字体系列名称的组中的字体。 这被假定为处理样式（斜体、斜体）和粗体（浅色、粗体、黑色等）。 在粗细或样式上没有特别差异的字体（例如，中等粗细，不是斜体和 fsSelection 位 6 集）应该在此位置存储字符串“Regular”。

public string SubFamilyName { get; }
