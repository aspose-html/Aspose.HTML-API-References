## HTMLTableElement.InsertRow method

在表中插入一个新的空行。新行插入到 之前并与当前 ` index` 中的第桌子。如果` index` 为 -1 或 等于行数，则追加新行。此外， 当表为空时，该行被插入到` TBODY` 中，它被创建并插入根据 [[HTML 4.01 ，表格行不能为 空](http://www.w3.org/TR/1999/REC-html401-19991224) ]。

```csharp
public Node InsertRow(int index)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| index | Int32 | 插入新行的行号。此索引 从 0 开始，并且与表中包含的所有行的逻辑顺序（不是文档 顺序）相关。 |

### 返回值

新创建的行。
