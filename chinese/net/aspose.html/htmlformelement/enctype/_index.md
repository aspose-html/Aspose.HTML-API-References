## HTMLFormElement.Enctype property

提交表单的内容类型，一般为 "application/x-www-form-urlencoded"。请参阅 HTML 4.01 中的 enctype 属性 定义。调用此方法时，不能保证 触发 onsubmit 偶数处理程序。由于历史原因，该行为 不一致，作者不应依赖 特定行为。

```csharp
public string Enctype { get; set; }
```
