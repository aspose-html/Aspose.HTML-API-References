## HTMLDocument.Referrer property

返回 URI [[IETF RFC 2396](http://www.ietf.org/rfc/rfc2396.txt) 链接到此页面的页面。如果用户直接导航到页面（不是通过 链接，而是通过书签等），则该值为 空字符串。

```csharp
public string Referrer { get; }
```
