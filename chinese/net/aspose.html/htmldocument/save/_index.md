## Save(Url) {#save_5}

将文档保存到指定的本地文件`网址`.本文档中使用的所有资源将保存在 到相邻文件夹中，其名称将构造为：output_file_name + "_files".

```csharp
public void Save(Url url)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| url | Url | 输出文件的本地 URL。 |

### 例外

| 例外 | （健康）状况 |
| --- | --- |
| ArgumentException | 如果指定则引发`网址`不是有效的本地文件 URL。 |
