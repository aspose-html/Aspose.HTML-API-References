## HTMLDocument.Images property

文档中所有` IMG` 元素的集合。 行为仅限于` IMG` 元素，以实现向后 兼容性。正如 [[HTML 4.01](http://www.w3.org/TR/1999/REC-html401-19991224) ] 所建议的，要包含图像，作者可以使用 ` OBJECT` 元素或` IMG` 元素。 因此，建议不要使用该属性来查找文档中的 图片而是` getElementsByTagName` 和 HTML 4.01 或` getElementsByTagNameNS` 与 XHTML 1.0。
