## IHTMLOptionsCollection interface

一个` HTMLOptionsCollection` 是代表 HTML 选项元素的节点列表。 单个节点可以通过序数 索引或节点的`访问 名称` 或` 编号` 属性。 HTML DOM 中的集合被假定为实时的，这意味着它们是 在底层文档更改时自动更新。

另请参见[文档对象模型 (DOM) 级别 2 HTML 规范](http://www.w3.org/TR/2003/REC-DOM-Level-2-HTML-20030109) . @since DOM Level 2
