## HTMLInputElement.Checked property

当`类型`元素的属性具有值 “radio”或“checkbox”，这表示在交互式用户代理中表单 控件的当前状态。更改此属性 会更改表单控件的状态，但不会更改 INPUT 元素的 HTML checked 属性的值。在处理具有 值“radio”或“checkbox”，某些实现可能会在 文档中调度事件之前更改此属性的 值。如果取消事件的默认操作，则属性的值 可能会更改回其原始值。这意味着 在处理单击事件期间此属性的值 取决于实现。

```csharp
public bool Checked { get; set; }
```
