## HTMLInputElement.DefaultValue property

当`类型`元素的属性具有值 “文本”、“文件”或“密码”，这表示元素的 HTML 值 属性。如果相应表单控件的内容在交互式 用户代理中发生更改，则此属性的值不会更改 。请参阅 HTML 4.01. 中的值属性定义

```csharp
public string DefaultValue { get; set; }
```
