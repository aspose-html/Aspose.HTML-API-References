## HTMLInputElement.Value property

当元素的` 类型` 属性的值为 "text ", "file" 或 "password", 这表示 相应表单控件的当前内容，在交互式用户代理中。 更改此属性会更改表单控件的内容，但 不会更改元素的 HTML value 属性的值。 当元素的` 类型` 属性值为 "button", "hidden", "submit "、"reset"、"image"、"checkbox" 或 "radio"，这表示元素的 HTML 值属性。请参阅 HTML 4.01 中的 value 属性定义。
