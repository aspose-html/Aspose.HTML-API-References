## HTMLInputElement.Value property

当`类型`元素的属性具有值 “文本”、“文件”或“密码”，这表示在交互式用户代理中 相应表单控件的当前内容。 更改此属性会更改表单控件的内容，但 不会更改该元素的 HTML value 属性的值。 当`类型`元素的属性具有值 "button"、"hidden"、"submit"、"reset"、"image"、"checkbox" 或 "radio"，这表示元素的 HTML 值属性。请参阅 HTML 4.01. 中的值属性定义

```csharp
public string Value { get; set; }
```
