## HTMLInputElement.DefaultChecked property

什么时候`类型`具有值“radio”或“checkbox”，这个 代表元素的HTML checked属性。如果相应表单 控件的状态在交互式用户代理中发生更改，则 此属性的值不会更改。请参阅 HTML 4.01. 中已检查的 属性定义

```csharp
public bool DefaultChecked { get; set; }
```
