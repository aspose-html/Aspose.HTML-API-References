## HTMLTableSectionElement.InsertRow method

在此部分中插入一行。新行立即插入 在当前` 索引`本节中的第 行之前。如果 ` index` 为 -1 或等于此 部分中的行数，则新行为附加。

```csharp
public HTMLElement InsertRow(int index)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| index | Int32 | 插入新行的行号。该索引 从 0 开始，并且仅与 此部分中包含的行相关，而不是表中的所有行。 |

### 返回值

新创建的行。
