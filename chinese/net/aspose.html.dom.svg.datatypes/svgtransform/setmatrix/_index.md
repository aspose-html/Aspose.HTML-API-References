## SVGTransform.SetMatrix method

将变换类型设置为 SVG_TRANSFORM_MATRIX，参数矩阵定义新的变换。复制参数矩阵的值，矩阵参数不会替换 SVGTransform::matrix.

```csharp
public void SetMatrix(SVGMatrix matrix)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| matrix | SVGMatrix | 转换的新矩阵。 |
