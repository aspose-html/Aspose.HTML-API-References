## Event.PreventDefault method

如果事件是可取消的，则`PreventDefault`方法用于表示事件将被取消， 意味着任何通常由实现作为事件结果而采取的默认操作都不会发生。

```csharp
public void PreventDefault()
```

### 评论

如果在事件流的任何阶段，`PreventDefault`方法被调用，事件被取消。 与事件相关的任何默认操作都不会发生。 对不可取消的事件调用此方法无效。 一旦`PreventDefault`被调用，它将在事件传播的其余部分保持有效。 此方法可用于事件流的任何阶段。
