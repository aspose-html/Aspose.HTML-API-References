## IEventTarget.DispatchEvent method

此方法允许将事件分派到实现事件模型中。

```csharp
public bool DispatchEvent(Event @event)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| event | Event | 指定在处理事件时要使用的事件类型、行为和上下文信息。 |

### 返回值

[`DispatchEvent`](../../../aspose.html.dom/eventtarget/dispatchevent)指示是否有任何处理名为[`PreventDefault`](../../event/preventdefault)的事件的侦听器。 如果[`PreventDefault`](../../event/preventdefault)被调用，则值为 false，否则值为 true。

### 评论

以这种方式分派的事件将具有与直接由实现分派的事件相同的捕获和冒泡行为。 事件的目标是[`EventTarget`](../../../aspose.html.dom/eventtarget)在其上[`DispatchEvent`](../../../aspose.html.dom/eventtarget/dispatchevent)被调用。
