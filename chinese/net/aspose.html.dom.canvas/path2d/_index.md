## Path2D class

Canvas 2D API 的 Path2D 接口用于声明稍后用于 CanvasRenderingContext2D 对象的路径。 CanvasRenderingContext2D 接口的路径方法也出现在此接口上，并允许您创建 路径，您可以根据需要在画布上保留和重放这些路径。

```csharp
public class Path2D : DOMObject, ICanvasPathMethods, IDisposable
```
