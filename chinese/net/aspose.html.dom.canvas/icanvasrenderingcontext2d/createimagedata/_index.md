## CreateImageData(double, double) {#createimagedata_1}

创建具有指定尺寸的新的空白 ImageData 对象。 新对象中的所有像素都是透明的黑色。

```csharp
public IImageData CreateImageData(double sw, double sh)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| sw | Double | 赋予新 ImageData 对象的宽度。 |
| sh | Double | 赋予新 ImageData 对象的高度。 |

### 返回值

具有指定宽度和高度的新 ImageData 对象。新对象填充有透明的黑色像素。
