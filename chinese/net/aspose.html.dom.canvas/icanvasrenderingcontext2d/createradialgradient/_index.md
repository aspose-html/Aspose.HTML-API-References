## ICanvasRenderingContext2D.CreateRadialGradient method

创建由参数表示的两个圆的坐标给定的径向渐变。

```csharp
public ICanvasGradient CreateRadialGradient(double x0, double y0, double r0, double x1, double y1, 
    double r1)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| x0 | Double | 起始圆坐标的 x 轴。 |
| y0 | Double | 起始圆坐标的y轴 |
| r0 | Double | 起始圆的半径。 |
| x1 | Double | 结束圆坐标的 x 轴。 |
| y1 | Double | 结束圆坐标的y轴。 |
| r1 | Double | 结束圆的半径。 |

### 返回值

用两个指定的圆圈初始化的径向 CanvasGradient。
