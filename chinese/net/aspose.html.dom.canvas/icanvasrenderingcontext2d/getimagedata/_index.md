## ICanvasRenderingContext2D.GetImageData method

返回一个 ImageData 对象，该对象表示画布区域的底层像素数据，该区域由从 (sx, sy) 开始并具有 sw 宽度和 sh 高度的矩形表示. 此方法不受画布变换矩阵的影响。

```csharp
public IImageData GetImageData(double sx, double sy, double sw, double sh)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| sx | Double | 要从中提取 ImageData 的矩形左上角的 x 坐标。 |
| sy | Double | 要从中提取 ImageData 的矩形左上角的 y 坐标。 |
| sw | Double | 从中提取 ImageData 的矩形的宽度。 |
| sh | Double | 从中提取 ImageData 的矩形的高度。 |

### 返回值

包含画布给定矩形的图像数据的 ImageData 对象。
