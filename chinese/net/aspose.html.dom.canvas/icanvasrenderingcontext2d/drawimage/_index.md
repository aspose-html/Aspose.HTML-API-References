## DrawImage(HTMLImageElement, double, double, double, double) {#drawimage_4}

绘制指定图像。

```csharp
public void DrawImage(HTMLImageElement image, double dx, double dy, double dw, double dh)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| image | HTMLImageElement | 要绘制到上下文中的 HTMLImageElement。 |
| dx | Double | 目标画布中放置源图像左上角的 X 坐标。 |
| dy | Double | 目标画布中放置源图像左上角的 Y 坐标。 |

## DrawImage(HTMLImageElement, double, double, double, double, double, double, double, double) {#drawimage_5}

```csharp
public void DrawImage(HTMLImageElement image, double sx, double sy, double sw, double sh, 
    double dx, double dy, double dw, double dh)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| image | HTMLImageElement | 要绘制到上下文中的 HTMLImageElement。 |
| sx | Double | 要绘制到目标上下文中的源图像子矩形左上角的 X 坐标。 |
| sy | Double | 要绘制到目标上下文中的源图像子矩形左上角的 Y 坐标。 |
| sw | Double | 要绘制到目标上下文中的源图像子矩形的宽度。如果未指定，则使用从 sx 和 sy 指定的坐标到图像右下角的整个矩形。 |
| sh | Double | 要绘制到目标上下文中的源图像子矩形的高度。 |
| dx | Double | 目标画布中放置源图像左上角的 X 坐标。 |
| dy | Double | 目标画布中放置源图像左上角的 Y 坐标。 |
| dw | Double | 在目标画布中绘制图像的宽度。这允许缩放绘制的图像。如果未指定，则图像在绘制时不会按宽度缩放。 |
| dh | Double | 在目标画布中绘制图像的高度。这允许缩放绘制的图像。如果未指定，则图像在绘制时不会按高度缩放。 |
