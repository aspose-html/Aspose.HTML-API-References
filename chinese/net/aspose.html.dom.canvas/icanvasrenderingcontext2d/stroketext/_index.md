
| 范围 | 类型 | 描述 |
| --- | --- | --- |
| text | String | 要使用当前字体、textAlign、textBaseline 和方向值绘制的文本。 |
| x | Double | 文本起点坐标的 x 轴。 |
| y | Double | 文本起点坐标的 y 轴。 |
| maxWidth | Nullable`1 | 要绘制的最大宽度。如果指定，并且字符串被计算为比此宽度更宽，则字体将被调整为使用更水平压缩的字体（如果可用，或者如果可以通过水平缩放当前字体来合成合理可读的字体）或更小字体。 |
