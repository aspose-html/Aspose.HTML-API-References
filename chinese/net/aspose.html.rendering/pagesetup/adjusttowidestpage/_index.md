## PageSetup.AdjustToWidestPage property

获取或设置确定何时将页面大小调整为文档中最宽页面的情况的标志。 此选项非常耗时，因此文档处理时间可以增加两倍。仅当文档中最宽的页面大于在[`PageSetup`](../../pagesetup)中确定的页面大小时，才会进行调整 。调整后的 页面大小将用于文档中的所有页面。

```csharp
public bool AdjustToWidestPage { get; set; }
```
