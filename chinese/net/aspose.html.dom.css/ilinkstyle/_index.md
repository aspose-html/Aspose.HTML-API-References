## ILinkStyle interface

LinkStyle 接口提供了一种机制，通过该机制可以从负责将样式表链接到文档的节点中检索样式表。 LinkStyle 接口的实例可以在链接节点的实例（DOM 级别 2 中的 HTMLLinkElement、HTMLStyleElement 或 ProcessingInstruction）上使用特定于绑定的转换方法来获得。
