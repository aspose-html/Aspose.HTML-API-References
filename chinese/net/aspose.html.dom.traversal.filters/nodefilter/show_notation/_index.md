## NodeFilter.SHOW_NOTATION field

显示符号节点。这仅在创建 迭代器或 tree-walker 时才有意义，其中 Notation 节点作为其 根；在这种情况下，这意味着 Notation 节点将出现在 遍历的第一个位置。由于符号不是文档树的一部分，因此在遍历文档树时它们确实 不会出现。

public const long SHOW_NOTATION;
