## TemplateLoadOptions.NamesAreCaseSensitive property

Ιδιότητα που καθορίζει εάν τα ονόματα του προτύπου και των στοιχείων δεδομένων θα ταιριάζουν ανεξάρτητα από τα πεζά γράμματα ή όχι. Εάν είναι true, η αντιστοίχιση έχει διάκριση πεζών-κεφαλαίων. εάν είναι false, τα πεζά γράμματα χαρακτήρων θα αγνοηθούν. Η προεπιλεγμένη τιμή είναι true.

```csharp
public bool NamesAreCaseSensitive { get; set; }
```
