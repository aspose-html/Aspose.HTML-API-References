## PageSetup.AdjustToWidestPage property

Λαμβάνει ή ορίζει τη σημαία που καθορίζει πότε το μέγεθος σελίδας θα προσαρμοστεί στην ευρύτερη σελίδα στο έγγραφο. Αυτή η επιλογή είναι χρονοβόρα, επομένως ο χρόνος επεξεργασίας του εγγράφου μπορεί να αυξηθεί κατά δύο φορές. Η προσαρμογή θα πραγματοποιηθεί μόνο εάν η ευρύτερη σελίδα στο έγγραφο είναι μεγαλύτερη από το μέγεθος σελίδας που καθορίζεται στο[`PageSetup`](../) . Το μέγεθος σελίδας Adjusted θα χρησιμοποιηθεί για όλες τις σελίδες του εγγράφου.

```csharp
public bool AdjustToWidestPage { get; set; }
```
