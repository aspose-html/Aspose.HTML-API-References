## StrokeText(string, double, double) {#stroketext}

```csharp
public void StrokeText(string text, double x, double y)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| text | String | Το κείμενο που θα σχεδιάσετε χρησιμοποιώντας τις τρέχουσες τιμές γραμματοσειράς, textAlign, textBaseline και κατεύθυνσης. |
| x | Double | Ο άξονας x της συντεταγμένης για το σημείο εκκίνησης του κειμένου. |

## StrokeText(string, double, double, double?) {#stroketext_1}

Σχεδιάζει (χτυπά) ένα δεδομένο κείμενο στη δεδομένη θέση (x, y).

```csharp
public void StrokeText(string text, double x, double y, double? maxWidth)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| text | String | Το κείμενο που θα σχεδιάσετε χρησιμοποιώντας τις τρέχουσες τιμές γραμματοσειράς, textAlign, textBaseline και κατεύθυνσης. |
| x | Double | Ο άξονας x της συντεταγμένης για το σημείο εκκίνησης του κειμένου. |
| y | Double | Ο άξονας y της συντεταγμένης για το σημείο εκκίνησης του κειμένου. |
| maxWidth | Nullable`1 | Το μέγιστο πλάτος για σχεδίαση. Εάν καθορίζεται, και η συμβολοσειρά υπολογίζεται ότι είναι ευρύτερη από αυτό το πλάτος, η γραμματοσειρά προσαρμόζεται ώστε να χρησιμοποιεί μια πιο οριζόντια συμπυκνωμένη γραμματοσειρά (εάν είναι διαθέσιμη ή εάν μια εύλογα αναγνώσιμη μπορεί να συντεθεί κλιμακώνοντας την τρέχουσα γραμματοσειρά οριζόντια) ή μικρότερη γραμματοσειρά. |
