## ICanvasRenderingContext2D.CreateRadialGradient method

Δημιουργεί μια ακτινική κλίση που δίνεται από τις συντεταγμένες των δύο κύκλων που αντιπροσωπεύονται από τις παραμέτρους.

```csharp
public ICanvasGradient CreateRadialGradient(double x0, double y0, double r0, double x1, double y1, 
    double r1)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x0 | Double | Ο άξονας x της συντεταγμένης του κύκλου έναρξης. |
| y0 | Double | Ο άξονας y της συντεταγμένης του κύκλου έναρξης |
| r0 | Double | Η ακτίνα του κύκλου έναρξης. |
| x1 | Double | Ο άξονας x της συντεταγμένης του ακραίου κύκλου. |
| y1 | Double | Ο άξονας y της συντεταγμένης του ακραίου κύκλου. |
| r1 | Double | Η ακτίνα του ακραίου κύκλου. |

### Επιστρεφόμενη Αξία

Ένα ακτινωτό CanvasGradient αρχικοποιημένο με τους δύο καθορισμένους κύκλους.
