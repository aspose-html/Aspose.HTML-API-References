## FillText(string, double, double, double) {#filltext_1}

Σχεδιάζει (γεμίζει) ένα δεδομένο κείμενο στη δεδομένη θέση (x,y).

```csharp
public void FillText(string text, double x, double y, double maxWidth)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| text | String | Το κείμενο που θα σχεδιάσετε χρησιμοποιώντας τις τρέχουσες τιμές γραμματοσειράς, textAlign, textBaseline και κατεύθυνσης. |
| x | Double | Ο άξονας x της συντεταγμένης για το σημείο εκκίνησης του κειμένου. |
| y | Double | Ο άξονας y της συντεταγμένης για το σημείο εκκίνησης του κειμένου. |
| maxWidth | Double | Το μέγιστο πλάτος για σχεδίαση. Εάν καθορίζεται, και η συμβολοσειρά υπολογίζεται ότι είναι ευρύτερη από αυτό το πλάτος, η γραμματοσειρά προσαρμόζεται ώστε να χρησιμοποιεί μια πιο οριζόντια συμπυκνωμένη γραμματοσειρά (εάν είναι διαθέσιμη ή εάν μια εύλογα αναγνώσιμη μπορεί να συντεθεί κλιμακώνοντας την τρέχουσα γραμματοσειρά οριζόντια) ή μικρότερη γραμματοσειρά. |
