## DrawImage(HTMLImageElement, double, double, double, double, double, double, double, double) {#drawimage_5}

Σχεδιάζει την καθορισμένη εικόνα.

```csharp
public void DrawImage(HTMLImageElement image, double sx, double sy, double sw, double sh, 
    double dx, double dy, double dw, double dh)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| image | HTMLImageElement | Το HTMLImageElement για να ενταχθεί στο περιβάλλον. |
| sx | Double | Η συντεταγμένη X της επάνω αριστερής γωνίας του υπο-ορθογώνου της εικόνας προέλευσης για να ενταχθεί στο περιβάλλον προορισμού. |
| sy | Double | Η συντεταγμένη Y της επάνω αριστερής γωνίας του υπο-ορθογώνου της εικόνας προέλευσης για να ενταχθεί στο περιβάλλον προορισμού. |
| sw | Double | Το πλάτος του υπο-παραλληλογράμμου της εικόνας προέλευσης που πρέπει να ενταχθεί στο πλαίσιο προορισμού. Εάν δεν καθορίζεται, χρησιμοποιείται ολόκληρο το ορθογώνιο από τις συντεταγμένες που καθορίζονται από τα sx και sy έως την κάτω δεξιά γωνία της εικόνας. |
| sh | Double | Το ύψος του υπο-παραλληλογράμμου της εικόνας προέλευσης που πρέπει να ενταχθεί στο περιβάλλον προορισμού. |
| dx | Double | Η συντεταγμένη X στον καμβά προορισμού στον οποίο θα τοποθετηθεί η επάνω αριστερή γωνία της εικόνας προέλευσης. |
| dy | Double | Η συντεταγμένη Y στον καμβά προορισμού στον οποίο θα τοποθετηθεί η επάνω αριστερή γωνία της εικόνας προέλευσης. |
| dw | Double | Το πλάτος για να σχεδιάσετε την εικόνα στον καμβά προορισμού. Αυτό επιτρέπει την κλιμάκωση της σχεδιασμένης εικόνας. Εάν δεν καθορίζεται, η εικόνα δεν κλιμακώνεται σε πλάτος όταν σχεδιάζεται. |
| dh | Double | Το ύψος για να σχεδιάσετε την εικόνα στον καμβά προορισμού. Αυτό επιτρέπει την κλιμάκωση της σχεδιασμένης εικόνας. Εάν δεν καθορίζεται, η εικόνα δεν κλιμακώνεται σε ύψος όταν σχεδιάζεται. |
