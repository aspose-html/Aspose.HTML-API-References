## IImageData interface

Δημιουργεί ένα αντικείμενο ImageData από ένα δεδομένο Uint8ClampedArray και το μέγεθος της εικόνας που περιέχει. Εάν δεν δίνεται πίνακας, δημιουργείται μια εικόνα ενός μαύρου ορθογωνίου.

```csharp
public interface IImageData
```
