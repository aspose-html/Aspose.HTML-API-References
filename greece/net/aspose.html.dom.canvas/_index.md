Το **Aspose.Html.Dom.Canvas** Το namespace παρέχει interfaces για πιστοποίηση λεπτομερειών απόδοσης 2D, καθώς και επιλογές για πλήρωση και κατηγορία Path2D.

## Τάξεις

| Τάξη | Περιγραφή |
| --- | --- |
| [Path2D](./path2d/) | Η διεπαφή Path2D του Canvas 2D API χρησιμοποιείται για τη δήλωση διαδρομών που στη συνέχεια χρησιμοποιούνται αργότερα σε αντικείμενα CanvasRenderingContext2D. Οι μέθοδοι διαδρομής της διεπαφής CanvasRenderingContext2D υπάρχουν και σε αυτήν τη διεπαφή και σας επιτρέπουν να δημιουργήσετε διαδρομές που μπορείτε να διατηρήσετε και να αναπαράγετε ξανά όπως απαιτείται σε έναν καμβά. |
## Διεπαφές

| Διεπαφή | Περιγραφή |
| --- | --- |
| [ICanvasDrawingStyles](./icanvasdrawingstyles/) | Η διεπαφή ICanvasDrawingStyles παρέχει μεθόδους και ιδιότητες ελέγχου του τρόπου σχεδίασης των γραμμών και της διάταξης του κειμένου. |
| [ICanvasGradient](./icanvasgradient/) | Αντιπροσωπεύει ένα αδιαφανές αντικείμενο που περιγράφει μια κλίση. |
| [ICanvasPathMethods](./icanvaspathmethods/) | Η διεπαφή ICanvasPathMethods χρησιμοποιείται για τον χειρισμό μονοπατιών αντικειμένων. |
| [ICanvasPattern](./icanvaspattern/) | Αντιπροσωπεύει ένα αδιαφανές αντικείμενο που περιγράφει ένα μοτίβο, βασισμένο σε μια εικόνα, έναν καμβά ή ένα βίντεο. |
| [ICanvasRenderingContext2D](./icanvasrenderingcontext2d/) | Η διεπαφή ICanvasRenderingContext2D χρησιμοποιείται για τη σχεδίαση ορθογωνίων, κειμένου, εικόνων και άλλων αντικειμένων στο στοιχείο καμβά. Παρέχει το περιβάλλον απόδοσης 2D για την επιφάνεια σχεδίασης ενός στοιχείου καμβά. |
| [IImageData](./iimagedata/) | Δημιουργεί ένα αντικείμενο ImageData από ένα δεδομένο Uint8ClampedArray και το μέγεθος της εικόνας που περιέχει. Εάν δεν δίνεται πίνακας, δημιουργείται μια εικόνα ενός μαύρου ορθογωνίου. |
| [ITextMetrics](./itextmetrics/) | Αντιπροσωπεύει τη διάσταση ενός κειμένου στον καμβά. |
