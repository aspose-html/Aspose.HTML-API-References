## Ellipse(double, double, double, double, double, double, double) {#ellipse}

Προσθέτει μια έλλειψη στη διαδρομή που είναι κεντραρισμένη στη θέση (x, y) με την ακτίνα ακτίναςX και ακτίνα Y να ξεκινά από startAngle και να τελειώνει στο endAngle πηγαίνει προς τη δεδομένη κατεύθυνση αριστερόστροφα (από προεπιλογή προς τη φορά των δεικτών του ρολογιού).

```csharp
public void Ellipse(double x, double y, double radiusX, double radiusY, double rotation, 
    double startAngle, double endAngle)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x | Double | Ο άξονας x της συντεταγμένης για το κέντρο της έλλειψης. |
| y | Double | Ο άξονας y της συντεταγμένης για το κέντρο της έλλειψης. |
| radiusX | Double | Η ακτίνα του κύριου άξονα της έλλειψης. |
| radiusY | Double | Η ακτίνα του δευτερεύοντος άξονα της έλλειψης. |
| rotation | Double | Η περιστροφή για αυτήν την έλλειψη, εκφρασμένη σε ακτίνια. |
| startAngle | Double | Το σημείο εκκίνησης, μετρημένο από τον άξονα x, από τον οποίο θα αντληθεί, εκφρασμένο σε ακτίνια. |
| endAngle | Double | Η γωνία της ακραίας έλλειψης προς την οποία θα τραβηχτεί, εκφρασμένη σε ακτίνια. |
