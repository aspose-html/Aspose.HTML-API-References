## Path2D.ArcTo method

Προσθέτει ένα τόξο στη διαδρομή με τα δεδομένα σημεία ελέγχου και ακτίνα, συνδεδεμένο με το προηγούμενο σημείο με ευθεία γραμμή.

```csharp
public void ArcTo(double x1, double y1, double x2, double y2, double radius)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| x1 | Double | συντεταγμένες του άξονα x για το πρώτο σημείο ελέγχου. |
| y1 | Double | συντεταγμένες του άξονα y για το πρώτο σημείο ελέγχου. |
| x2 | Double | συντεταγμένες του άξονα x για το δεύτερο σημείο ελέγχου. |
| y2 | Double | συντεταγμένες του άξονα y για το δεύτερο σημείο ελέγχου. |
| radius | Double | Η ακτίνα του τόξου. |
