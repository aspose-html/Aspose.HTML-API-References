## ICSSImportRule interface

Η διεπαφή CSSImportRule αντιπροσωπεύει έναν κανόνα @import μέσα σε ένα φύλλο στυλ CSS. Ο κανόνας @import χρησιμοποιείται για την εισαγωγή κανόνων στυλ από άλλα φύλλα στυλ.

```csharp
public interface ICSSImportRule : ICSSRule
```

| Ονομα | Περιγραφή |
| --- | --- |
| [Href](../../aspose.html.dom.css/icssimportrule/href/) { get; } | Η θέση του φύλλου στυλ που θα εισαχθεί. Το χαρακτηριστικό δεν θα περιέχει τον προσδιοριστή "url(...)" γύρω από το URI. |
| [Media](../../aspose.html.dom.css/icssimportrule/media/) { get; } | Μια λίστα τύπων μέσων για τα οποία μπορεί να χρησιμοποιηθεί αυτό το φύλλο στυλ. |
| [StyleSheet](../../aspose.html.dom.css/icssimportrule/stylesheet/) { get; } | Το φύλλο στυλ που αναφέρεται από αυτόν τον κανόνα, εάν έχει φορτωθεί.
