## HTMLSelectElement.SelectedIndex property

Ο τακτικός δείκτης της επιλεγμένης επιλογής, ξεκινώντας από το 0. Η τιμή -1 επιστρέφεται εάν δεν έχει επιλεγεί κανένα στοιχείο. Εάν έχουν επιλεγεί πολλές επιλογές , επιστρέφεται το ευρετήριο της πρώτης επιλεγμένης επιλογής.

```csharp
public int SelectedIndex { get; set; }
```
