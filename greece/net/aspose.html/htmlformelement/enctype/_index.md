## HTMLFormElement.Enctype property

Ο τύπος περιεχομένου της υποβληθείσας φόρμας, γενικά "application/x-www-form-urlencoded". Δείτε τον ορισμό του χαρακτηριστικού enctype στην HTML 4.01. Ο χειριστής onsubmit even δεν είναι εγγυημένος ότι θα ενεργοποιηθεί κατά την επίκληση αυτής της μεθόδου. Η συμπεριφορά είναι ασυνεπής για ιστορικούς λόγους και οι συγγραφείς δεν πρέπει να βασίζονται σε ένα συγκεκριμένο .

```csharp
public string Enctype { get; set; }
```
