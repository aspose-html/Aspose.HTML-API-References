## HTMLTextAreaElement.Value property

Αντιπροσωπεύει τα τρέχοντα περιεχόμενα του αντίστοιχου στοιχείου ελέγχου φόρμας, σε έναν διαδραστικό παράγοντα χρήστη. Η αλλαγή αυτού του χαρακτηριστικού αλλάζει τα περιεχόμενα του στοιχείου ελέγχου φόρμας, αλλά δεν αλλάζει τα περιεχόμενα του στοιχείου . Εάν το σύνολο των δεδομένων δεν χωράει σε ένα `DOMString` , η υλοποίηση μπορεί να περικόψει τα δεδομένα.

```csharp
public string Value { get; set; }
```
