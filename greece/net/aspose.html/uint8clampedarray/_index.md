## Uint8ClampedArray class

Αντιπροσωπεύει έναν πίνακα 8-bit ανυπόγραφων ακεραίων αριθμών με σύσφιξη 0-255. Εάν καθορίσατε μια τιμή που είναι εκτός του εύρους των [0,255], θα οριστεί 0 ή 255.

```csharp
public class Uint8ClampedArray : TypedArray<byte>
```
