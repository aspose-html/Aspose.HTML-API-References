## IHTMLOptionsCollection interface

Αν`HTMLOptionsCollection` είναι μια λίστα κόμβων που αντιπροσωπεύουν το στοιχείο επιλογής HTML . Ένας μεμονωμένος κόμβος μπορεί να είναι προσβάσιμος είτε από τον τακτικό δείκτη είτε από τον κόμβο`όνομα` ή`ταυτότητα` γνωρίσματα. Οι συλλογές στο HTML DOM θεωρείται ότι είναι ζωντανές, πράγμα που σημαίνει ότι ενημερώνονται αυτόματα όταν αλλάζει το υποκείμενο έγγραφο.

Δείτε επίσης το[Μοντέλο αντικειμένου εγγράφου (DOM) Επίπεδο 2 Προδιαγραφή HTML](http://www.w3.org/TR/2003/REC-DOM-Level-2-HTML-20030109). @since DOM Level 2

```csharp
public interface IHTMLOptionsCollection : IEnumerable<Element>
```
