## HTMLDocument.Images property

Μια συλλογή από όλα τα`IMG` στοιχεία σε ένα έγγραφο. Η συμπεριφορά περιορίζεται σε`IMG` στοιχεία για συμβατότητα προς τα πίσω . Όπως προτείνεται από [[HTML 4.01](http://www.w3.org/TR/1999/REC-html401-19991224)], για να συμπεριλάβουν εικόνες, οι συγγραφείς μπορούν να χρησιμοποιήσουν το`ΑΝΤΙΚΕΙΜΕΝΟ` στοιχείο ή το`IMG` στοιχείο. Επομένως, συνιστάται να μην χρησιμοποιείτε αυτό το χαρακτηριστικό για να βρείτε τις εικόνες στο έγγραφο, αλλά`getElementsByTagName` με HTML 4.01 ή`getElementsByTagNameNS` με XHTML 1.0.

```csharp
public HTMLCollection Images { get; }
```
