## HTMLDocument.Referrer property

Επιστρέφει το URI [[IETF RFC 2396](http://www.ietf.org/rfc/rfc2396.txt) της σελίδας που συνδέθηκε με αυτήν τη σελίδα. Η τιμή είναι μια κενή συμβολοσειρά εάν ο χρήστης πλοηγήθηκε απευθείας στη σελίδα (όχι μέσω ενός συνδέσμου, αλλά, για παράδειγμα, μέσω ενός σελιδοδείκτη).

```csharp
public string Referrer { get; }
```
