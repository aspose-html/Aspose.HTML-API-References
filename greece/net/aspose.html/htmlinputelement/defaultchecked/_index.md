## HTMLInputElement.DefaultChecked property

Πότε`τύπος` έχει την τιμή "radio" ή "checkbox", αυτό το αντιπροσωπεύει το χαρακτηριστικό HTML checked του στοιχείου. Η τιμή του αυτού του χαρακτηριστικού δεν αλλάζει εάν αλλάξει η κατάσταση του αντίστοιχου στοιχείου ελέγχου φόρμας , σε έναν διαδραστικό παράγοντα χρήστη. Δείτε τον επιλεγμένο ορισμό του χαρακτηριστικού στην HTML 4.01.

```csharp
public bool DefaultChecked { get; set; }
```
