## HTMLInputElement.Checked property

Όταν το`τύπος`Το χαρακτηριστικό του στοιχείου έχει την τιμή "radio" ή "checkbox", αυτό αντιπροσωπεύει την τρέχουσα κατάσταση του στοιχείου ελέγχου , σε έναν διαδραστικό παράγοντα χρήστη. Οι αλλαγές σε αυτό το χαρακτηριστικό αλλάζουν την κατάσταση του στοιχείου ελέγχου φόρμας, αλλά δεν αλλάζουν την τιμή του του χαρακτηριστικού που ελέγχεται με HTML του στοιχείου INPUT. Κατά τον χειρισμό ενός γεγονότος κλικ σε ένα στοιχείο εισόδου με χαρακτηριστικό τύπου που έχει τιμή "radio" ή "checkbox", ορισμένες υλοποιήσεις ενδέχεται να αλλάξουν την τιμή αυτής της ιδιότητας πριν από την αποστολή του συμβάντος στο έγγραφο . Εάν η προεπιλεγμένη ενέργεια του συμβάντος ακυρωθεί, η τιμή της ιδιότητας μπορεί να αλλάξει ξανά στην αρχική της τιμή. Αυτό σημαίνει ότι η τιμή αυτής της ιδιότητας κατά τον χειρισμό συμβάντων κλικ εξαρτάται από την υλοποίηση.

```csharp
public bool Checked { get; set; }
```
