## HTMLInputElement.DefaultValue property

Όταν το`τύπος`Το χαρακτηριστικό του στοιχείου έχει την τιμή "text", "file" ή "password", αυτό αντιπροσωπεύει την τιμή HTML χαρακτηριστικό του στοιχείου. Η τιμή αυτού του χαρακτηριστικού δεν αλλάζει εάν αλλάξουν τα περιεχόμενα του αντίστοιχου στοιχείου ελέγχου φόρμας, σε έναν διαδραστικό παράγοντα χρήστη . Δείτε τον ορισμό του χαρακτηριστικού τιμής στην HTML 4.01.

```csharp
public string DefaultValue { get; set; }
```
