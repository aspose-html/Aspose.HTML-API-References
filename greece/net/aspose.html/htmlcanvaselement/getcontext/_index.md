## HTMLCanvasElement.GetContext method

Επιστρέφει ένα περιβάλλον σχεδίασης στον καμβά ή μηδενικό εάν το αναγνωριστικό περιβάλλοντος δεν υποστηρίζεται. Ένα πλαίσιο σχεδίασης σάς επιτρέπει να σχεδιάζετε στον καμβά.

```csharp
public object GetContext(string contextId)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| contextId | String | Είναι μια συμβολοσειρά που περιέχει το αναγνωριστικό περιβάλλοντος που ορίζει το πλαίσιο σχεδίασης που σχετίζεται με τον καμβά |
