Το **Aspose.Html.Αποθήκευση** Ο χώρος ονομάτων παρουσιάζεται από κλάσεις δεδομένων για περιγραφή συγκεκριμένων επιλογών αποθήκευσης στο διαδικασία μετατροπής/αποθήκευσης.

## Τάξεις

| Τάξη | Περιγραφή |

| [HTMLSaveOptions](./htmlsaveoptions/) | Αντιπροσωπεύει επιλογές αποθήκευσης HTML. |
| [ImageSaveOptions](./imagesaveoptions/) | Κατηγορία δεδομένων ειδικών επιλογών. |
| [MarkdownSaveOptions](./markdownsaveoptions/) | Αντιπροσωπεύει τις επιλογές αποθήκευσης Markdown. |
| [MHTMLSaveOptions](./mhtmlsaveoptions/) | Αντιπροσωπεύει επιλογές αποθήκευσης MHTML. |
| [PdfSaveOptions](./pdfsaveoptions/) | Κατηγορία δεδομένων ειδικών επιλογών. |
| [ResourceHandlingOptions](./resourcehandlingoptions/) | Αντιπροσωπεύει επιλογές χειρισμού πόρων. |
| [SaveOptions](./saveoptions/) | Αυτή είναι μια αφηρημένη βασική κλάση για κλάσεις που επιτρέπουν στο χρήστη να καθορίσει πρόσθετες επιλογές κατά την αποθήκευση ενός εγγράφου σε μια συγκεκριμένη μορφή. |
| [XpsSaveOptions](./xpssaveoptions/) | Κατηγορία δεδομένων ειδικών επιλογών. |
