## MarkdownFeatures enumeration

Α`MarkdownFeatures` Το σύνολο σημαιών είναι ένα σύνολο μηδέν ή περισσότερων από τις ακόλουθες σημαίες, οι οποίες χρησιμοποιούνται για την επιλογή στοιχείων που έχουν μετατραπεί σε σήμανση.

```csharp
[Flags]
public enum MarkdownFeatures
```

### Αξίες

| Ονομα | αξία | Περιγραφή |
| --- | --- | --- |
| InlineHTML | `1` | Αυτή η σημαία επιτρέπει την ενσωμάτωση στοιχείων HTML. Εάν αυτή η σημαία έχει οριστεί από στοιχεία επιπέδου μπλοκ (όπως π.χ`div` ) του οποίου`χαμήλωση τιμής` τιμή χαρακτηριστικού ισούται`στη γραμμή` θα εισαχθεί στη σήμανση που προκύπτει. |
| AutomaticParagraph | `2` | Αυτή η σημαία επιτρέπει τη μετατροπή του`παράγραφος` στοιχεία. Το περιεχόμενο τέτοιων στοιχείων θα τοποθετηθεί σε ξεχωριστές γραμμές, έτσι οι χειριστές σήμανσης θα το τυλίξουν. |
| Header | `4` | Αυτή η σημαία επιτρέπει τη μετατροπή του`επί κεφαλής` στοιχεία. |
| Blockquote | `8` | Αυτή η σημαία επιτρέπει τη μετατροπή του`μπλοκ` στοιχεία. |
| List | `10` | Αυτή η σημαία επιτρέπει τη μετατροπή του`λίστα` στοιχεία. |
| CodeBlock | `20` | Αυτή η σημαία επιτρέπει τη μετατροπή μπλοκ κώδικα. Το μπλοκ κώδικα αποτελείται από 2 στοιχεία`προ` και`κώδικας` , το περιεχόμενο μιας τέτοιας κατασκευής είναι διαδικασίες "ως έχουν". |
| HorizontalRule | `40` | Αυτή η σημαία επιτρέπει τη μετατροπή του`οριζόντιους κανόνες` . |
| Link | `80` | Αυτή η σημαία επιτρέπει τη μετατροπή του`ένα` στοιχεία. |
| Emphasis | `100` | Αυτή η σημαία επιτρέπει τη μετατροπή του`έμφαση` στοιχεία. |
| InlineCode | `200` | Αυτή η σημαία επιτρέπει τη μετατροπή του`κώδικας` στοιχεία. |
| Image | `400` | Αυτή η σημαία επιτρέπει τη μετατροπή του`img` στοιχεία. |
| LineBreak | `800` | Αυτή η σημαία επιτρέπει τη μετατροπή του`br` στοιχεία. |
| Video | `1000` | Αυτή η σημαία επιτρέπει τη μετατροπή του`βίντεο` στοιχεία. |
| Table | `2000` | Αυτή η σημαία επιτρέπει τη μετατροπή του`τραπέζι` στοιχεία. |
| TaskList | `4000` | Αυτή η σημαία επιτρέπει τη μετατροπή λιστών εργασιών. Η λίστα εργασιών αποτελείται από`εισαγωγή` στοιχείο, το οποίο πρέπει να είναι το πρώτο παιδί του`λίστα` στοιχείο και του οποίου`τύπος` η τιμή του χαρακτηριστικού πρέπει να είναι ίση`πλαίσιο ελέγχου` . |
| Strikethrough | `8000` | Αυτή η σημαία επιτρέπει τη μετατροπή του`del` στοιχεία. |
| Strong | `10000` | Αυτή η σημαία επιτρέπει τη μετατροπή του`ισχυρός` στοιχεία. |
