## SVGMarkerElement.OrientAngle property

Αντιστοιχεί στο χαρακτηριστικό 'orient' στο δεδομένο στοιχείο 'marker'. Εάν το markerUnits είναι SVG_MARKER_ORIENT_ANGLE, η τιμή γωνίας για το χαρακτηριστικό 'orient'. διαφορετικά, θα οριστεί στο μηδέν.

```csharp
public SVGAnimatedAngle OrientAngle { get; }
```

### Αξία περιουσίας

Το χαρακτηριστικό 'orient' του δεδομένου στοιχείου 'marker'.
