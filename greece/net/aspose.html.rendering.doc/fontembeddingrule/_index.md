## FontEmbeddingRule enumeration

Αντιπροσωπεύει τους κανόνες ενσωμάτωσης γραμματοσειράς.

```csharp
public enum FontEmbeddingRule
```

### Αξίες

| Ονομα | αξία | Περιγραφή |
| --- | --- | --- |
| Full | `1` | Όλες οι γραμματοσειρές που χρησιμοποιούνται θα ενσωματωθούν ως έχουν. Λάβετε υπόψη ότι κάποια γραμματοσειρά θα μπορούσε να έχει άδεια χρήσης, αυτό θα καταστήσει το έγγραφο που θα προκύψει μη επεξεργάσιμο. Επίσης τα ενσωματωμένα αρχεία γραμματοσειράς θα μπορούσαν να αυξήσουν σημαντικά το μέγεθος του εγγράφου που προκύπτει. |
| None | `2` | Οι γραμματοσειρές δεν θα ενσωματωθούν. |
