## ErrorEvent.LineNo property

Το χαρακτηριστικό lineno πρέπει να επιστρέψει την τιμή στην οποία είχε αρχικοποιηθεί. Όταν δημιουργείται το αντικείμενο, αυτό το χαρακτηριστικό πρέπει να αρχικοποιηθεί στο μηδέν. Αντιπροσωπεύει τον αριθμό γραμμής όπου παρουσιάστηκε το σφάλμα στο σενάριο.

```csharp
public uint LineNo { get; }
```
