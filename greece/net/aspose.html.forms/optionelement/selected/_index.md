## OptionElement.Selected property

Αντιπροσωπεύει την τρέχουσα κατάσταση του αντίστοιχου στοιχείου ελέγχου φόρμας, σε έναν διαδραστικό παράγοντα χρήστη. Η αλλαγή αυτού του χαρακτηριστικού αλλάζει την κατάσταση του στοιχείου ελέγχου φόρμας, αλλά δεν αλλάζει την τιμή του επιλεγμένου χαρακτηριστικού HTML του στοιχείου.

```csharp
public bool Selected { get; set; }
```
