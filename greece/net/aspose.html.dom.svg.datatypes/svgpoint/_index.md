## SVGPoint class

Πολλές από τις διεπαφές SVG DOM αναφέρονται σε αντικείμενα της κλάσης SVGPoint. Ένα SVGPoint είναι ένα ζεύγος συντεταγμένων (x, y). Όταν χρησιμοποιείται σε λειτουργίες μήτρας, ένα SVGPoint αντιμετωπίζεται ως διάνυσμα της μορφής: [x] [y] [1] Εάν ένα αντικείμενο SVGRect οριστεί ως μόνο για ανάγνωση, τότε η προσπάθεια εκχώρησης σε ένα από τα χαρακτηριστικά του θα έχει ως αποτέλεσμα την εξαίρεση.

```csharp
public class SVGPoint : SVGValueType
```
