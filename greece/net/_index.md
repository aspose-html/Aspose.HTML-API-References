| [Aspose.Html](./aspose.html/) | Το **Aspose.Html** Ο χώρος ονομάτων περιέχει κλάσεις και μεθόδους για χειρισμό εγγράφων HTML, είτε απλά είτε σύνθετα και εν κινήσει.  **Aspose.HTML**για το .NET επιτρέπει στους προγραμματιστές να εισάγουν, να αφαιρέσουν, να αντικαταστήσουν κόμβους HTML, να εξάγουν πληροφορίες στυλ CSS, να πλοηγηθούν στο έγγραφο HTML είτε από το NodeIterator, TreeWalker που παρέχονται από το Traversal Specifications, το XPath. ή τον επιλογέα CSS Προσφέρει επίσης το σενάριο που επιτρέπει να χειριστεί το HTML DOM μέσω JavaScript. Εκτός από το HTML, αυτό το API παρέχει επίσης τις δυνατότητες φόρτωσης EPUB και MHTML. Τα Aspose API είναι διάσημα για τις δυνατότητες μετατροπής μεταξύ των μορφών αρχείων τους και αυτό το API παρέχει επίσης τις δυνατότητες φόρτωσης αρχείου HTML και απόδοσης της εξόδου σε μορφές εικόνας PDF, XPS και ράστερ, συμπεριλαμβανομένων JPEG, PNG, BMP και TIFF. |
