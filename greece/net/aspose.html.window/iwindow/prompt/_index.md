## IWindow.Prompt method

Εμφανίζει μια προτροπή πεδίου κειμένου με το δεδομένο μήνυμα, περιμένει να το απορρίψει ο χρήστης και επιστρέφει την τιμή που εισήγαγε ο χρήστης. Εάν ο χρήστης ακυρώσει την προτροπή, τότε επιστρέφει null. Εάν υπάρχει το δεύτερο όρισμα, τότε η δεδομένη τιμή χρησιμοποιείται ως προεπιλογή.

```csharp
public string Prompt(string message, string @default)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| message | String | Το μήνυμα. |
| default | String | Η προεπιλεγμένη. |

### Επιστρεφόμενη Αξία

Επιστρέφει την τιμή που εισήγαγε ο χρήστης
