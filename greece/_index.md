## Aspose.HTML για .NET

{{% alert color="primary" %}} 

![Λογότυπο προϊόντος Aspose.HTML για .NET](../home_1.png)


Το Aspose.HTML για .NET είναι μια βιβλιοθήκη κλάσεων πολλαπλών πλατφορμών που επιτρέπει στις εφαρμογές σας να εκτελούν ένα ευρύ φάσμα εργασιών χειρισμού HTML. Δεδομένου ότι η HTML χρησιμοποιείται γενικά σε συνδυασμό με άλλες τεχνολογίες, το Aspose.HTML υποστηρίζει όχι μόνο προδιαγραφές HTML5 αλλά και CSS3, SVG και HTML Canvas. Χρησιμοποιώντας το Aspose.HTML, μπορείτε να φορτώσετε ιστοσελίδες, να αναλύσετε και να επεξεργαστείτε το περιεχόμενο, να αποθηκεύσετε και να μετατρέψετε σελίδες σε και από τις ακόλουθες μορφές: HTML, XHTML, MHTML, SVG, Markdown, PDF, XPS,
EPUB, PNG, TIFF, JPEG, BMP και άλλες μορφές.

{{% /alert %}}

## Aspose.HTML για Java

{{% alert color="primary" %}} 

![Λογότυπο προϊόντος Aspose.HTML για Java](../home_2.png)


Το Aspose.HTML για Java είναι ένα προηγμένο API χειρισμού HTML που έχει σχεδιαστεί για να εκτελεί ένα ευρύ φάσμα εργασιών χειρισμού HTML απευθείας εντός των εφαρμογών Java. Το API παρέχει επίσης μια μηχανή απόδοσης υψηλής πιστότητας για μορφές σταθερής διάταξης όπως PDF και XPS και μια σειρά από μορφές εικόνας ράστερ.

{{% /alert %}}
