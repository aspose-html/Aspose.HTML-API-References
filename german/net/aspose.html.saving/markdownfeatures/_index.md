## MarkdownFeatures enumeration

A[`MarkdownFeatures`](../markdownfeatures) Flag-Set ist ein Satz von null oder mehr der folgenden Flags, die verwendet werden, um Elemente auszuwählen, die in Markdown konvertiert werden.

```csharp
[Flags]
public enum MarkdownFeatures
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
