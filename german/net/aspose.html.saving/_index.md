Die **Aspose.Html.Speichern** Namensraum wird präsentiert von Datenklassen zur Beschreibung spezifischer Speicheroptionen bei Konvertierungs-/Speicherprozess.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [DocSaveOptions](./docsaveoptions/) | Spezifische Optionen Datenklasse. |
| [HTMLSaveOptions](./htmlsaveoptions/) | Repräsentiert HTML-Speicheroptionen. |
| [ImageSaveOptions](./imagesaveoptions/) | Spezifische Optionen Datenklasse. |
| [MarkdownSaveOptions](./markdownsaveoptions/) | Stellt Markdown-Speicheroptionen dar. |
| [MHTMLSaveOptions](./mhtmlsaveoptions/) | Stellt MHTML-Speicheroptionen dar. |
| [PdfSaveOptions](./pdfsaveoptions/) | Spezifische Optionen Datenklasse. |
| [ResourceHandlingOptions](./resourcehandlingoptions/) | Repräsentiert Ressourcenverwaltungsoptionen. |
| [SaveOptions](./saveoptions/) | Dies ist eine abstrakte Basisklasse für Klassen, die es dem Benutzer ermöglichen, zusätzliche Optionen anzugeben, wenn ein Dokument in einem bestimmten Format gespeichert wird. |
| [XpsSaveOptions](./xpssaveoptions/) | Spezifische Optionen Datenklasse. |
## Aufzählung

| Aufzählung | Beschreibung |
| --- | --- |
| [HTMLSaveFormat](./htmlsaveformat/) | Gibt das Format an, in dem das Dokument gespeichert wird. |
| [MarkdownFeatures](./markdownfeatures/) | A[`MarkdownFeatures`](../aspose.html.saving/markdownfeatures/) Flag-Set ist ein Satz von null oder mehr der folgenden Flags, die verwendet werden, um Elemente auszuwählen, die in Markdown konvertiert werden. |
| [MarkdownFormatter](./markdownformatter/) | Gibt an, wie die Ausgabe formatiert wird. |
| [ResourceHandling](./resourcehandling/) | Diese Aufzählung repräsentiert Ressourcenhandhabungsoptionen. |
| [UrlRestriction](./urlrestriction/) | Diese Aufzählung stellt eine Einschränkung dar, die auf URLs von verarbeiteten Ressourcen angewendet wird. |
