Die **Aspose.Html.Dom.Mutations** bietet die Möglichkeit, DOM-Mutationen zu beobachten und spezifische Rückrufinformationen zu erhalten.

| Klasse | Beschreibung |

| [MutationCallback](./mutationcallback/) | Stellt die Methode dar, die ein Mutationsereignis behandelt. |
| [MutationObserver](./mutationobserver/) | A[`MutationObserver`](../aspose.html.dom.mutations/mutationobserver/) Objekt kann verwendet werden, um Mutationen im Baum von zu beobachten[`Node`](../aspose.html.dom/node/) . |
| [MutationObserverInit](./mutationobserverinit/) | Diese Klasse stellt eine Optionssammlung dar, die zum Konfigurieren verwendet wird[`MutationObserver`](../aspose.html.dom.mutations/mutationobserver/) . |
| [MutationRecord](./mutationrecord/) | Ein MutationRecord repräsentiert eine einzelne DOM-Mutation. Es ist das Objekt, an das übergeben wird[`MutationObserver`](../aspose.html.dom.mutations/mutationobserver/) S[`MutationCallback`](../aspose.html.dom.mutations/mutationcallback/) . |
