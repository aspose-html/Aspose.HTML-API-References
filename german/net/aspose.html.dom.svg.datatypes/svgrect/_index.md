## SVGRect class

Steht für rechteckige Geometrie. Rechtecke bestehen aus einem (x,y)-Koordinatenpaar, das einen minimalen X-Wert, einen minimalen Y-Wert und eine Breite und Höhe identifiziert, die normalerweise nicht negativ sein dürfen.
