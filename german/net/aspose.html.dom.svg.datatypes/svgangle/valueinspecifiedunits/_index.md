## SVGAngle.ValueInSpecifiedUnits property

Der Winkelwert als Fließkommawert in den Einheiten, ausgedrückt durch unitType. Das Festlegen dieses Attributs bewirkt, dass value und valueAsString automatisch aktualisiert werden, um diese Einstellung widerzuspiegeln.

```csharp
public float ValueInSpecifiedUnits { get; set; }
```

### Eigentumswert

Der Wert in angegebenen Einheiten.
