## SVGAngle.ValueAsString property

Der Winkelwert als Zeichenfolgenwert in den durch unitType ausgedrückten Einheiten. Das Festlegen dieses Attributs bewirkt, dass value, valueInSpecifiedUnits und unitType automatisch aktualisiert werden, um diese Einstellung widerzuspiegeln.

```csharp
public string ValueAsString { get; set; }
```

### Eigentumswert

Der Wert als Zeichenfolge.
