## TemplateLoadOptions.NamesAreCaseSensitive property

Eigenschaft, die angibt, ob die Vorlagen- und Datenelementnamen unabhängig von der Groß-/Kleinschreibung abgeglichen werden oder nicht. Wenn wahr, wird beim Abgleich zwischen Groß- und Kleinschreibung unterschieden; wenn falsch, wird die Groß-/Kleinschreibung ignoriert. Der Standardwert ist wahr.

```csharp
public bool NamesAreCaseSensitive { get; set; }
```
