## OptionElement.Selected property

Repräsentiert den aktuellen Status des entsprechenden Formularsteuerelements in einem interaktiven Benutzeragenten. Das Ändern dieses Attributs ändert den Status des Formularsteuerelements, ändert jedoch nicht den Wert des HTML-Attributs selected des Elements.

```csharp
public bool Selected { get; set; }
```
