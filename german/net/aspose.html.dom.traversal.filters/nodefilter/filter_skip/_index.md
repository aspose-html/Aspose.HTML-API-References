## NodeFilter.FILTER_SKIP field

Überspringen Sie diesen einzelnen Knoten. Für NodeIterator oder TreeWalker definierte Navigationsmethoden geben diesen Knoten nicht zurück. Sowohl für NodeIterator als auch für TreeWalker werden die Kinder dieses Knotens immer noch berücksichtigt.

```csharp
public const short FILTER_SKIP;
```
