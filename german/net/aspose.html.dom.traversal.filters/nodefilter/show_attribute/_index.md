## NodeFilter.SHOW_ATTRIBUTE field

Attr-Knoten anzeigen. Dies ist nur sinnvoll, wenn ein -Iterator oder Tree-Walker mit einem Attributknoten als -Wurzel erstellt wird; in diesem Fall bedeutet dies, dass der Attributknoten an der ersten Position der Iteration oder Traversierung erscheint. Da Attribute nie Kinder anderer Knoten sind, erscheinen sie nicht, wenn der Dokumentenbaum durchlaufen wird.

```csharp
public const long SHOW_ATTRIBUTE;
```
