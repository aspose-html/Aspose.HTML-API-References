## CanvasFillRule enumeration

Diese Aufzählung wird verwendet, um den Füllregelalgorithmus auszuwählen, mit dem bestimmt wird, ob sich ein Punkt innerhalb oder außerhalb eines Pfads befindet.

```csharp
public enum CanvasFillRule
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| NonZero | `0` | Der Wert "Nicht-Null" gibt die Nicht-Null-Wicklungsregel an, bei der ein Punkt als außerhalb einer -Form liegend betrachtet wird, wenn eine halb-unendliche gerade Linie, die von diesem Punkt aus gezogen wird, den Pfad der Form in einer Richtung kreuzt ist gleich der Anzahl, wie oft er den Weg in die andere Richtung kreuzt. |
| EvenOdd | `1` | Der Wert „gerade ungerade“ gibt die Gerade-Ungerade-Regel an, bei der ein Punkt als außerhalb einer Form liegend angesehen wird, wenn die Anzahl von Malen, mit der eine halb unendliche gerade Linie, die von diesem Punkt gezogen wird, den Pfad der Form kreuzt, gerade ist. |
