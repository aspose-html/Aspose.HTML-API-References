## IImageData interface

Erstellt ein ImageData-Objekt aus einem gegebenen Uint8ClampedArray und der Größe des darin enthaltenen Bildes. Wenn kein Array angegeben ist, wird ein Bild eines schwarzen Rechtecks erstellt.
