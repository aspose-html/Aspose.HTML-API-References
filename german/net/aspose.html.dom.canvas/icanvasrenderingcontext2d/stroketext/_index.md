## StrokeText(string, double, double, double?) {#stroketext_1}

Zeichnet (Striche) einen gegebenen Text an der gegebenen (x, y) Position.

```csharp
public void StrokeText(string text, double x, double y, double? maxWidth)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| text | String | Der Text, der mit den aktuellen Werten für Schriftart, textAlign, textBaseline und Richtung gezeichnet werden soll. |
| x | Double | Die x-Achse der Koordinate für den Textstartpunkt. |
| y | Double | Die y-Achse der Koordinate für den Textstartpunkt. |
| maxWidth | Nullable`1 | Die maximale zu zeichnende Breite. Wenn angegeben und die Zeichenfolge breiter als diese Breite berechnet wird, wird die Schriftart angepasst, um eine horizontal stärker komprimierte Schriftart (falls verfügbar oder wenn eine einigermaßen lesbare Schriftart durch horizontales Skalieren der aktuellen Schriftart synthetisiert werden kann) oder eine kleinere zu verwenden Schriftart. |
