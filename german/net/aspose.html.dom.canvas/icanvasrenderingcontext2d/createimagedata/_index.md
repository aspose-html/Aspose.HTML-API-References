## CreateImageData(double, double) {#createimagedata_1}

Erstellt ein neues, leeres ImageData-Objekt mit den angegebenen Abmessungen. Alle Pixel im neuen Objekt sind transparent schwarz.

```csharp
public IImageData CreateImageData(double sw, double sh)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| sw | Double | Die Breite für das neue ImageData-Objekt. |
| sh | Double | Die Höhe, die dem neuen ImageData-Objekt zugewiesen werden soll. |

### Rückgabewert

Ein neues ImageData-Objekt mit der angegebenen Breite und Höhe. Das neue Objekt wird mit transparenten schwarzen Pixeln gefüllt.
