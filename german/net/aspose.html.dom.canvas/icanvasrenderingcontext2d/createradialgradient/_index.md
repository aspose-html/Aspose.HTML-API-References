## ICanvasRenderingContext2D.CreateRadialGradient method

Erstellt einen radialen Farbverlauf, der durch die Koordinaten der beiden durch die Parameter repräsentierten Kreise gegeben ist.

```csharp
public ICanvasGradient CreateRadialGradient(double x0, double y0, double r0, double x1, double y1, 
    double r1)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| x0 | Double | Die x-Achse der Koordinate des Startkreises. |
| y0 | Double | Die y-Achse der Koordinate des Startkreises |
| r0 | Double | Der Radius des Startkreises. |
| x1 | Double | Die x-Achse der Koordinate des Endkreises. |
| y1 | Double | Die y-Achse der Koordinate des Endkreises. |
| r1 | Double | Der Radius des Endkreises. |

### Rückgabewert

Ein radialer CanvasGradient, der mit den beiden angegebenen Kreisen initialisiert wurde.
