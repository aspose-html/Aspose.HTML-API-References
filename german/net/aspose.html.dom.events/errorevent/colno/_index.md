## ErrorEvent.ColNo property

Das colno-Attribut muss den Wert zurückgeben, mit dem es initialisiert wurde. Wenn das Objekt erstellt wird, muss dieses Attribut auf Null initialisiert werden. Es stellt die Spaltennummer dar, in der der Fehler im Skript aufgetreten ist.

```csharp
public uint ColNo { get; }
```
