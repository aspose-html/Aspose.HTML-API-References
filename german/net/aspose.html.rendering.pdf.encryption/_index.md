Die **Aspose.Html.Rendering.Pdf.Encryption**Namespace enthält Datenklassen, die PDF-Verschlüsselungsdetails wie Passwort, Verschlüsselungsalgorithmus usw. beschreiben.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [PdfEncryptionInfo](./pdfencryptioninfo/) | Enthält Details für eine PDF-Verschlüsselung. |
## Aufzählung

| Aufzählung | Beschreibung |
| --- | --- |
| [PdfEncryptionAlgorithm](./pdfencryptionalgorithm/) | Verschlüsselungsmodus enum. Beschreiben Sie anhand von Algorithmus und Schlüssellänge. Dieses Enum wird erweitert, um die Funktionalität weiter erhöhen zu können. Dieses Enum implementiert das "Base-to-Core"-Muster. |
| [PdfPermissions](./pdfpermissions/) | Diese Aufzählung repräsentiert die Berechtigungen des Benutzers für ein PDF. |
