## SVGSVGElement.CreateSVGTransformFromMatrix method

Erstellt ein SVGTransform-Objekt außerhalb von Dokumentbäumen. Das Objekt wird mit der gegebenen Matrixtransformation (dh SVG_TRANSFORM_MATRIX) initialisiert. Die Werte aus der Parametermatrix werden kopiert, der Matrixparameter wird nicht als SVGTransform::matrix. übernommen

