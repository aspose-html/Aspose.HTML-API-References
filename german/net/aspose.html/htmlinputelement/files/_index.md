## HTMLInputElement.Files property

Das IDL-Attribut files ermöglicht Skripten den Zugriff auf die ausgewählten Dateien des Elements. Wenn das IDL-Attribut zutrifft, muss es beim Abrufen ein FileList-Objekt zurückgeben, das die aktuell ausgewählten Dateien darstellt. Dasselbe Objekt muss zurückgegeben werden, bis sich die Liste der ausgewählten Dateien ändert. Wenn das IDL-Attribut nicht zutrifft, muss es stattdessen null zurückgeben. [FILEAPI]

```csharp
public IFileList Files { get; }
```
