## HTMLInputElement.DefaultChecked property

Wann`Typ` den Wert "radio" oder "checkbox" hat, repräsentiert dieses das HTML-checked-Attribut des Elements. Der Wert von dieses Attributs ändert sich nicht, wenn sich der Zustand des entsprechenden Formular-Steuerelements in einem interaktiven Benutzeragenten ändert. Siehe die Definition des Attributs „checked “ in HTML 4.01.

```csharp
public bool DefaultChecked { get; set; }
```
