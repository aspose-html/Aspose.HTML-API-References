## HTMLInputElement.Checked property

Wenn die`Typ`Das Attribut des Elements hat den Wert "Radio" oder "Checkbox", dies repräsentiert den aktuellen Zustand des Formular-Steuerelements in einem interaktiven Benutzeragenten. Änderungen an diesem Attribut ändern den Status des Formularsteuerelements, ändern jedoch nicht den Wert von dem HTML-geprüften Attribut des INPUT-Elements Wert "radio" oder "checkbox", können einige Implementierungen den -Wert dieser Eigenschaft ändern, bevor das Ereignis im -Dokument abgesetzt wird. Wenn die Standardaktion des Ereignisses abgebrochen wird, kann der Wert der Eigenschaft wieder auf seinen ursprünglichen Wert geändert werden. Das bedeutet, dass der Wert dieser Eigenschaft während der Verarbeitung von Klickereignissen implementierungsabhängig ist.

public bool Checked { get; set; }
