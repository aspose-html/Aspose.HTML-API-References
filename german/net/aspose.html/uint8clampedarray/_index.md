## Uint8ClampedArray class

Stellt ein Array von 8-Bit-Ganzzahlen ohne Vorzeichen dar, die auf 0-255 geklemmt sind; wenn Sie einen Wert angegeben haben, der außerhalb des Bereichs von [0,255] liegt, wird stattdessen 0 oder 255 eingestellt;
