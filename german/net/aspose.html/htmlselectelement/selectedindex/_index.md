## HTMLSelectElement.SelectedIndex property

Der Ordnungsindex der ausgewählten Option, beginnend bei 0. Der Wert -1 wird zurückgegeben, wenn kein Element ausgewählt ist. Wenn mehrere Optionen ausgewählt sind, wird der Index der ersten ausgewählten Option zurückgegeben.
