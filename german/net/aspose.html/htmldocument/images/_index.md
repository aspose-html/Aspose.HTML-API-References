## HTMLDocument.Images property

Eine Sammlung aller`IMG` Elemente in einem Dokument. Das Verhalten von ist beschränkt auf`IMG` Elemente für die Abwärtskompatibilität . Wie vorgeschlagen von [[HTML4.01](http://www.w3.org/TR/1999/REC-html401-19991224)], um Bilder einzufügen, können Autoren verwenden`OBJEKT` Element oder die`IMG` Element. Daher wird empfohlen, dieses Attribut nicht zu verwenden, um die -Bilder im Dokument zu finden, sondern`getElementsByTagName` mit HTML 4.01 bzw`getElementsByTagNameNS` mit XHTML 1.0.
