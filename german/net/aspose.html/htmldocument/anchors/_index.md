## HTMLDocument.Anchors property

Eine Sammlung aller Anker (`EIN` ) Elemente in einem Dokument mit einem Wert für die`Name`Attribut. Aus Gründen der Abwärtskompatibilität enthält der zurückgegebene Satz von Ankern nur die Anker, die mit erstellt wurden`Name` Attribut, nicht die mit dem erstellten`Ich würde` Attribut. Beachten Sie, dass in [[XHTML 1.0](http://www.w3.org/TR/2002/REC-xhtml1-20020801) ], die `Name` -Attribut (siehe Abschnitt 4.10) hat keine Semantik und ist nur für Legacy-Benutzeragenten vorhanden: the`Ich würde` stattdessen wird das Attribut verwendet. Benutzer sollten die Iterator-Mechanismen bevorzugen, die von [[DOM Level 2 Traversierung](http://www.w3.org/TR/2000/REC-DOM-Level-2-Traversal-Range-20001113) ] stattdessen.
