## HTMLCanvasElement.Height property

Ist eine positive ganze Zahl, die das HTML-Attribut height des Canvas-Elements widerspiegelt, das in CSS-Pixeln interpretiert wird. Wenn das Attribut nicht angegeben ist oder wenn es auf einen ungültigen Wert gesetzt ist, wie z. B. negativ, wird der Standardwert von 150 verwendet.
