## HTMLCanvasElement.Width property

Ist eine positive ganze Zahl, die das HTML-Attribut width des Canvas-Elements widerspiegelt, das in CSS-Pixeln interpretiert wird. Wenn das Attribut nicht angegeben ist oder auf einen ungültigen Wert gesetzt ist, z. B. negativ, wird der Standardwert 300 verwendet.
