## PageSetup.AdjustToWidestPage property

Erhält oder setzt ein Flag, das bestimmt, wann die Seitengröße auf die breiteste Seite im Dokument angepasst wird. Diese Option ist zeitaufwändig, sodass die Zeit der Dokumentverarbeitung um das Zweifache erhöht werden kann. Die Anpassung wird nur durchgeführt, wenn die breiteste Seite im Dokument breiter ist als die in festgelegte Seitengröße[`PageSetup`](../../pagesetup) . Angepasste Seitengröße wird für alle Seiten im Dokument verwendet.

public bool AdjustToWidestPage { get; set; }
