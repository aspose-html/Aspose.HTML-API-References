Die **Aspose.Html.Rendering.Doc** Namespace bietet eine spezifische Geräteklasse sowie einige wenige Rendering Optionsklassen, die für das Rendern in ein DOCX-Dokument verantwortlich sind.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [DocDevice](./docdevice/) | Stellt das Rendern in ein DOCX-Dokument dar. |
| [DocRenderingOptions](./docrenderingoptions/) | Repräsentiert die Rendering-Optionen für[`DocDevice`](../aspose.html.rendering.doc/docdevice/) . |
## Aufzählung

| Aufzählung | Beschreibung |
| --- | --- |
| [DocumentFormat](./documentformat/) | Stellt das Dateiformat des Ausgabedokuments dar. |
| [FontEmbeddingRule](./fontembeddingrule/) | Repräsentiert die Schrifteinbettungsregeln. |
