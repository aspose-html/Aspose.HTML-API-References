## ILinkStyle interface

Die LinkStyle-Schnittstelle bietet einen Mechanismus, mit dem ein Stylesheet von dem Knoten abgerufen werden kann, der für die Verknüpfung mit einem Dokument verantwortlich ist. Eine Instanz der LinkStyle-Schnittstelle kann mit bindungsspezifischen Casting-Methoden auf eine Instanz eines Verknüpfungsknotens (HTMLLinkElement, HTMLStyleElement oder ProcessingInstruction in DOM Level 2) bezogen werden.
