## TemplateLoadOptions.NamesAreCaseSensitive property

대소문자 여부에 관계없이 템플릿과 데이터 요소 이름을 일치시킬지 여부를 지정하는 속성입니다. true인 경우 일치는 대소문자를 구분합니다. false인 경우 대소문자는 무시됩니다. 기본값은 true입니다.

```csharp
public bool NamesAreCaseSensitive { get; set; }
```
