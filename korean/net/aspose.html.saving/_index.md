**Aspose.Html.Saving** 네임스페이스는 변환/저장 프로세스. 에서 특정 저장 옵션에 대한 설명을 위해 데이터 클래스에 의해 제공됩니다.

## 클래스

| 수업 | 설명 |
| --- | --- |
| [DocSaveOptions](./docsaveoptions/) | 특정 옵션 데이터 클래스. |
| [HTMLSaveOptions](./htmlsaveoptions/) | HTML 저장 옵션을 나타냅니다. |
| [ImageSaveOptions](./imagesaveoptions/) | 특정 옵션 데이터 클래스. |
| [MarkdownSaveOptions](./markdownsaveoptions/) | 마크다운 저장 옵션을 나타냅니다. |
| [MHTMLSaveOptions](./mhtmlsaveoptions/) | MHTML 저장 옵션을 나타냅니다. |
| [PdfSaveOptions](./pdfsaveoptions/) | 특정 옵션 데이터 클래스. |
| [ResourceHandlingOptions](./resourcehandlingoptions/) | 리소스 처리 옵션을 나타냅니다. |
| [SaveOptions](./saveoptions/) | 문서를 특정 형식으로 저장할 때 사용자가 추가 옵션을 지정할 수 있도록 하는 클래스의 추상 기본 클래스입니다. |
| [XpsSaveOptions](./xpssaveoptions/) | 특정 옵션 데이터 클래스. |
