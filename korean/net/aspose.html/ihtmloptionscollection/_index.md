## IHTMLOptionsCollection interface

안`HTMLOptionsCollection` HTML 옵션 요소를 나타내는 노드 목록입니다. 개별 노드는 서수 인덱스 또는 노드의`이름` 또는`ID` 속성. HTML DOM의 컬렉션은 기본 문서가 변경될 때 자동으로 업데이트된다는 의미에서 라이브로 간주됩니다.
