## HTMLInputElement.Checked property

때`유형`요소의 속성은 "라디오" 또는 "체크박스" 값을 가지며, 이는 대화형 사용자 에이전트에서 컨트롤 형식의 현재 상태를 나타냅니다. 이 속성 를 변경하면 양식 컨트롤의 상태가 변경되지만 INPUT 요소의 HTML 검사 속성인 값은 변경되지 않습니다. 값 "radio" 또는 "checkbox", 일부 구현은 이벤트가 문서에서 발송되기 전에 이 속성의 값을 변경할 수 있습니다. 이벤트의 기본 동작이 취소되면 속성의 값이 다시 원래 값으로 변경될 수 있습니다. 이것은 클릭 이벤트 를 처리하는 동안 이 속성의 값이 구현에 따라 다르다는 것을 의미합니다.

```csharp
public bool Checked { get; set; }
```
