## HTMLInputElement.Value property

때`유형` 요소의 속성은 "텍스트", "파일" 또는 "비밀번호" 값을 가지며, 이는 대화식 사용자 에이전트에서 해당 양식 컨트롤의 현재 내용을 나타냅니다. 이 속성을 변경하면 양식 컨트롤의 내용이 변경되지만 는 요소의 HTML 값 속성 값을 변경하지 않습니다. 때`유형`요소의 속성은 "버튼", "숨김", "제출", "재설정", "이미지", "체크박스" 또는 "라디오" 값을 가지며 이는 요소의 HTML 값 속성을 나타냅니다. HTML 4.01의 값 속성 정의를 참조하십시오.

```csharp
public string Value { get; set; }
```
