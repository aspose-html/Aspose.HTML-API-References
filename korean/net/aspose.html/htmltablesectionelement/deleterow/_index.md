## HTMLTableSectionElement.DeleteRow method

이 섹션에서 행을 삭제합니다.

```csharp
public void DeleteRow(int index)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| index | Int32 | 삭제할 행의 인덱스 또는 마지막 행을 삭제하려면 -1입니다. 이 인덱스는 0부터 시작하며 테이블의 모든 행이 아니라 이 섹션에 포함된 행 에만 상대적입니다. |
