## HTMLSelectElement.SelectedIndex property

0부터 시작하는 선택한 옵션의 서수 인덱스입니다. 선택한 요소가 없으면 -1 값이 반환됩니다. 여러 옵션이 선택된 경우 첫 번째로 선택된 옵션의 인덱스가 반환됩니다.

```csharp
public int SelectedIndex { get; set; }
```
