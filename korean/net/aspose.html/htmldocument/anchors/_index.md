## HTMLDocument.Anchors property

모든 앵커(`ㅏ` ) 문서 에 대한 값이 있는 요소`이름`기인하다. 이전 버전과의 호환성 때문에 반환된 앵커 집합에는`이름` 속성으로 생성된 가 아닙니다.`ID` 기인하다. [[XHTML 1.0](http://www.w3.org/TR/2002/REC-xhtml1-20020801) ], `이름` 속성(섹션 4.10 참조)에는 의미 체계가 없으며 는 레거시 사용자 에이전트에만 존재합니다.`ID` 속성 가 대신 사용됩니다. 사용자는 가 제공하는 반복자 메커니즘을 [[DOM 레벨 2 순회](http://www.w3.org/TR/2000/REC-DOM-Level-2-Traversal-Range-20001113) ] 대신에.

```csharp
public HTMLCollection Anchors { get; }
```
