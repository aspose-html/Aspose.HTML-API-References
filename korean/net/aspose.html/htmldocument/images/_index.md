## HTMLDocument.Images property

모든 컬렉션`IMG` 문서의 요소. 동작은 다음으로 제한됩니다.`IMG` 하위 호환성을 위한 요소. [에서 제안한 바와 같이[HTML 4.01](http://www.w3.org/TR/1999/REC-html401-19991224)], 이미지를 포함하기 위해 작성자는 를 사용할 수 있습니다.`물체` 요소 또는`IMG` 요소. 따라서 문서에서 이미지를 찾기 위해 이 속성을 사용하지 않는 것이 좋습니다.`getElementsByTagName` HTML 4.01 또는`getElementsByTagNameNS` XHTML 1.0. 사용

```csharp
public HTMLCollection Images { get; }
```
