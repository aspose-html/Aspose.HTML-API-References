## HTMLTableElement.DeleteRow method

테이블 행을 삭제합니다.

```csharp
public void DeleteRow(int index)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| index | Int32 | 삭제할 행의 인덱스입니다. 이 인덱스는 0부터 를 시작하고 테이블 내부에 포함된 모든 행의 의 논리적 순서(문서 순서가 아님)에 상대적입니다. 인덱스가 -1이면 테이블의 마지막 행이 삭제됩니다. |

### 예외

| 예외 | 상태 |
| --- | --- |
| [DOMException](../../../aspose.html.dom/domexception/) | INDEX_SIZE_ERR: 지정된 인덱스가 행 수보다 크거나 가 같거나 인덱스가 -1. 이외의 음수 인 경우 발생 @version DOM Level 2 |
