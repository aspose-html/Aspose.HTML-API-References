**Aspose.HTML** 네임스페이스에는 간단하거나 복잡한 HTML 문서를 즉석에서 조작하기 위한 클래스와 메서드가 포함되어 있습니다.  **Aspose.HTML**for .NET을 사용하면 개발자가 삽입, 제거, HTML 노드 교체, CSS 스타일 정보 추출, NodeIterator, TreeWalker(순회 사양), XPath 또는 CSS 선택기 쿼리를 통해 HTML 문서를 탐색할 수 있습니다. 또한 JavaScript를 통해 HTML DOM을 to 조작할 수 있는 스크립팅을 제공합니다. 이 API는 HTML뿐만 아니라 EPUB 및 MHTML을 로드하는 기능도 제공합니다 . Aspose API는 파일 형식 간 변환 기능으로 유명하며 이 API는 또한 HTML 파일을 로드하고 출력을 PDF, XPS 및 JPEG, PNG, BMP 및 TIFF를 포함한 래스터 이미지 형식으로 렌더링하는 기능을 제공합니다.
