## HTMLTableRowElement.InsertCell method

빈 삽입`TD` 셀을 이 행에 넣습니다. 인 경우`색인` -1이거나 셀 수와 같으면 새 셀이 추가됩니다.

```csharp
public HTMLElement InsertCell(int index)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| index | Int32 | 0부터 시작하여 셀을 삽입할 위치입니다. |

### 반환 값

새로 생성된 셀입니다.

### 예외

| 예외 | 상태 |
| --- | --- |
| [DOMException](../../../aspose.html.dom/domexception/) | INDEX_SIZE_ERR: 지정된 경우 발생`색인` 가 셀 수보다 크거나 인덱스가 -1. 가 아닌 이외의 음수인 경우 @version DOM Level 2 |
