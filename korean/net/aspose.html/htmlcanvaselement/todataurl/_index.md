## ToDataURL() {#todataurl}

는 type 매개변수(기본값은 PNG)에 지정된 형식으로 이미지 표현을 포함하는 데이터 URI를 반환합니다. 반환된 이미지의 해상도는 96dpi입니다.

```csharp
public string ToDataURL()
```

### 반환 값

요청된 데이터 URI를 포함하는 DOMString입니다.

## ToDataURL(string, double) {#todataurl_2}

는 type 매개변수(기본값은 PNG)에 지정된 형식으로 이미지 표현을 포함하는 데이터 URI를 반환합니다. 반환된 이미지의 해상도는 96dpi입니다.

```csharp
public string ToDataURL(string type, double encoderOptions)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| type | String | 이미지 형식을 나타내는 문자열입니다. 기본 형식 유형은 image/png입니다. |
| encoderOptions | Double | 요청된 유형이 image/jpeg 또는 image/webp인 경우 이미지 품질을 나타내는 0에서 1 사이의 숫자입니다. |

### 반환 값

요청된 데이터 URI를 포함하는 DOMString입니다.
