## HTMLFormElement.Enctype property

제출된 양식의 콘텐츠 유형으로, 일반적으로 "application/x-www-form-urlencoded"입니다. HTML 4.01의 enctype 속성 정의를 참조하십시오. onsubmit 짝수 핸들러는 이 메서드를 호출할 때 트리거되는 가 보장되지 않습니다. 동작은 역사적인 이유로 일관성이 없으며 작성자는 특정 동작에 의존해서는 안 됩니다.

```csharp
public string Enctype { get; set; }
```
