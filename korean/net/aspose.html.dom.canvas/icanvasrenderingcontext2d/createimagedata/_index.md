## CreateImageData(double, double) {#createimagedata_1}

지정된 크기로 비어 있는 새 ImageData 개체를 만듭니다. 새 개체의 모든 픽셀은 투명한 검은색입니다.

```csharp
public IImageData CreateImageData(double sw, double sh)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| sw | Double | 새 ImageData 개체에 제공할 너비입니다. |
| sh | Double | 새 ImageData 개체에 제공할 높이입니다. |

### 반환 값

너비와 높이가 지정된 새 ImageData 객체입니다. 새 개체는 투명한 검정색 픽셀로 채워집니다.
