## StrokeText(string, double, double, double?) {#stroketext_1}

주어진 (x, y) 위치에 주어진 텍스트를 그립니다.

```csharp
public void StrokeText(string text, double x, double y, double? maxWidth)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| text | String | 현재 글꼴, textAlign, textBaseline 및 방향 값을 사용하여 그릴 텍스트입니다. |
| x | Double | 텍스트 시작점 좌표의 x축입니다. |
| y | Double | 텍스트 시작점 좌표의 y축입니다. |
| maxWidth | Nullable`1 | 그릴 최대 너비입니다. 지정되고 문자열이 이 너비보다 더 넓은 것으로 계산되면 글꼴이 더 가로로 압축된 글꼴(사용 가능한 경우 또는 현재 글꼴을 가로로 크기 조정하여 합리적으로 읽을 수 있는 글꼴을 합성할 수 있는 경우) 또는 더 작은 글꼴을 사용하도록 조정됩니다. 폰트. |
