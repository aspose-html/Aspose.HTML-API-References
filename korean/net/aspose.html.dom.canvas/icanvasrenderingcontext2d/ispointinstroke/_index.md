## IsPointInStroke(double, double) {#ispointinstroke_1}

지정된 지점이 경로 스트로크에 포함된 영역 내부에 있는지 여부를 보고합니다.

```csharp
public bool IsPointInStroke(double x, double y)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| x | Double | 확인할 점의 X좌표. |
| y | Double | 확인할 점의 Y좌표. |

### 반환 값

포인트가 패스 스트로크에 포함된 영역 안에 있으면 true를 반환하고, 그렇지 않으면 false를 반환합니다.
