## CanvasFillRule enumeration

이 열거형은 점이 경로 내부 또는 외부에 있는지 확인하는 채우기 규칙 알고리즘을 선택하는 데 사용됩니다.

```csharp
public enum CanvasFillRule
```

### 가치

| 이름 | 값 | 설명 |
| --- | --- | --- |
| NonZero | `0` | 값 "0이 아닌" 값은 0이 아닌 굴곡 규칙을 나타냅니다. 여기서 해당 점에서 그려진 반무한 직선이 한 방향으로 가는 모양의 경로 를 교차하는 횟수가 모양 외부에 있는 것으로 간주됩니다. 다른 방향으로 가는 경로를 가로지르는 횟수와 같습니다. |
| EvenOdd | `1` | "evenodd" 값은 짝수-홀수 규칙을 나타내며, 해당 점에서 그어진 반무한 직선이 도형의 경로를 가로지르는 횟수가 짝수이면 해당 점이 도형 외부에 있는 것으로 간주됩니다. |
