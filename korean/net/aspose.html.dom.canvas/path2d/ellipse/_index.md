## Ellipse(double, double, double, double, double, double, double) {#ellipse}

startAngle 에서 시작하여 주어진 방향으로 반시계 방향으로 가는 endAngle에서 끝나는 반경 radiusX 및 radiusY를 사용하여 (x, y) 위치를 중심으로 하는 경로에 타원을 추가합니다(기본값은 시계 방향).

```csharp
public void Ellipse(double x, double y, double radiusX, double radiusY, double rotation, 
    double startAngle, double endAngle)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| x | Double | 타원 중심 좌표의 x축입니다. |
| y | Double | 타원 중심 좌표의 y축입니다. |
| radiusX | Double | 타원의 장축 반지름입니다. |
| radiusY | Double | 타원의 단축 반지름입니다. |
| rotation | Double | 라디안으로 표시되는 이 타원의 회전입니다. |
| startAngle | Double | x축에서 측정한 시작점으로, x축에서 시작하여 라디안으로 표시됩니다. |
| endAngle | Double | 라디안으로 표시되는 끝 타원이 그려질 각도입니다. |

## Ellipse(double, double, double, double, double, double, double, bool) {#ellipse_1}

startAngle 에서 시작하여 주어진 방향으로 반시계 방향으로 가는 endAngle에서 끝나는 반경 radiusX 및 radiusY를 사용하여 (x, y) 위치를 중심으로 하는 경로에 타원을 추가합니다(기본값은 시계 방향).

```csharp
public void Ellipse(double x, double y, double radiusX, double radiusY, double rotation, 
    double startAngle, double endAngle, bool anticlockwise)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| x | Double | 타원 중심 좌표의 x축입니다. |
| y | Double | 타원 중심 좌표의 y축입니다. |
| radiusX | Double | 타원의 장축 반지름입니다. |
| radiusY | Double | 타원의 단축 반지름입니다. |
| rotation | Double | 라디안으로 표시되는 이 타원의 회전입니다. |
| startAngle | Double | x축에서 측정한 시작점으로, x축에서 시작하여 라디안으로 표시됩니다. |
| endAngle | Double | 라디안으로 표시되는 끝 타원이 그려질 각도입니다. |
| anticlockwise | Boolean | true인 경우 타원을 시계 반대 방향(시계 반대 방향)으로 그리고 그렇지 않으면 시계 방향으로 그리는 선택적 부울입니다. |
