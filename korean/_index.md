## .NET용 Aspose.HTML

{{% alert color="primary" %}} 

![.NET용 Aspose.HTML 제품 로고](../home_1.png)


.NET용 Aspose.HTML은 애플리케이션이 광범위한 HTML 조작 작업을 수행할 수 있도록 하는 크로스 플랫폼 클래스 라이브러리입니다. HTML은 일반적으로 다른 기술과 결합하여 사용되므로 Aspose.HTML은 HTML5뿐만 아니라 CSS3, SVG 및 HTML Canvas 사양도 지원합니다. Aspose.HTML을 사용하여 웹 페이지를 로드하고, 콘텐츠를 분석 및 편집하고, HTML, XHTML, MHTML, SVG, Markdown, PDF, XPS,
EPUB, PNG, TIFF, JPEG, BMP 및 기타 형식.

{{% /alert %}} 

다음은 몇 가지 유용한 리소스에 대한 링크입니다.
- [.NET API 참조용 Aspose.HTML](/html/ko/net/)


## 자바용 Aspose.HTML

{{% alert color="primary" %}} 

![Aspose.HTML for Java 제품 로고](../home_2.png)


Aspose.HTML for Java는 Java 애플리케이션 내에서 직접 광범위한 HTML 조작 작업을 수행하도록 구축된 고급 HTML 조작 API입니다. API는 또한 PDF 및 XPS와 같은 고정 레이아웃 형식과 다양한 래스터 이미지 형식을 위한 충실도 높은 렌더링 엔진을 제공합니다.

{{% /alert %}}

- [.Java API 참조용 Aspose.HTML](/html/java/)
