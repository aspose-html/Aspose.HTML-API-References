## IFile.LastModified property

Дата последнего изменения файла. При получении, если пользовательские агенты могут сделать эту информацию доступной, это должно вернуть длинный длинный набор времени, когда файл был в последний раз изменен, как количество миллисекунд с начала эпохи Unix.

public long LastModified { get; }
