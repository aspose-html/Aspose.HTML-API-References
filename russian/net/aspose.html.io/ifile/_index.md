## IFile interface

Объект File — это объект Blob с атрибутом имени, который представляет собой строку; он может быть создан в веб-приложении с помощью конструктора или является ссылкой на последовательность байтов из файла базовой файловой системы (ОС).

```csharp
public interface IFile : IBlob
```

## Характеристики

| Имя | Описание |
| --- | --- |
| [LastModified](../../aspose.html.io/ifile/lastmodified/) { get; } | Дата последнего изменения файла. При получении, если пользовательские агенты могут сделать эту информацию доступной, это должно вернуть длинный длинный набор времени, когда файл был в последний раз изменен, как количество миллисекунд с начала эпохи Unix. |
| [Name](../../aspose.html.io/ifile/name/) { get; } | Имя файла. При получении должно возвращаться имя файла в виде строки. |
