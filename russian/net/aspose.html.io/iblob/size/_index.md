## IBlob.Size property

Возвращает размер последовательности байтов в байтах. При получении соответствующие пользовательские агенты должны возвращать общее количество байтов, которые могут быть прочитаны объектом FileReader или FileReaderSync, или 0, если в большом двоичном объекте нет байтов для чтения.
