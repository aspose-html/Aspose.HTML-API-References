## IBlob.Type property

Строка в кодировке ASCII в нижнем регистре, представляющая тип мультимедиа BLOB-объекта. При получении пользовательские агенты должны возвращать тип Blob в виде строки в кодировке ASCII в нижнем регистре, таким образом, чтобы при преобразовании в последовательность байтов это был анализируемый тип MIME, или пустая строка – 0 байт – если тип не может быть определен.
