## IBlob.Slice method

Возвращает новый объект Blob с байтами в диапазоне от необязательного начального параметра до, но не включая необязательный конечный параметр, и с атрибутом типа это значение необязательного параметра contentType.

```csharp
public IBlob Slice(ulong start, ulong end, string contentType)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| start | UInt64 | Параметр является значением начальной точки среза |
| end | UInt64 | Параметр представляет собой значение начальной точки среза |
| contentType | String | Параметр представляет собой медиа-тип Blob |

### Возвращаемое значение

Возвращает новый объект Blob с байтами в диапазоне от необязательного начального параметра до необязательного конечного параметра, но не включая его, и с атрибутом типа, который является значением необязательного параметра contentType.
