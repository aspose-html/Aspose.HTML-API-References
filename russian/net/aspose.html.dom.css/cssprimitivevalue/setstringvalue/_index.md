## CSSPrimitiveValue.SetStringValue method

Метод установки строкового значения с указанными единицами измерения. Если свойство, прикрепленное к этому значению, не может принять указанную единицу измерения или строковое значение, значение останется неизменным и будет возбуждено исключение DOMException.

```csharp
public abstract void SetStringValue(ushort stringType, string stringValue)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| stringType | UInt16 | Тип строки. |
| stringValue | String | Строковое значение. |
