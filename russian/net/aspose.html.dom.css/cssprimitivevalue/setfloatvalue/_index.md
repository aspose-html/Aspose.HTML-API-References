## CSSPrimitiveValue.SetFloatValue method

Метод для установки значения с плавающей запятой с указанными единицами измерения. Если свойство, присоединенное к этому значению, не может принять указанную единицу измерения или значение с плавающей запятой, значение останется неизменным и будет возбуждено исключение DOMException.
