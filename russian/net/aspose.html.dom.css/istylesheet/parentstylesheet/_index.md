## IStyleSheet.ParentStyleSheet property

Для языков таблиц стилей, которые поддерживают концепцию включения таблиц стилей, этот атрибут представляет включающую таблицу стилей, если она существует. Если таблица стилей является таблицей стилей верхнего уровня или язык таблицы стилей не поддерживает включение, значение этого атрибута равно null.

```csharp
public IStyleSheet ParentStyleSheet { get; }
```
