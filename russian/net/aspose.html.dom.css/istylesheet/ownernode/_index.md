## IStyleSheet.OwnerNode property

Узел, который связывает эту таблицу стилей с документом. Для HTML это может быть соответствующий элемент LINK или STYLE. Для XML это может быть инструкция по обработке ссылок. Для таблиц стилей, которые включены в другие таблицы стилей, значение этого атрибута равно null.

```csharp
public Node OwnerNode { get; }
```
