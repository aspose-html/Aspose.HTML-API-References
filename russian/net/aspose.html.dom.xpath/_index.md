Пространство имен содержит методы для навигации по элементам и атрибутам в XML-документе.

| Интерфейс | Описание |
| --- | --- |

| [IXPathExpression](./ixpathexpression) | Интерфейс` XPathExpression` представляет проанализированное и разрешенное выражение XPath. |
| [IXPathNamespace](./ixpathnamespace) | Интерфейс XPathNamespace возвращается интерфейсами XPathResult для представления типа узла пространства имен XPath, которого нет в DOM. |
| [IXPathNSResolver](./ixpathnsresolver) | ` XPathNSResolver`разрешение интерфейса `префикс ` строк в выражение, которое должно быть правильно связано с` namespaceURI` строк. [`IXPathEvaluator`](../aspose.html.dom.xpath/ixpathevaluator)может построить реализацию [`IXPathNSResolver`](../aspose.html.dom.xpath/ixpathnsresolver)из узла, или интерфейс может быть реализован любым приложением. |
| [IXPathResult](./ixpathresult) | Интерфейс` XPathResult` представляет результат оценки XPath 1.0 выражение в контексте конкретного узла. Так как вычисление выражения XPath может привести к различным типам результатов, этот объект позволяет обнаруживать и управлять типом и значением результата. |
