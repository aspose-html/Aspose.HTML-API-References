## Конструкторы

| Имя | Описание |
| --- | --- |
| [MutationObserver](mutationobserver)(MutationCallback) | Создает объект MutationObserver и устанавливает для его[`MutationCallback`](../mutationcallback)обратный вызов. Обратный вызов вызывается со списком объектов MutationRecord в качестве первого аргумента и созданным объектом MutationObserver в качестве второго аргумента. Он вызывается после изменения узлов, зарегистрированных с помощью метода &lt;see cref="!:Observe(Node, IMutationObserverInit)" /&gt;. |
