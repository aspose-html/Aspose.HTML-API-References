## ICanvasDrawingStyles interface

Интерфейс ICanvasDrawingStyles предоставляет методы и свойства, управляющие тем, как рисуются линии и как размещается текст.

```csharp
public interface ICanvasDrawingStyles
```

| Имя | Описание |
| --- | --- |

| [LineCap](../../aspose.html.dom.canvas/icanvasdrawingstyles/linecap/) { get; set; } | Тип окончания на конце строк. Возможные значения: приклад (по умолчанию), круглый, квадратный. |
