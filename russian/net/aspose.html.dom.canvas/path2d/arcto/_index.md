## Path2D.ArcTo method

Добавляет к пути дугу с заданными контрольными точками и радиусом, соединенную с предыдущей точкой прямой линией.

```csharp
public void ArcTo(double x1, double y1, double x2, double y2, double radius)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| x1 | Double | координаты по оси x для первой контрольной точки. |
| y1 | Double | Координаты по оси Y для первой контрольной точки. |
| x2 | Double | Координаты по оси x для второй контрольной точки. |
| y2 | Double | Координаты по оси Y для второй контрольной точки. |
| radius | Double | Радиус дуги. |
