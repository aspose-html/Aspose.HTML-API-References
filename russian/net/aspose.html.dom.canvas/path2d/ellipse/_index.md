## Ellipse(double, double, double, double, double, double, double) {#ellipse}

Добавляет эллипс к пути с центром в позиции (x, y) с радиусами радиуса X и радиуса Y, начинающимися в startAngle и заканчивающимися в endAngle, идущими в заданном направлении против часовой стрелки (по умолчанию по часовой стрелке).

```csharp
public void Ellipse(double x, double y, double radiusX, double radiusY, double rotation, 
    double startAngle, double endAngle)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | Double | Ось x координаты центра эллипса. |
| y | Double | Ось Y координат центра эллипса. |
| radiusX | Double | Радиус большой оси эллипса. |
| radiusY | Double | Радиус малой оси эллипса. |
| rotation | Double | Вращение для этого эллипса, выраженное в радианах. |
| startAngle | Double | Начальная точка, отсчитываемая от оси x, из которой будет отрисовываться, выраженная в радианах. |
| endAngle | Double | Конечный угол эллипса, на который он будет нарисован, выраженный в радианах. |

## Ellipse(double, double, double, double, double, double, double, bool) {#ellipse_1}

Добавляет эллипс к пути с центром в позиции (x, y) с радиусами радиуса X и радиуса Y, начинающимися в startAngle и заканчивающимися в endAngle, идущими в заданном направлении против часовой стрелки (по умолчанию по часовой стрелке).

```csharp
public void Ellipse(double x, double y, double radiusX, double radiusY, double rotation, 
    double startAngle, double endAngle, bool anticlockwise)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | Double | Ось x координаты центра эллипса. |
| y | Double | Ось Y координат центра эллипса. |
| radiusX | Double | Радиус большой оси эллипса. |
| radiusY | Double | Радиус малой оси эллипса. |
| rotation | Double | Вращение для этого эллипса, выраженное в радианах. |
| startAngle | Double | Начальная точка, отсчитываемая от оси x, из которой будет отрисовываться, выраженная в радианах. |
| endAngle | Double | Конечный угол эллипса, на который он будет нарисован, выраженный в радианах. |
| anticlockwise | Boolean | Необязательное логическое значение, которое, если оно истинно, рисует эллипс против часовой стрелки (против часовой стрелки), в противном случае — по часовой стрелке. |
