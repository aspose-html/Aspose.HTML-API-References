## Ellipse(double, double, double, double, double, double, double) {#ellipse}

Добавляет эллипс к пути с центром в позиции (x, y) с радиусами radiusX и radiusY, начинающимися в startAngle и заканчивающимися в endAngle движется в заданном направлении против часовой стрелки (по умолчанию по часовой стрелке).

```csharp
public void Ellipse(double x, double y, double radiusX, double radiusY, double rotation, 
    double startAngle, double endAngle)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | Double | Ось x координат центра эллипса. |
| y | Double | Ось Y координат центра эллипса. |
| radiusX | Double | Радиус большой оси эллипса. |
| radiusY | Double | Радиус малой оси эллипса. |
| rotation | Double | Вращение для этого эллипса, выраженное в радианах. |
| startAngle | Double | Начальная точка, отсчитываемая от оси x, из которой будет отрисовываться, выраженная в радианах. |
| endAngle | Double | Конечный угол эллипса, под которым он будет нарисован, выраженный в радианах. |
