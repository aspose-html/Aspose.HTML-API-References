## Path2D.BezierCurveTo method

Добавляет кубическую кривую Безье к пути. Требует три балла. Первые две точки — контрольные, а третья — конечная. Начальная точка — это последняя точка текущего пути, которую можно изменить с помощью функции moveTo() перед созданием кривой Безье.

```csharp
public void BezierCurveTo(double cp1x, double cp1y, double cp2x, double cp2y, double x, double y)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| cp1x | Double | Ось x координат первой контрольной точки. |
| cp1y | Double | Ось Y координат первой контрольной точки. |
| cp2x | Double | Ось x координат второй контрольной точки. |
| cp2y | Double | Ось Y координат второй контрольной точки. |
| x | Double | Ось x координат конечной точки. |
| y | Double | Ось Y координат конечной точки. |
