## IImageData interface

Создает объект ImageData из заданного Uint8ClampedArray и размера содержащегося в нем изображения. Если массив не задан, создается изображение черного прямоугольника.

```csharp
public interface IImageData
```

| --- | --- |
| [Data](../../aspose.html.dom.canvas/iimagedata/data/) { get; } | Является Uint8ClampedArray, представляющим одномерный массив, содержащий данные в порядке RGBA, с целочисленными значениями от 0 до 255 (включительно). |
| [Height](../../aspose.html.dom.canvas/iimagedata/height/) { get; } | Длинное число без знака, представляющее фактическую высоту в пикселях ImageData. |
