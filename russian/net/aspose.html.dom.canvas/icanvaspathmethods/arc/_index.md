## Arc(double, double, double, double, double) {#arc}

Добавляет дугу к пути с центром в позиции (x, y) с радиусом r, начинающимся в startAngle и заканчивающимся в endAngle и идущим в заданном направлении против часовой стрелки (по умолчанию по часовой стрелке).

```csharp
public void Arc(double x, double y, double radius, double startAngle, double endAngle)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | Double | Координата x центра дуги. |
| y | Double | Y-координата центра дуги. |
| radius | Double | Радиус дуги. |
| startAngle | Double | Угол начала дуги, измеренный по часовой стрелке от положительной оси x и выраженный в радианах. |
| endAngle | Double | Угол, под которым заканчивается дуга, измеренный по часовой стрелке от положительной оси x и выраженный в радианах. |
