## CanvasFillRule enumeration

Это перечисление используется для выбора алгоритма правила заполнения, с помощью которого определяется, находится ли точка внутри или вне пути.

```csharp
public enum CanvasFillRule
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| NonZero | `0` | Значение "ненулевое" значение указывает на ненулевое правило намотки, при котором точка считается находящейся за пределами формы , если полубесконечная прямая линия, проведенная из этой точки, сколько раз пересекает путь фигуры в одном направлении равно количеству раз, которое он пересекает путь, идущий в другом направлении. |
| EvenOdd | `1` | Значение "evenodd" указывает на правило четного-нечетного, согласно которому точка считается находящейся вне фигуры, если количество раз, когда полубесконечная прямая, проведенная из этой точки, пересекает путь фигуры, является четным. |
