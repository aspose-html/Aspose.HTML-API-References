## CanvasFillRule enumeration

Это перечисление используется для выбора алгоритма правила заполнения, с помощью которого определяется, находится ли точка внутри или вне пути.

```csharp
public enum CanvasFillRule
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| NonZero | `0` | Значение "ненулевое" указывает на ненулевое правило обмотки, при котором точка считается находящейся вне формы, если количество количество раз, когда полубесконечная прямая, проведенная из этой точки, пересекает путь фигуры в одном направлении, равно количеству раз, когда она пересекает путь в другом направлении. |
| EvenOdd | `1` | Значение "evenodd" указывает на правило четного-нечетного, согласно которому точка считается находящейся вне формы, если количество раз полубесконечная прямая, проведенная из этой точки, пересекает траекторию фигуры. |
