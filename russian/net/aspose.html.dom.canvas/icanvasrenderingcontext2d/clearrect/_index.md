## ICanvasRenderingContext2D.ClearRect method

Устанавливает все пиксели в прямоугольнике, определяемом начальной точкой (x, y) и размером (шириной, высотой), в прозрачный черный цвет, стирая все ранее нарисованное содержимое.

```csharp
public void ClearRect(double x, double y, double w, double h)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | Double | Ось x координат начальной точки прямоугольника. |
| y | Double | Ось Y координат начальной точки прямоугольника. |
| w | Double | Ширина прямоугольника. |
| h | Double | Высота прямоугольника. |
