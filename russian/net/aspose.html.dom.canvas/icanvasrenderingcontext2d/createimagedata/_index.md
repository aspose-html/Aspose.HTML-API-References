## CreateImageData(double, double) {#createimagedata_1}

Создает новый пустой объект ImageData с указанными размерами. Все пиксели в новом объекте прозрачно-черные.

```csharp
public IImageData CreateImageData(double sw, double sh)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| sw | Double | Ширина для предоставления нового объекта ImageData. |
| sh | Double | Высота для создания нового объекта ImageData. |

### Возвращаемое значение

Новый объект ImageData с указанной шириной и высотой. Новый объект заполнен прозрачными черными пикселями.

## CreateImageData(IImageData) {#createimagedata}

```csharp
public IImageData CreateImageData(IImageData imagedata)
```

| Параметр | Тип | Описание |
