## IsPointInPath(double, double) {#ispointinpath_2}

Сообщает, содержится ли указанная точка в текущем пути.

```csharp
public bool IsPointInPath(double x, double y)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | Double | Координата X точки для проверки. |
| y | Double | Координата Y точки для проверки. |

### Возвращаемое значение

Возвращает true, если точка находится внутри области, содержащейся в заполнении пути, иначе false.
