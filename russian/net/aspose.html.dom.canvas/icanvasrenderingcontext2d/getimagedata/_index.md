## ICanvasRenderingContext2D.GetImageData method

Возвращает объект ImageData, представляющий базовые пиксельные данные для области холста, обозначенной прямоугольником, который начинается с (sx, sy) и имеет ширину sw и высоту sh . На этот метод не влияет матрица преобразования холста.

```csharp
public IImageData GetImageData(double sx, double sy, double sw, double sh)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| sx | Double | Координата x левого верхнего угла прямоугольника, из которого будут извлечены ImageData. |
| sy | Double | Координата y верхнего левого угла прямоугольника, из которого будут извлечены ImageData. |
| sw | Double | Ширина прямоугольника, из которого будут извлечены ImageData. |
| sh | Double | Высота прямоугольника, из которого будут извлечены ImageData. |

### Возвращаемое значение

Объект ImageData, содержащий данные изображения для данного прямоугольника холста.
