## IsPointInStroke(double, double) {#ispointinstroke_1}

Сообщает, находится ли указанная точка внутри области, содержащейся при обводке пути.

```csharp
public bool IsPointInStroke(double x, double y)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| x | Double | Координата X точки для проверки. |
| y | Double | Координата Y точки для проверки. |

### Возвращаемое значение

Возвращает true, если точка находится внутри области, содержащейся в контуре, в противном случае — false.
