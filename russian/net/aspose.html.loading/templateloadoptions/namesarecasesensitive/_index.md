## TemplateLoadOptions.NamesAreCaseSensitive property

Свойство, указывающее, будут ли имена шаблона и элемента данных совпадать независимо от регистра символов или нет. Если true, поиск чувствителен к регистру; если false, регистр символов будет игнорироваться. Значение по умолчанию — true.

public bool NamesAreCaseSensitive { get; set; }
