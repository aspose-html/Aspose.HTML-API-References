## TemplateLoadOptions.NamesAreCaseSensitive property

Свойство, указывающее, будут ли имена шаблонов и элементов данных сопоставляться независимо от регистра символов или нет. Если true, сопоставление чувствительно к регистру; если false, регистр символов будет игнорироваться. Значение по умолчанию — true.

```csharp
public bool NamesAreCaseSensitive { get; set; }
```
