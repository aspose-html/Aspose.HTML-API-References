## ITrueTypeFont.FullFontName property

Это должно быть сочетание "FamilyName" и "SubFamilyName". Исключение:если шрифт «Обычный», как указано в «SubFamilyName», тогда используйте только имя семейства, содержащееся в «FamilyName». Исключением из приведенного выше определения полного имени шрифта являются строки платформы Microsoft для шрифтов CFF OpenType:в этом случае строка полного имени шрифта должна быть идентична PostScript FontName в ИНДЕКСе имени CFF.

```csharp
public string FullFontName { get; }
```

### Возвращаемое значение

Полное название шрифта
