## Event.PreventDefault method

Если событие можно отменить, метод`PreventDefault`используется для обозначения того, что событие должно быть отменено, что означает, что любое действие по умолчанию, обычно выполняемое реализацией в результате события, не произойдет.

```csharp
public void PreventDefault()
```

### Примечания

Если на любом этапе потока событий файл`PreventDefault`метод вызывается событие отменяется. Любое действие по умолчанию, связанное с событием, не будет выполняться. Вызов этого метода для неотменяемого события не имеет никакого эффекта. После вызова`PreventDefault`он будет действовать до конца распространения события. Этот метод можно использовать на любой стадии потока событий.
