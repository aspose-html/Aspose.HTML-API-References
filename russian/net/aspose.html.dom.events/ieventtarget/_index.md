## IEventTarget interface

Интерфейс[`EventTarget`](../../aspose.html.dom/eventtarget)реализуется всеми узлами в реализации, которая поддерживает модель событий DOM. Таким образом, этот интерфейс может быть получен с помощью методов приведения конкретных привязок к экземпляру интерфейса Node. Интерфейс позволяет регистрировать и удалять прослушиватели событий на[`EventTarget`](../../aspose.html.dom/eventtarget)и отправлять события на этот[`IEventTarget`](../ieventtarget).
