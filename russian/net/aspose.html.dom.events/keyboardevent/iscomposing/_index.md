## KeyboardEvent.IsComposing property

true, если ключевое событие происходит как часть сеанса композиции, т. е. после события Compositionstart и перед соответствующим событием CompositionEnd. Неинициализированное значение этого атрибута ДОЛЖНО быть ложным.

```csharp
public bool IsComposing { get; }
```
