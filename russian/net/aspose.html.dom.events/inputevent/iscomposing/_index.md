## InputEvent.IsComposing property

истина, если событие ввода происходит как часть сеанса композиции, т. е. после события «начало композиции» и до соответствующего события «конец композиции». Неинициализированное значение этого атрибута ДОЛЖНО быть ложным.

```csharp
public bool IsComposing { get; }
```

### Стоимость имущества

` true` если этот экземпляр компонуется; в противном случае` false` .
