## IEventListener interface

Интерфейс[`IEventListener`](../ieventlistener)является основным методом обработки событий. Пользователи реализуют интерфейс[`IEventListener`](../ieventlistener)и регистрируют свой слушатель на[`EventTarget`](../../aspose.html.dom/eventtarget)используя метод[`AddEventListener`](../../aspose.html.dom/eventtarget/addeventlistener). Пользователи также должны удалить свой[`IEventListener`](../ieventlistener)из его[`EventTarget`](../../aspose.html.dom/eventtarget)после того, как они завершили использование слушателя.

### Примечания

Когда узел копируется с помощью метода cloneNode, прослушиватели событий, прикрепленные к исходному узлу, не присоединяются. на скопированный узел. Если пользователь желает, чтобы те же прослушиватели событий были добавлены во вновь созданную копию, пользователь должен добавить их вручную.
