## HTMLSelectElement.SelectedIndex property

Порядковый индекс выбранной опции, начиная с 0. Значение -1 возвращается, если ни один элемент не выбран. Если выбрано несколько параметров , возвращается индекс первого выбранного параметра.

```csharp
public int SelectedIndex { get; set; }
```
