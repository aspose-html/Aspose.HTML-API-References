## HTMLDocument.Anchors property

Сборник всех якорей(`А` ) элементы в документе со значением для`имя`атрибут. Из соображений обратной совместимости возвращаемый набор анкоров содержит только тех анкоров, созданных с помощью`имя` атрибут, а не те , созданные с помощью`идентификатор` атрибут. Обратите внимание, что в [[XHTML 1.0](http://www.w3.org/TR/2002/REC-xhtml1-20020801) ], `имя` атрибут (см. раздел 4.10) не имеет семантики, а присутствует только для устаревших пользовательских агентов:`идентификатор` вместо этого используется атрибут . Пользователям следует предпочесть механизмы итераторов, предоставляемые компанией [[Обход уровня DOM 2](http://www.w3.org/TR/2000/REC-DOM-Level-2-Traversal-Range-20001113) ] вместо.

public HTMLCollection Anchors { get; }
