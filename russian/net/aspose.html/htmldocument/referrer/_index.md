## HTMLDocument.Referrer property

Возвращает URI [[IETF RFC 2396](http://www.ietf.org/rfc/rfc2396.txt) ] страницы, которая ссылалась на эту страницу. Значением является пустая строка, если пользователь перешел на страницу напрямую (не через ссылку, а, например, через закладку).
