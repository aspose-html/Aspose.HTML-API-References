## HTMLDocument.Images property

Коллекция всех элементов` IMG` в документе. Поведение ограничено элементами` IMG` для обратной совместимости . Согласно [[HTML 4.01](http://www.w3.org/TR/1999/REC-html401-19991224) ], для включения изображений авторы могут использовать элемент` OBJECT` или элемент` IMG` . Поэтому рекомендуется не использовать этот атрибут для поиска изображений в документе, а` getElementsByTagName` с HTML 4.01 или` getElementsByTagNameNS` с XHTML 1.0.

public HTMLCollection Images { get; }
