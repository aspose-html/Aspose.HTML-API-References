## HTMLDocument.Images property

Коллекция всех`ИМГ` элементы в документе. Поведение ограничено`ИМГ` элементы для обратной совместимости с . Как предложил [[HTML 4.01](http://www.w3.org/TR/1999/REC-html401-19991224)], чтобы включить изображения, авторы могут использовать `ОБЪЕКТ` элемент или`ИМГ` элемент. Поэтому рекомендуется не использовать этот атрибут для поиска изображений в документе, а`getElementsByTagName` с HTML 4.01 или`getElementsByTagNameNS` с XHTML 1.0.

```csharp
public HTMLCollection Images { get; }
```
