## HTMLTextAreaElement.Value property

Представляет текущее содержимое соответствующего элемента управления формы, в интерактивный пользовательский агент. Изменение этого атрибута изменяет содержимое элемента управления формы, но не изменяет содержимое элемента . Если все данные не помещаются в один `DOMString` , реализация может обрезать данные.

```csharp
public string Value { get; set; }
```
