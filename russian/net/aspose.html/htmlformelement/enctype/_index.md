## HTMLFormElement.Enctype property

Тип содержимого отправленной формы, обычно "application/x-www-form-urlencoded". См. определение атрибута enctype в HTML 4.01. Обработчик onsubmit даже не гарантирует, что будет запущен при вызове этого метода. Поведение непоследовательно по историческим причинам, и авторам не следует полагаться на конкретное .

```csharp
public string Enctype { get; set; }
```
