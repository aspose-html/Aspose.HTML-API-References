## IFormAssociatedElement interface

Элемент, связанный с формой, может иметь связь с элементом формы, который называется владельцем формы элемента. Если связанный с формой элемент не связан с элементом формы, его владелец формы считается нулевым. он повторно связан, может иметь атрибут формы, указанный для переопределения этого.
