## HTMLInputElement.List property

Атрибут списка используется для идентификации элемента, в котором перечислены предопределенные параметры, предлагаемые пользователю. Если он присутствует, его значение должно быть идентификатором элемента списка данных в том же документе.

```csharp
public string List { get; set; }
```
