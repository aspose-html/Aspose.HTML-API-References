## HTMLInputElement.Value property

Когда`тип` Атрибут элемента имеет значение "текст", "файл" или "пароль", это представляет текущее содержимое соответствующего элемента управления формы в интерактивном пользовательском агенте. Изменение этого атрибута изменяет содержимое элемента управления формы, но не изменяет значение атрибута значения HTML элемента. Когда`тип`атрибут элемента имеет значение "кнопка", "скрытый", "отправить", "сброс", "изображение", "флажок" или "радио", это представляет атрибут значения HTML элемента. См. определение атрибута значения в HTML 4.01.

```csharp
public string Value { get; set; }
```
