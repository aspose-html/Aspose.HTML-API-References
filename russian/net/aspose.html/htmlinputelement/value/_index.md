## HTMLInputElement.Value property

Когда атрибут` типа` элемента имеет значение "text ", "файл" или "пароль", это представляет текущее содержимое соответствующего элемента управления формы в интерактивном пользовательском агенте. Изменение этого атрибута изменяет содержимое элемента управления формы, но не изменяет значение HTML-атрибута value элемента. Когда атрибут` типа` элемента имеет значение "кнопка", "скрытый", "отправить ", "reset", "image", "checkbox" или "radio", это представляет атрибут значения HTML элемента. См. определение атрибута value в HTML 4.01.

public string Value { get; set; }
