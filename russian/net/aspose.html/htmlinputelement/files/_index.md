## HTMLInputElement.Files property

IDL-атрибут files позволяет сценариям обращаться к выбранным файлам элемента. При получении, если применяется атрибут IDL, он должен вернуть объект FileList, представляющий текущие выбранные файлы. Этот же объект должен быть возвращен до тех пор, пока не изменится список выбранных файлов. Если атрибут IDL не применяется, вместо этого он должен возвращать значение null. [FILEAPI]

```csharp
public IFileList Files { get; }
```
