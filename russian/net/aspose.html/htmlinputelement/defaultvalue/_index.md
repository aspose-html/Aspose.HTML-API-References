## HTMLInputElement.DefaultValue property

Когда атрибут` типа` элемента имеет значение "text ", "файл" или "пароль", это представляет значение HTML атрибут элемента. Значение этого атрибута не изменяется если содержимое соответствующего элемента управления формы в интерактивном пользовательском агенте изменяется. См. определение атрибута value в HTML 4.01.

public string DefaultValue { get; set; }
