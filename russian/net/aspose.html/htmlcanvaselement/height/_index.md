## HTMLCanvasElement.Height property

— положительное целое число, отражающее HTML-атрибут высоты элемента холста, интерпретируемый в пикселях CSS. Если атрибут не указан или для него задано недопустимое значение, например отрицательное, используется значение по умолчанию 150.

```csharp
public ulong Height { get; set; }
```
