## HTMLCanvasElement.Width property

— положительное целое число, отражающее HTML-атрибут ширины элемента холста, интерпретируемый в пикселях CSS. Если атрибут не указан или для него задано недопустимое значение, например отрицательное, используется значение по умолчанию 300.

```csharp
public ulong Width { get; set; }
```
