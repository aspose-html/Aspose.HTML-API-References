## HTMLCanvasElement.GetContext method

Возвращает контекст рисования на холсте или null, если идентификатор контекста не поддерживается. Контекст рисования позволяет рисовать на холсте.

```csharp
public object GetContext(string contextId)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| contextId | String | Строка, содержащая идентификатор контекста, определяющий контекст рисования, связанный с холстом. |

### Возвращаемое значение

Контекст рендеринга
