## ToDataURL() {#todataurl}

возвращает URI данных, содержащий представление изображения в формате, заданном параметром типа (по умолчанию PNG). Возвращаемое изображение имеет разрешение 96 dpi.

```csharp
public string ToDataURL()
```

### Возвращаемое значение

DOMString, содержащий запрошенный URI данных.

## ToDataURL(string, double) {#todataurl_2}

возвращает URI данных, содержащий представление изображения в формате, заданном параметром типа (по умолчанию PNG). Возвращаемое изображение имеет разрешение 96 dpi.

```csharp
public string ToDataURL(string type, double encoderOptions)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| type | String | Строка, указывающая формат изображения. Тип формата по умолчанию — image/png. |
| encoderOptions | Double | Число от 0 до 1, указывающее качество изображения, если запрошенный тип — изображение/jpeg или изображение/webp. |

DOMString, содержащий запрошенный URI данных.
