## Uint8ClampedArray class

Представляет собой массив 8-битных целых чисел без знака, ограниченный 0-255; если вы указали значение, выходящее за пределы диапазона [0,255], вместо него будет установлено 0 или 255;

```csharp
public class Uint8ClampedArray : TypedArray<byte>
```
