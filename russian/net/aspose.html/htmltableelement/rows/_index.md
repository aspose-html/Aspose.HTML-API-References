## HTMLTableElement.Rows property

Возвращает таблицу`ТФУТ` , или`нулевой` если не существует.

Возвращает коллекцию всех строк в таблице, включая все в `ГОЛОВА` ,`ТФУТ` , все`TBODY` элементов.

```csharp
public HTMLCollection Rows { get; }
```
