## HTMLTableElement.DeleteRow method

Удалить строку таблицы.

```csharp
public void DeleteRow(int index)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | Int32 | Индекс удаляемой строки. Этот индекс начинается с 0 и относится к логическому порядку (не порядку документа) всех строк, содержащихся в таблице. Если индекс равен -1, то последняя строка в таблице удаляется. |
