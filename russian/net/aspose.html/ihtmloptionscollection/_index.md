## IHTMLOptionsCollection interface

Ан`HTMLOptionsCollection` представляет собой список узлов, представляющих элемент опции HTML . К отдельному узлу можно получить доступ либо по порядковому индексу , либо по номеру узла.`имя` или`идентификатор` атрибуты. Коллекции в HTML DOM считаются активными, что означает, что они автоматически обновляются при изменении базового документа.

См. также[Объектная модель документа (DOM) Уровень 2 HTML-спецификация](http://www.w3.org/TR/2003/REC-DOM-Level-2-HTML-20030109). @с уровня DOM 2
