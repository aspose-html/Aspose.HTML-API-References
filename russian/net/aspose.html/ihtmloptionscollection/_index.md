## IHTMLOptionsCollection interface

An` HTMLOptionsCollection` — это список узлов, представляющих элемент HTML option. К отдельному узлу можно получить доступ либо по порядковому номеру индексу, либо по имени узла или` id` атрибуты. Коллекции в HTML DOM считаются живыми, что означает, что они автоматически обновляются при изменении базового документа.

См. также[Модель объекта документа (DOM) Уровень 2 Спецификация HTML](http://www.w3.org/TR/2003/REC-DOM-Level-2-HTML-20030109) . @since DOM Level 2
