## HTMLOptionElement.Selected property

Представляет текущее состояние соответствующего элемента управления формы в интерактивном пользовательском агенте . Изменение этого атрибута изменяет состояние элемента управления формы, но не изменяет значение атрибута HTML selected элемента.

```csharp
public bool Selected { get; set; }
```
