## SVGListBase&lt;T&gt;.Initialize method

Удаляет все существующие текущие элементы из списка и повторно инициализирует список для хранения одного элемента, указанного параметром.

```csharp
public T Initialize(T newItem)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| newItem | T | Элемент, который должен стать единственным членом списка. |

### Возвращаемое значение

Элемент вставляется в список.

### Исключения

| исключение | условие |
| --- | --- |
| [DOMException](../../../aspose.html.dom/domexception/) | Код[`NO_MODIFICATION_ALLOWED_ERR`](../../../aspose.html.dom/domexception/no_modification_allowed_err/). Возникает, когда список нельзя изменить. |
