## FormSubmitter class

Этот класс позволяет подготовить указанные[`HTMLFormElement`](../../aspose.html/htmlformelement/) , собирает значения из элемента формы, отправляет их на удаленный сервер и получает ответ.

```csharp
public class FormSubmitter : IDisposable
```
