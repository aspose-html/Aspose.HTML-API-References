## InputElement.SetColorValue method

Этот метод используется для установки цвета в качестве значения для элемента ввода. Этот метод действителен, если только тип элемента ввода "color"

```csharp
public void SetColorValue(Color value)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | Color | Цветовой объект. |
