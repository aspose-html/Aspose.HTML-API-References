## InputElement.SetDateTimeLocalValue method

Этот метод используется для установкиDateTimeобъект как значение для элемента ввода. Этот метод действителен, если только тип элемента ввода "datetime-local"

```csharp
public void SetDateTimeLocalValue(DateTime value)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| value | DateTime | Объект даты. |
