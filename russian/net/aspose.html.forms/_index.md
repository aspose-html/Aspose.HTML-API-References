Пространство имен обеспечивает доступ к многочисленным объектам (элементам), которые отвечают за интерактивность пользователя в форме как компоненте веб-страницы.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [ButtonElement](./buttonelement/) | ButtonElement представляет оболочку, связанную с HTMLButtonElement. |
| [DataListElement](./datalistelement/) | DataListElement представляет оболочку, связанную с HTMLDataListElement |
| [FormEditor](./formeditor/) | Этот класс представляет редактор над[`HTMLFormElement`](../aspose.html/htmlformelement/) это упрощает разработчикам .net редактирование html-форм. |
| [FormElement](./formelement/) | Представляет базовый класс для элементов формы. |
| [FormElement&lt;T&gt;](./formelement-1/) | [`FormElement`](../aspose.html.forms/formelement/) представлять общее field |
| [FormSubmitter](./formsubmitter/) | Этот класс позволяет подготовить указанные[`HTMLFormElement`](../aspose.html/htmlformelement/) , собирает значения из элемента формы, отправляет их на удаленный сервер и получает ответ. |
| [InputElement](./inputelement/) | InputElement представляет оболочку, связанную с HTMLInputElement. |
| [OptionCollection](./optioncollection/) | OptionElements представляет оболочку, связанную с IHTMLOptionsCollection |
| [OptionElement](./optionelement/) | OptionElement представляет собой оболочку, связанную с HTMLOptionElement |
| [SelectElement](./selectelement/) | SelectElement представляет собой оболочку, связанную с HTMLSelectElement |
| [SubmissionResult](./submissionresult/) | Этот класс представляет результат отправки данных формы на сервер. |
| [TextAreaElement](./textareaelement/) | TextAreaElement представляет оболочку, связанную с HTMLTextAreaElement |
