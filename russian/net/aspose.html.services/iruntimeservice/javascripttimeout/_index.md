## IRuntimeService.JavaScriptTimeout property

Получает или устанавливаетTimeSpan, что ограничивает время выполнения JavaScript. Если скрипт выполняется дольше, чем предусмотреноTimeSpan, он будет отменен. Вы можете указать бесконечный тайм-аут, установивTimeSpanравным -1 миллисекунде. Значение по умолчанию — 1 минута.

```csharp
public TimeSpan JavaScriptTimeout { get; set; }
```
