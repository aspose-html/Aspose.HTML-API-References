## IUserAgentService.Language property

`Language`указывает основной язык для содержимого элемента и для любого из атрибутов элемента которые содержат текст. Его значение должно быть допустимым языковым тегом BCP 47 () или пустым нить. Установка для атрибута пустой строки означает, что основной язык неизвестен.

public string Language { get; set; }
