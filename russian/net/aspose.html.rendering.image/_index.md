**Aspose.Html.Rendering.Image** namespace предоставляет конкретный класс устройства, а также несколько классов options рендеринга, отвечающих за рендеринг в растровые форматы: jpeg, png, bmp, gif, tiff.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [ImageDevice](./imagedevice/) | Представляет рендеринг в растровые форматы: jpeg, png, bmp, gif, tiff. |
| [ImageRenderingOptions](./imagerenderingoptions/) | Представляет параметры рендеринга для[`ImageDevice`](../aspose.html.rendering.image/imagedevice/) . Эти параметры используются для указания формата выходного изображения, сжатия, разрешения и т. д. |
| [TextOptions](./textoptions/) | Представляет параметры рендеринга текста для[`ImageDevice`](../aspose.html.rendering.image/imagedevice/) . |
## перечисление

| перечисление | Описание |
| --- | --- |
| [Compression](./compression/) | Определяет возможные схемы сжатия растровых изображений в формате Tagged Image File Format (TIFF). |
| [ImageFormat](./imageformat/) | Определяет формат файла изображения. |
