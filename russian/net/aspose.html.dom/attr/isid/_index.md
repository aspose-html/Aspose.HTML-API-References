## Attr.IsId property

Возвращает, известно ли, что этот атрибут имеет идентификатор типа (т.е. содержит идентификатор для своего элемента-владельца) или нет.

```csharp
public bool IsId { get; }
```

### Стоимость имущества

`истинный` если этот экземпляр id; в противном случае,`ЛОЖЬ` .
