## TypeInfo.TypeNamespace property

Получает пространство имен типа. Пространство имен типа, объявленного для связанного элемента или атрибута, или null, если у элемента нет объявления или если информация о пространстве имен недоступна.

```csharp
public string TypeNamespace { get; }
```

### Стоимость имущества

Пространство имен типов.
