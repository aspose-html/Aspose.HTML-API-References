## TypeInfo.DERIVATION_RESTRICTION field

Если схема документа представляет собой XML-схему [XML-схема, часть 1], эта константа представляет происхождение по ограничению, если задействованы сложные типы, или ограничение, если задействованы простые типы.

```csharp
public const ulong DERIVATION_RESTRICTION;
```
