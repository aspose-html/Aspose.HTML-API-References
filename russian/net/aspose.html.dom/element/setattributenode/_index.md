## Element.SetAttributeNode method

Добавляет новый узел атрибута. Если атрибут с таким именем (nodeName) уже присутствует в элементе, он заменяется новым.

```csharp
public Attr SetAttributeNode(Attr newAttr)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| newAttr | Attr | Новый атрибут. |

### Возвращаемое значение

Возвращает атрибут
