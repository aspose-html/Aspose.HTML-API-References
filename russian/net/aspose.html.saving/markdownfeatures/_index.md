## MarkdownFeatures enumeration

А`MarkdownFeatures` набор флагов — это набор из нуля или более следующих флагов, которые используются для выбора элементов, преобразованных в уценку.

```csharp
[Flags]
public enum MarkdownFeatures
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| InlineHTML | `1` | Этот флаг включает встраивание элементов HTML. Если этот флаг установлен, то элементы уровня блока (такие как`див` ) чей`уценка` значение атрибута равно`в соответствии` будет вставлен в результирующую уценку. |
| AutomaticParagraph | `2` | Этот флаг включает преобразование`параграф` элементы. Содержимое таких элементов будет размещено на отдельных строках, поэтому обработчики уценки будут оборачивать его. |
| Header | `4` | Этот флаг включает преобразование`заголовок` элементы. |
| Blockquote | `8` | Этот флаг включает преобразование`цитата` элементы. |
| List | `10` | Этот флаг включает преобразование`список` элементы. |
| CodeBlock | `20` | Этот флаг разрешает преобразование блоков кода. Кодовый блок состоит из 2-х элементов`до` и`код` , содержанием такой конструкции является процесс «как есть». |
| HorizontalRule | `40` | Этот флаг включает преобразование`горизонтальные правила` . |
| Link | `80` | Этот флаг включает преобразование`а` элементы. |
| Emphasis | `100` | Этот флаг включает преобразование`акцент` элементы. |
| InlineCode | `200` | Этот флаг включает преобразование`код` элементы. |
| Image | `400` | Этот флаг включает преобразование`изображение` элементы. |
| LineBreak | `800` | Этот флаг включает преобразование`бр` элементы. |
| Video | `1000` | Этот флаг включает преобразование`видео` элементы. |
| Table | `2000` | Этот флаг включает преобразование`стол` элементы. |
| TaskList | `4000` | Этот флаг включает преобразование списков задач. Список задач состоит из`вход` элемент, который должен быть первым дочерним элементом`список` элемент и чей`тип` значение атрибута должно быть равно`флажок` . |
| Strikethrough | `8000` | Этот флаг включает преобразование`дель` элементы. |
| Strong | `10000` | Этот флаг включает преобразование`сильный` элементы. |
