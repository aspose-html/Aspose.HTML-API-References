## ISVGAnimatedPathData.PathSegList property

Предоставляет доступ к базовому (т.е. статическому) содержимому атрибута 'd' в форме, которая полностью соответствует синтаксису SVG. Таким образом, если атрибут 'd' имеет команду "абсолютный переход (M)" и "абсолютный arcto (A)", то pathSegList будет иметь две записи: SVG_PATHSEG_MOVETO_ABS и SVG_PATHSEG_ARC_ABS.

```csharp
public SVGPathSegList PathSegList { get; }
```
