## NamedNodeMap indexer (1 of 2)

Возвращает индексный элемент на карте. Если индекс больше или равен количеству узлов на этой карте, возвращается null.

```csharp
public Attr this[int index] { get; }
```

| Параметр | Описание |
| --- | --- |
| index | Индекс. |

### Возвращаемое значение

Возвращает узел.
