## SVGPolygonElement.AnimatedPoints property

Предоставляет доступ к текущему анимированному содержимому атрибута «точки». Если данный атрибут или свойство анимируются, содержит текущее анимированное значение атрибута или свойства. Если данный атрибут или свойство в настоящее время не анимируются, содержит то же значение, что и точки.

```csharp
public SVGPointList AnimatedPoints { get; }
```

### Стоимость имущества

Анимированные точки.
