## SVGDocument.GetOverrideStyle method

Этот метод используется для получения объявления стиля переопределения для указанного элемента и указанного псевдоэлемента.

```csharp
public ICSSStyleDeclaration GetOverrideStyle(Element elt, string pseudoElt)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| elt | Element | Элемент, стиль которого нужно изменить. Этот параметр не может быть нулевым. |
| pseudoElt | String | Псевдоэлемент или нуль, если нет. |

### Возвращаемое значение

Объявление стиля переопределения
