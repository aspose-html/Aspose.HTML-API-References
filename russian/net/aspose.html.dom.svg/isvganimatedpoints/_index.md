## ISVGAnimatedPoints interface

Интерфейс SVGAnimatedPoints поддерживает элементы, которые имеют атрибут 'points', который содержит список значений координат и поддерживает возможность анимации этого атрибута. Кроме того, атрибут 'points' исходного элемента, доступ к которому осуществляется через XML DOM (например, с помощью вызова метода getAttribute()), будет отражать любые изменения, внесенные в точки.
