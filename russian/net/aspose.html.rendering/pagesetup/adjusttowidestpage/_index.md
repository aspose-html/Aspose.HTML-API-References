## PageSetup.AdjustToWidestPage property

Получает или устанавливает флаг, определяющий случай, когда размер страницы будет скорректирован до самой широкой страницы в документе. Этот вариант требует много времени, поэтому время обработки документов может быть увеличено в два раза. Регулировка будет иметь место, только если самая широкая страница в документе шире, чем размер страницы, определенный в[`PageSetup`](../../pagesetup). Скорректированный размер страницы будет использоваться для всех страниц в документе.

public bool AdjustToWidestPage { get; set; }
