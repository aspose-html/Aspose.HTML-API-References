## PageSetup.PageLayoutOptions property

Получает или задает[`PageLayoutOptions`](../../pagelayoutoptions/) . Значение по умолчаниюNone , любое другое значение переопределит[`AdjustToWidestPage`](../adjusttowidestpage/) поведение. Работает только с документами HTML.

public PageLayoutOptions PageLayoutOptions { get; set; }
