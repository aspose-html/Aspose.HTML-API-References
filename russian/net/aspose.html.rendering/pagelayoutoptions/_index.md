## PageLayoutOptions enumeration

Указывает флаги, которые вместе с другими параметрами PageSetup определяют размеры и макеты страниц. Эти флаги можно комбинировать в соответствии с их описанием.

```csharp
[Flags]
public enum PageLayoutOptions
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| None | `0` | Значение по умолчанию, указывающее, что PageLayoutOptions не повлияет на размеры и макеты страниц. |
| FitToContentWidth | `1` | Этот флаг указывает на то, что ширина страниц определяется размером самого контента, а не заданной шириной страницы. Ширина контента рассчитывается индивидуально для каждой страницы. |
| UseWidestPage | `2` | В сочетании сFitToContentWidthуказывает, что ширина каждой страницы будет одинаковой и будет быть равным самому широкому размеру контента среди всех страниц. |
| FitToWidestContentWidth | `3` | Этот флаг указывает, что ширина страницы определяется размером самого контента, а не заданной шириной страницы. Ширина каждой страницы будет одинаковой и будет равна максимальному размеру содержимого среди всех страниц. |
| FitToContentHeight | `10` | Этот флаг указывает, что высота страницы определяется исходя из размера самого содержимого, а не из заданной высоты страницы. Все содержимое документов будет размещено на одной странице, если установлен этот флаг. |
| ScaleToPageWidth | `100` | Этот флаг указывает, что содержимое документа будет масштабироваться для размещения на странице, где разница между доступной шириной страницы и перекрывающимся содержимым является наибольшей. Он сталкивается с флагомFitToContentWidth, если указаны толькоScaleToPageWidthповлияет. |
| ScaleToPageHeight | `1000` | Этот флаг указывает, что содержимое документа будет масштабировано по высоте первой страницы. Он конфликтует с флагомFitToContentHeightи, если указаны оба флага, толькоScaleToPageHeightповлияет. Все содержимое документа будет размещено только на одной странице. |
