## SVGPoint class

Многие интерфейсы SVG DOM ссылаются на объекты класса SVGPoint. SVGPoint — это пара координат (x, y). При использовании в матричных операциях SVGPoint рассматривается как вектор вида: [x] [y] [1] Если объект SVGRect назначен только для чтения, то попытка присвоить значение одному из его атрибутов приведет к возникновению исключения.
