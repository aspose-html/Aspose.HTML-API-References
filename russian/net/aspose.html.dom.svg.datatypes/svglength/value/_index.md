## SVGLength.Value property

Значение в виде числа с плавающей запятой в пользовательских единицах. Установка этого атрибута приведет к автоматическому обновлению valueInSpecifiedUnits и valueAsString в соответствии с этим параметром.

```csharp
public float Value { get; set; }
```
