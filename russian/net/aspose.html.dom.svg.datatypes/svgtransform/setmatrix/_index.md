## SVGTransform.SetMatrix method

Устанавливает тип преобразования в SVG_TRANSFORM_MATRIX с матрицей параметров, определяющей новое преобразование. Копируются значения из матрицы параметров, параметр матрицы не заменяет SVGTransform::matrix.

```csharp
public void SetMatrix(SVGMatrix matrix)
```
