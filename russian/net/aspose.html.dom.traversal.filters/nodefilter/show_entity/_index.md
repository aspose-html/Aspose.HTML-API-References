## NodeFilter.SHOW_ENTITY field

Показать узлы Entity. Это имеет смысл только при создании итератора или обходчика дерева с узлом Entity в качестве его корня; в данном случае это означает, что узел Entity появится в первой позиции обхода. Поскольку сущности не являются частью дерева документа, они не появляются при обходе по дереву документа.

public const long SHOW_ENTITY;
