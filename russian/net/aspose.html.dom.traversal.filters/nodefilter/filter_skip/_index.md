## NodeFilter.FILTER_SKIP field

Пропустить этот единственный узел. Методы навигации, определенные для NodeIterator или TreeWalker, не будут возвращать этот узел. И для NodeIterator, и для TreeWalker дочерние элементы этого узла по-прежнему будут учитываться .

```csharp
public const short FILTER_SKIP;
```
