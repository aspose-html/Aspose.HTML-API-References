## NodeFilter.FILTER_REJECT field

Отклонить узел. Методы навигации, определенные для NodeIterator или TreeWalker, не будут возвращать этот узел. Для TreeWalker дети этого узла также будут отклонены. Итераторы узлов рассматривают это как синоним для FILTER_SKIP.

```csharp
public const short FILTER_REJECT;
```
