**Aspose.Html.Rendering.Doc** Пространство имен предоставляет определенный класс устройства , а также несколько классов параметров rendering , отвечающих за рендеринг в документ DOCX.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [DocDevice](./docdevice/) | Представляет рендеринг в документ DOCX. |
| [DocRenderingOptions](./docrenderingoptions/) | Представляет параметры рендеринга для[`DocDevice`](../aspose.html.rendering.doc/docdevice/) . |
## перечисление

| перечисление | Описание |
| --- | --- |
| [DocumentFormat](./documentformat/) | Представляет формат файла выходного документа. |
| [FontEmbeddingRule](./fontembeddingrule/) | Представляет правила внедрения шрифтов. |
