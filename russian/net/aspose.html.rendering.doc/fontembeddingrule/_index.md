## FontEmbeddingRule enumeration

Представляет правила внедрения шрифтов.

```csharp
public enum FontEmbeddingRule
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| Full | `1` | Все используемые шрифты будут встроены как есть. Имейте в виду, что некоторые шрифты могут быть лицензированы, это сделает результирующий документ нередактируемым. Также встроенные файлы шрифтов могут значительно увеличить размер результирующего документа. |
| None | `2` | Шрифты не будут встроены. |
