## Aspose.HTML para .NET

{{% alert color="primary" %}} 

![Aspose.HTML para el logotipo del producto .NET](../home_1.png)


Aspose.HTML para .NET es una biblioteca de clases multiplataforma que permite que sus aplicaciones realicen una amplia gama de tareas de manipulación de HTML. Dado que HTML generalmente se usa en combinación con otras tecnologías, Aspose.HTML admite no solo las especificaciones HTML5 sino también CSS3, SVG y HTML Canvas. Con Aspose.HTML, puede cargar páginas web, analizar y editar el contenido, guardar y convertir páginas desde y hacia los siguientes formatos: HTML, XHTML, MHTML, SVG, Markdown, PDF, XPS,
EPUB, PNG, TIFF, JPEG, BMP y otros formatos.

## Aspose.HTML para Java

{{% alert color="primary" %}} 

![Logotipo del producto Aspose.HTML para Java](../home_2.png)


Aspose.HTML para Java es una API avanzada de manipulación de HTML creada para realizar una amplia gama de tareas de manipulación de HTML directamente dentro de las aplicaciones de Java. La API también proporciona un motor de renderizado de alta fidelidad para formatos de diseño fijo, como PDF y XPS, y varios formatos de imágenes de trama.
