## GraphicContext.MiterLimit property

Establece u obtiene la longitud máxima de las uniones de líneas a inglete para trayectos trazados. Este parámetro limita la longitud de los "picos" producidos cuando los segmentos de línea se unen en ángulos agudos.

```csharp
public virtual float MiterLimit { get; set; }
```
