## PageSetup.AdjustToWidestPage property

Obtiene o establece el indicador que determina el caso en que el tamaño de la página se ajustará a la página más ancha del documento. Esta opción requiere mucho tiempo, por lo que el tiempo de procesamiento del documento se puede aumentar dos veces. El ajuste tendrá lugar solo si la página más ancha del documento es más ancha que el tamaño de página determinado en[`PageSetup`](../) . Se utilizará el tamaño de página ajustado para todas las páginas del documento.

```csharp
public bool AdjustToWidestPage { get; set; }
```
