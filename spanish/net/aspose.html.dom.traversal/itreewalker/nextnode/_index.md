## ITreeWalker.NextNode method

Mueve el TreeWalker al siguiente nodo visible en el orden document relativo al nodo actual y devuelve el nuevo nodo. Si el nodo actual no tiene un nodo siguiente, o si la búsqueda de nextNode intenta subir desde el nodo raíz del TreeWalker, devuelve un valor nulo y retiene el nodo actual.

```csharp
public Node NextNode()
```

### Valor_devuelto

El nuevo nodo, o nulo si el nodo actual no tiene siguiente nodo en la vista lógica del TreeWalker.
