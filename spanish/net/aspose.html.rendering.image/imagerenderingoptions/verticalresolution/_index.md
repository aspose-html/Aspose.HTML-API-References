## ImageRenderingOptions.VerticalResolution property

Establece u obtiene la resolución vertical para imágenes de salida e internas (que se utilizan durante el procesamiento de filtros), en píxeles por pulgada. Por defecto esta propiedad es 300 dpi.

```csharp
public override Resolution VerticalResolution { get; set; }
```
