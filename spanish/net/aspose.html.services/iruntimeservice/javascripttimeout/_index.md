## IRuntimeService.JavaScriptTimeout property

Obtiene o estableceTimeSpanque limita el tiempo de ejecución de JavaScript. Si el script se ejecuta durante más tiempo del proporcionadoTimeSpan , será cancelado. Puede especificar un tiempo de espera infinito configurandoTimeSpan igual a -1 milisegundo. El valor predeterminado es 1 minuto.
