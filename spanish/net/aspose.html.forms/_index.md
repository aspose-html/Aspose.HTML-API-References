El espacio de nombres proporciona acceso a numerosos objetos (elementos) que son responsables de la interactividad del usuario dentro del formulario como componente de la página web.

## Clases

| Clase | Descripción |
| --- | --- |
| [ButtonElement](./buttonelement/) | ButtonElement representa un envoltorio que está asociado con HTMLButtonElement. |
| [DataListElement](./datalistelement/) | DataListElement representa un contenedor asociado con HTMLDataListElement |
| [FormEditor](./formeditor/) | Esta clase representa el editor sobre el[`HTMLFormElement`](../aspose.html/htmlformelement/) que crea una forma más fácil para que los desarrolladores de .net editen los formularios html. |
| [FormElement](./formelement/) | Representa la clase base para elementos de formulario. |
| [FormElement&lt;T&gt;](./formelement-1/) | El[`FormElement`](../aspose.html.forms/formelement/) representar el campo genérico |
| [FormSubmitter](./formsubmitter/) | Esta clase permite preparar determinados[`HTMLFormElement`](../aspose.html/htmlformelement/) , recopila valores del elemento de formulario, los envía al servidor remoto y recibe una respuesta. |
| [InputElement](./inputelement/) | InputElement representa un contenedor asociado con HTMLInputElement. |
| [OptionCollection](./optioncollection/) | OptionElements representa un contenedor asociado con IHTMLOptionsCollection |
| [OptionElement](./optionelement/) | OptionElement representa un contenedor que está asociado con HTMLOptionElement |
| [SelectElement](./selectelement/) | SelectElement representa un contenedor que está asociado con HTMLSelectElement |
| [SubmissionResult](./submissionresult/) | Esta clase representa el resultado del envío de datos del formulario al servidor. |
| [TextAreaElement](./textareaelement/) | TextAreaElement representa un contenedor asociado con HTMLTextAreaElement |
