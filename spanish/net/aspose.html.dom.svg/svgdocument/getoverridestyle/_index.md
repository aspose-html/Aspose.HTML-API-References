## SVGDocument.GetOverrideStyle method

Este método se utiliza para recuperar la declaración de estilo anulada para un elemento especificado y un pseudoelemento especificado.

```csharp
public ICSSStyleDeclaration GetOverrideStyle(Element elt, string pseudoElt)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| elt | Element | El elemento cuyo estilo se va a modificar. Este parámetro no puede ser nulo. |
| pseudoElt | String | El pseudo-elemento o nulo si ninguno. |

### Valor_devuelto

La declaración de estilo de anulación
