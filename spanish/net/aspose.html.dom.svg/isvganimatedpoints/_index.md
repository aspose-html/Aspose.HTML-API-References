## ISVGAnimatedPoints interface

La interfaz SVGAnimatedPoints admite elementos que tienen un atributo de "puntos" que contiene una lista de valores de coordenadas y que admite la capacidad de animar ese atributo. Además, el atributo de "puntos" en el elemento original al que se accede a través del XML DOM (p. utilizando la llamada al método getAttribute()) reflejará cualquier cambio realizado en points.
