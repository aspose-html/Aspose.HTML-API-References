## IStyleSheet.OwnerNode property

El nodo que asocia esta hoja de estilo con el documento. Para HTML, este puede ser el elemento LINK o STYLE correspondiente. Para XML, puede ser la instrucción de procesamiento de enlace. Para las hojas de estilo que están incluidas en otras hojas de estilo, el valor de este atributo es nulo.
