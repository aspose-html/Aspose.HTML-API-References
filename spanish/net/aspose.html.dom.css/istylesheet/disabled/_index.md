## IStyleSheet.Disabled property

falso si la hoja de estilo se aplica al documento. cierto si no lo es. La modificación de este atributo puede provocar una nueva resolución de estilo para el documento. Una hoja de estilo solo se aplica si está presente una definición de medio adecuada y el atributo deshabilitado es falso. Por lo tanto, si los medios no se aplican al agente de usuario actual, se ignora el atributo deshabilitado.

```csharp
public bool Disabled { get; set; }
```

### El valor de la propiedad

`verdadero` si está deshabilitado; de lo contrario,`falso` .
