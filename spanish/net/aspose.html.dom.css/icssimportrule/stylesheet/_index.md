## ICSSImportRule.StyleSheet property

La hoja de estilo a la que hace referencia esta regla, si se ha cargado. El valor de este atributo es nulo si la hoja de estilo aún no se ha cargado o si no se cargará (p. ej., si la hoja de estilo es para un tipo de medio no compatible con el agente de usuario).

```csharp
public ICSSStyleSheet StyleSheet { get; }
```
