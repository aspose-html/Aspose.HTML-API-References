## ICSSPageRule interface

La interfaz CSSPageRule representa una regla @page dentro de una hoja de estilo CSS. La regla @page se usa para especificar las dimensiones, la orientación, los márgenes, etc. de un cuadro de página para medios paginados.

public interface ICSSPageRule : ICSSRule
