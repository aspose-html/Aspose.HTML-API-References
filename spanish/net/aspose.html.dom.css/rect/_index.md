## Rect class

La interfaz Rect se utiliza para representar cualquier valor rect. Esta interfaz refleja los valores en la propiedad de estilo subyacente. Por lo tanto, las modificaciones realizadas a los objetos CSSPrimitiveValue modifican la propiedad de estilo.

```csharp
public class Rect : DOMObject
```
