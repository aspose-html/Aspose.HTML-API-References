## IDocumentStyle interface

La interfaz DocumentStyle proporciona un mecanismo mediante el cual se pueden recuperar las hojas de estilo incrustadas en un documento. La expectativa es que se pueda obtener una instancia de la interfaz DocumentStyle mediante el uso de métodos de conversión específicos de enlace en una instancia de la interfaz Document.
