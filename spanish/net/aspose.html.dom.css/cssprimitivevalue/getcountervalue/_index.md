## CSSPrimitiveValue.GetCounterValue method

Este método se utiliza para obtener el valor del contador. Si este valor de CSS no contiene un valor de contador, se genera una DOMException. La modificación de la propiedad de estilo correspondiente se puede lograr usando la interfaz Counter.

```csharp
public abstract Counter GetCounterValue()
```

### Valor_devuelto

Devuelve el valor del contador
