## ILinkStyle interface

La interfaz LinkStyle proporciona un mecanismo mediante el cual se puede recuperar una hoja de estilo del nodo responsable de vincularla a un documento. Se puede obtener una instancia de la interfaz LinkStyle utilizando métodos de conversión específicos de enlace en una instancia de un nodo de enlace (HTMLLinkElement, HTMLStyleElement o ProcessingInstruction en DOM Nivel 2).
