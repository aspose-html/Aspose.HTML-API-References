## XpsDevice.Fill method

Rellena toda la región encerrada por la ruta actual. Si el camino consta de varios subcaminos desconectados, llena el interior de todos los subcaminos, considerados juntos. Este método finaliza la ruta actual.

```csharp
public override void Fill(FillMode mode)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| mode | FillMode | El modo de relleno especifica cómo se llena el interior de un camino cerrado |
