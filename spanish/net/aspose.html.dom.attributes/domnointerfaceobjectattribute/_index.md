## DOMNoInterfaceObjectAttribute class

Si el atributo extendido [NoInterfaceObject] aparece en una interfaz, indica que no existirá un objeto de interfaz para la interfaz en el enlace de ECMAScript.

```csharp
[AttributeUsage(AttributeTargets.All)]
public sealed class DOMNoInterfaceObjectAttribute : Attribute
```
