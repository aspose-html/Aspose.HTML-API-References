## IXPathNSResolver.LookupNamespaceURI method

Busque el URI del espacio de nombres asociado al prefijo del espacio de nombres proporcionado. El evaluador XPath nunca debe llamar a esto con un`nulo` o vacía el argumento , porque el resultado de hacer esto no está definido.

```csharp
public string LookupNamespaceURI(string prefix)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| prefix | String | El prefijo a buscar. |

### Valor_devuelto

Devuelve el URI del espacio de nombres asociado o`nulo` si no se encuentra ninguno .
