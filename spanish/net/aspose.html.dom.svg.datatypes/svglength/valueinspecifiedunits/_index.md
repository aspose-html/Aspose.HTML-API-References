## SVGLength.ValueInSpecifiedUnits property

El valor como valor de punto flotante, en las unidades expresadas por unitType. La configuración de este atributo hará que value y valueAsString se actualicen automáticamente para reflejar esta configuración.

```csharp
public float ValueInSpecifiedUnits { get; set; }
```

### El valor de la propiedad

El valor en unidades especificadas.
