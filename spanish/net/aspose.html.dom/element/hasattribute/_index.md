## Element.HasAttribute method

Devuelve verdadero cuando se especifica un atributo con un nombre determinado en este elemento o tiene un valor predeterminado; de lo contrario, devuelve falso.

```csharp
public bool HasAttribute(string name)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| name | String | El nombre del atributo. |

### Valor_devuelto

`verdadero` si el nombre especificado tiene atributo; de lo contrario,`FALSO` .
