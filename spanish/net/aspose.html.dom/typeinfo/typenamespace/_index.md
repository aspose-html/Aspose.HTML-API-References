## TypeInfo.TypeNamespace property

Obtiene el espacio de nombres de tipo. El espacio de nombres del tipo declarado para el elemento o atributo asociado o nulo si el elemento no tiene declaración o si no hay información de espacio de nombres disponible.

```csharp
public string TypeNamespace { get; }
```

### El valor de la propiedad

El espacio de nombres de tipo.
