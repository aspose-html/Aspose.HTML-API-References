## Node.ReplaceChild method

Reemplaza el nodo secundario oldChild con newChild en la lista de elementos secundarios y devuelve el nodo oldChild. Si newChild es un objeto DocumentFragment, oldChild se reemplaza por todos los hijos de DocumentFragment, que se insertan en el mismo orden. Si newChild ya está en el árbol, primero se elimina.

```csharp
public Node ReplaceChild(Node node, Node child)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| node | Node | El nuevo nodo. |
| child | Node | El niño viejo. |
