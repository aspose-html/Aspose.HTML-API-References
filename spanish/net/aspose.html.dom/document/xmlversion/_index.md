## Document.XmlVersion property

Un atributo que especifica, como parte de la declaración XML, el número de versión de este documento. Si no hay declaración y si este documento admite la función "XML", el valor es "1.0". Si este documento no es compatible con la función "XML", el valor siempre es nulo.

```csharp
public string XmlVersion { get; set; }
```

### El valor de la propiedad

La versión XML.
