Además de las herramientas estándar de E/S (entrada-salida) , **Aspose.Html.IO** el espacio de nombres contiene clases auxiliares e interfaces.

## Clases

| Clase | Descripción |
| --- | --- |
| [FileCreateStreamProvider](./filecreatestreamprovider/) | RepresentaFileStream implementación para proporcionar flujos a los dispositivos de representación. |
| [OutputStream](./outputstream/) | Un flujo sustituto envuelve el flujo de salida real y controla el acceso a él. [`OutputStream`](../aspose.html.io/outputstream/) contiene datos de URI que describen la ubicación del flujo de salida. |
| [OutputStreamContext](./outputstreamcontext/) | Un contexto para la inicialización del flujo de salida. |
## Interfaces

| Interfaz | Descripción |
| --- | --- |
| [IBlob](./iblob/) | Un objeto Blob se refiere a una secuencia de bytes y tiene un atributo de tamaño que es el número total de bytes en la secuencia de bytes y un atributo de tipo, que es una cadena codificada en ASCII en minúsculas que representa el tipo de medio de la secuencia de bytes . |
| [ICreateStreamProvider](./icreatestreamprovider/) | Representa una interfaz que pueden implementar las clases que proporcionan flujos a los dispositivos de representación. |
| [IFile](./ifile/) | Un objeto File es un objeto Blob con un atributo de nombre, que es una cadena; se puede crear dentro de la aplicación web a través de un constructor, o es una referencia a una secuencia de bytes de un archivo del sistema de archivos subyacente (SO). |
| [IFileList](./ifilelist/) | Representa la interfaz para la lista de archivos. |
| [IOutputStorage](./ioutputstorage/) | Proporciona la creación y gestión de flujos de salida[`OutputStream`](../aspose.html.io/outputstream/) . |
