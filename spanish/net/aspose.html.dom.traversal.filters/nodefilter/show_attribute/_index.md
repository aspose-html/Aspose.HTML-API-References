## NodeFilter.SHOW_ATTRIBUTE field

Mostrar nodos Attr. Esto es significativo solo cuando se crea un iterador o un caminante de árboles con un nodo de atributo como su raíz ; en este caso, significa que el nodo de atributo aparecerá en la primera posición de la iteración o recorrido. Dado que los atributos nunca son elementos secundarios de otros nodos, no aparecen al atravesar el árbol del documento.

public const long SHOW_ATTRIBUTE;
