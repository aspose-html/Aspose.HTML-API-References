## TemplateLoadOptions.NamesAreCaseSensitive property

Propiedad que especifica si la plantilla y los nombres de los elementos de datos coincidirán independientemente del uso de mayúsculas y minúsculas o no. Si es verdadero, la coincidencia distingue entre mayúsculas y minúsculas; si es falso, se ignorarán las mayúsculas y minúsculas. El valor predeterminado es verdadero.

```csharp
public bool NamesAreCaseSensitive { get; set; }
```
