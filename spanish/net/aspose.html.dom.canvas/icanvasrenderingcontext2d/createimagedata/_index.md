## CreateImageData(double, double) {#createimagedata_1}

Crea un nuevo objeto ImageData en blanco con las dimensiones especificadas. Todos los píxeles del nuevo objeto son negros transparentes.

```csharp
public IImageData CreateImageData(double sw, double sh)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| sw | Double | El ancho para dar el nuevo objeto ImageData. |
| sh | Double | La altura para dar el nuevo objeto ImageData. |

### Valor_devuelto

Un nuevo objeto ImageData con el ancho y el alto especificados. El nuevo objeto se rellena con píxeles negros transparentes.
