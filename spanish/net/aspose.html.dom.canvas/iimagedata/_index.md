## IImageData interface

Crea un objeto ImageData a partir de un Uint8ClampedArray dado y el tamaño de la imagen que contiene. Si no se proporciona una matriz, crea una imagen de un rectángulo negro.

```csharp
public interface IImageData
```

## Propiedades

| Nombre | Descripción |
| --- | --- |
| [Data](../../aspose.html.dom.canvas/iimagedata/data/) { get; } | Es un Uint8ClampedArray que representa una matriz unidimensional que contiene los datos en el orden RGBA, con valores enteros entre 0 y 255 (incluidos). |
| [Height](../../aspose.html.dom.canvas/iimagedata/height/) { get; } | Es un largo sin signo que representa la altura real, en píxeles, de ImageData. |
| [Width](../../aspose.html.dom.canvas/iimagedata/width/) { get; } | Es un largo sin signo que representa el ancho real, en píxeles, de ImageData. |
