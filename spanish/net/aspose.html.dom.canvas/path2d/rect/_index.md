## Path2D.Rect method

```csharp
public void Rect(double x, double y, double w, double h)
```

| Parámetro | Escribe | Descripción |

| x | Double | El eje x de la coordenada para el punto inicial del rectángulo. |
| y | Double | El eje y de la coordenada para el punto inicial del rectángulo. |
| w | Double | El ancho del rectángulo. |
| h | Double | La altura del rectángulo. |
