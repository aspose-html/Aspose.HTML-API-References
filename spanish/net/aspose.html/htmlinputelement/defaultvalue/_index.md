## HTMLInputElement.DefaultValue property

Cuando el`tipo`atributo del elemento tiene el valor "texto", "archivo" o "contraseña", esto representa el valor HTML atributo del elemento. El valor de este atributo no cambia si cambia el contenido del control de formulario correspondiente, en un agente de usuario interactivo. Consulte la definición del atributo de valor en HTML 4.01.

```csharp
public string DefaultValue { get; set; }
```
