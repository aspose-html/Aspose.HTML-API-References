## HTMLInputElement.Files property

El atributo IDL de archivos permite que los scripts accedan a los archivos seleccionados del elemento. Al obtener, si se aplica el atributo IDL, debe devolver un objeto FileList que represente los archivos seleccionados actualmente. Se debe devolver el mismo objeto hasta que cambie la lista de archivos seleccionados. Si el atributo IDL no se aplica, debe devolver un valor nulo. [API DE ARCHIVO]

```csharp
public IFileList Files { get; }
```
