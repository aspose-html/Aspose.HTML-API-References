## HTMLSelectElement.SelectedIndex property

El índice ordinal de la opción seleccionada, comenzando desde 0. El valor -1 se devuelve si no se selecciona ningún elemento. Si se seleccionan varias opciones , se devuelve el índice de la primera opción seleccionada.

```csharp
public int SelectedIndex { get; set; }
```
