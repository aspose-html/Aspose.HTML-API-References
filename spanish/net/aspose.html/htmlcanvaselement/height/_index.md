## HTMLCanvasElement.Height property

Es un número entero positivo que refleja el atributo HTML de altura del elemento de lienzo interpretado en píxeles CSS. Cuando no se especifica el atributo, o si se establece en un valor no válido, como un negativo, se usa el valor predeterminado de 150.

public ulong Height { get; set; }
