## HTMLCanvasElement.Width property

Es un número entero positivo que refleja el atributo HTML de ancho del elemento de lienzo interpretado en píxeles CSS. Cuando no se especifica el atributo, o si se establece en un valor no válido, como un negativo, se utiliza el valor predeterminado de 300.

```csharp
public ulong Width { get; set; }
```
