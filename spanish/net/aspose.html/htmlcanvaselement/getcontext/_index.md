## HTMLCanvasElement.GetContext method

Devuelve un contexto de dibujo en el lienzo o nulo si no se admite el ID de contexto. Un contexto de dibujo le permite dibujar en el lienzo.

```csharp
public object GetContext(string contextId)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| contextId | String | Es una cadena que contiene el identificador de contexto que define el contexto de dibujo asociado al lienzo. |

### Valor_devuelto

Un contexto de representación
