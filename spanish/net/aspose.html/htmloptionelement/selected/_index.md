## HTMLOptionElement.Selected property

Representa el estado actual del control de formulario correspondiente, en un agente de usuario interactivo . Cambiar este atributo cambia el estado de el control de formulario, pero no cambia el valor del atributo HTML seleccionado del elemento.

```csharp
public bool Selected { get; set; }
```
