## HTMLDocument.Images property

Una colección de todos los`IMG` elementos en un documento. El comportamiento de está limitado a`IMG` elementos para compatibilidad con versiones anteriores de . Como sugiere [[HTML 4.01](http://www.w3.org/TR/1999/REC-html401-19991224)], para incluir imágenes, los autores pueden usar el`OBJETO` elemento o el`IMG` elemento. Por lo tanto, se recomienda no utilizar este atributo para buscar las imágenes en el documento, sino`getElementsByTagName` con HTML 4.01 o`getElementsByTagNameNS` con XHTML 1.0.
