## Save(Url) {#save_5}

Guarda el documento en el archivo local especificado por`URL` Todos los recursos utilizados en este documento se guardarán en en la carpeta adyacente, cuyo nombre se construirá como: nombre_archivo_salida + "_archivos".

```csharp
public void Save(Url url)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| url | Url | URL local al archivo de salida. |

### Excepciones

| excepción | condición |
| --- | --- |
| ArgumentException | Aumentado si el especificado`URL` no es una URL de archivo local válida. |
