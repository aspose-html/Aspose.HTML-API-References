## HTMLDocument.Referrer property

Devuelve la URI [[IETFRFC 2396](http://www.ietf.org/rfc/rfc2396.txt) de la página que enlaza con esta página. El valor es una cadena vacía si el usuario navegó a la página directamente (no a través de un enlace , sino, por ejemplo, a través de un marcador).

```csharp
public string Referrer { get; }
```
