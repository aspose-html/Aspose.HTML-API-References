## Node.PreviousSibling property

The read-only previousSibling property of the [`Node`](../../node) interface returns the node immediately preceding the specified one in its parent's [`childNodes`](../firstchild) list, or null if the specified node is the first in that list.

```csharp
public Node PreviousSibling { get; }
```

## Property Value

A [`Node`](../../node) representing the previous sibling of the current node, or null if there are none.
