## ICSSRuleList indexer

Used to retrieve a CSS rule by method item() (http://www.w3.org/TR/DOM-Level-2-Style/css.html#CSS-CSSRuleList). The order in this collection represents the order of the rules in the CSS style sheet. If index is greater than or equal to the number of rules in the list, this returns null.

```csharp
public ICSSRule this[int index] { get; }
```

### Return Value

The item(index) method must return the indexth CSSRule object in the collection. If there is no indexth object in the collection, then the method must return null.

### Property Value

The index into the collection of CSS rules.
