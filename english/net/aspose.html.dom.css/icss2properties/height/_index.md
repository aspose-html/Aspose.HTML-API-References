## ICSS2Properties.Height property

This property specifies the [content height](https://www.w3.org/TR/1998/REC-CSS2-19980512/box.html#content-height) of boxes generated by block-level and [replaced](https://www.w3.org/TR/1998/REC-CSS2-19980512/conform.html#replaced-element) elements.

This property does not apply to non-replaced [inline-level](https://www.w3.org/TR/1998/REC-CSS2-19980512/visuren.html#inline-level) elements. The height of a non-replaced inline element's boxes is given by the element's (possibly inherited) ['line-height'](https://www.w3.org/TR/1998/REC-CSS2-19980512/visudet.html#propdef-line-height) value.
