## ICSS2Properties.CounterIncrement property

The ['counter-increment'](https://www.w3.org/TR/1998/REC-CSS2-19980512/generate.html#propdef-counter-increment) property accepts one or more names of counters (identifiers), each one optionally followed by an integer. The integer indicates by how much the counter is incremented for every occurrence of the element. The default increment is 1. Zero and negative integers are allowed.
