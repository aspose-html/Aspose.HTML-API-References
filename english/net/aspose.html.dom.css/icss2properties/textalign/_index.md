## ICSS2Properties.TextAlign property

This property describes how inline content of a block is aligned. Values have the following meanings:

left, right, center, and justify - Left, right, center, and double justify text, respectively.'[string](https://www.w3.org/TR/1998/REC-CSS2-19980512/syndata.html#value-def-string)' - Specifies a string on which cells in a table column will align (see the section on [horizontal alignment in a column](https://www.w3.org/TR/1998/REC-CSS2-19980512/tables.html#column-alignment) for details and an example). This value applies only to [table](https://www.w3.org/TR/1998/REC-CSS2-19980512/tables.html) cells. If set on other elements, it will be treated as 'left' or 'right', depending on whether ['direction'](https://www.w3.org/TR/1998/REC-CSS2-19980512/visuren.html#propdef-direction) is 'ltr', or 'rtl', respectively.
