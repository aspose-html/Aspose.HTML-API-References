## ICSS2Properties.ListStyleImage property

This property sets the image that will be used as the list item marker. When the image is available, it will replace the marker set with the ['list-style-type'](https://www.w3.org/TR/1998/REC-CSS2-19980512/generate.html#propdef-list-style-type) marker.
