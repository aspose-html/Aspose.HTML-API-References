## ICSS2Properties.TextIndent property

This property specifies the indentation of the first line of text in a block. More precisely, it specifies the indentation of the first box that flows into the block's first [line box](https://www.w3.org/TR/1998/REC-CSS2-19980512/visuren.html#line-box). The box is indented with respect to the left (or right, for right-to-left layout) edge of the line box. User agents should render this indentation as blank space.

Values have the following meanings:

'[length](https://www.w3.org/TR/1998/REC-CSS2-19980512/syndata.html#value-def-length)' - The indentation is a fixed length.'[percentage](https://www.w3.org/TR/1998/REC-CSS2-19980512/syndata.html#value-def-percentage)' - The indentation is a percentage of the containing block width.
