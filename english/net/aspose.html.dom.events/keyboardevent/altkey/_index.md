## KeyboardEvent.AltKey property

true if the Alt (alternative) (or "Option") key modifier was active. The un-initialized value of this attribute MUST be false.

```csharp
public bool AltKey { get; }
```

### Property Value

`true` if Alt; otherwise, `false`.
