## HTMLObjectElement class

Generic embedded object.In principle, all properties on the object element are read-write but in some environments some properties may be read-only once the underlying object is instantiated. See the OBJECT element definition in [[HTML 4.01](http://www.w3.org/TR/1999/REC-html401-19991224)].

See also the [Document object Model (DOM) Level 2 HTML Specification](http://www.w3.org/TR/2003/REC-DOM-Level-2-HTML-20030109).

```csharp
public class HTMLObjectElement : HTMLElement
```
