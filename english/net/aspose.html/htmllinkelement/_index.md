## HTMLLinkElement class

The `LINK` element specifies a link to an external resource, and defines this document's relationship to that resource (or vice versa). See the LINK element definition in HTML 4.01 (see also the `LinkStyle` interface in the StyleSheet module [[DOM Level 2 Style Sheets and CSS](http://www.w3.org/TR/2000/REC-DOM-Level-2-Style-20001113)]).

See also the [Document object Model (DOM) Level 2 HTML Specification](http://www.w3.org/TR/2003/REC-DOM-Level-2-HTML-20030109).

```csharp
public class HTMLLinkElement : HTMLElement, ILinkStyle
```
