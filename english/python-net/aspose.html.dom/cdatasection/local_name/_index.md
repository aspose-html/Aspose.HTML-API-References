## local_name property


Returns the local part of the qualified name of this node. 
For nodes of any type other than [`Node.ELEMENT_NODE`](/html/python-net/aspose.html.dom/node) and [`Node.ATTRIBUTE_NODE`](/html/python-net/aspose.html.dom/node)
and nodes created with a DOM Level 1 method, such as [`Document.create_element`](/html/python-net/aspose.html.dom/document/create_element), this is always null.
