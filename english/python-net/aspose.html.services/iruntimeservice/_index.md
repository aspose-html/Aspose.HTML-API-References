## IRuntimeService class

This service is used to configure runtime related properties.



The IRuntimeService type exposes the following members:

### Properties

| :- | :- |
| [java_script_timeout](/html/python-net/aspose.html.services/iruntimeservice/java_script_timeout) | Gets or sets TimeSpan which limits JavaScript execution time. If script is executed longer than provided TimeSpan, it will be cancelled. You can specify infinite timeout by setting TimeSpan equal to -1 millisecond. Default value is 1 minute. |
