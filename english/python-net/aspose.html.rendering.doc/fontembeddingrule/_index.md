## FontEmbeddingRule enumeration

Represents the font embedding rules.



The FontEmbeddingRule type exposes the following members:

### Fields
| Field | Description |
| :- | :- |
| FULL | All the used fonts will be embedded as-is. Be aware that some font could be licensed, this will make the resulting document uneditable. Also the embedded font files could significantly increase the size of the resulting document. |
| NONE | Fonts won't be embedded. |
