## Location class

Location objects provide a representation of the address of the active document of their Document's browsing context, and allow the current entry of the browsing context's session history to be changed, by adding or replacing entries in the history object.



**Inheritance:** [`Location`](/html/python-net/aspose.html.window/location) → 
[`DOMObject`](/html/python-net/aspose.html.dom/domobject)



The Location type exposes the following members:

### Properties
| Property | Description |
| :- | :- |
| [href](/html/python-net/aspose.html.window/location/href) | Returns the Location object's URL.<br/>Can be set, to navigate to the given URL. |
| [origin](/html/python-net/aspose.html.window/location/origin) | Returns the Location object's URL's origin. |
| [protocol](/html/python-net/aspose.html.window/location/protocol) | Returns the Location object's URL's scheme.<br/>Can be set, to navigate to the same URL with a changed scheme. |
| [host](/html/python-net/aspose.html.window/location/host) | Returns the Location object's URL's host and port (if different from the default port for the scheme).<br/>Can be set, to navigate to the same URL with a changed host and port. |
| [hostname](/html/python-net/aspose.html.window/location/hostname) | Returns the Location object's URL's host.<br/>Can be set, to navigate to the same URL with a changed host. |
| [port](/html/python-net/aspose.html.window/location/port) | Returns the Location object's URL's port.<br/>Can be set, to navigate to the same URL with a changed port. |
| [pathname](/html/python-net/aspose.html.window/location/pathname) | Returns the Location object's URL's path.<br/>Can be set, to navigate to the same URL with a changed path. |
| [search](/html/python-net/aspose.html.window/location/search) | Returns the Location object's URL's query (includes leading "?" if non-empty).<br/>Can be set, to navigate to the same URL with a changed query(ignores leading "?"). |
| [hash](/html/python-net/aspose.html.window/location/hash) | Returns the Location object's URL's fragment (includes leading "#" if non-empty).<br/>Can be set, to navigate to the same URL with a changed fragment(ignores leading "#"). |
