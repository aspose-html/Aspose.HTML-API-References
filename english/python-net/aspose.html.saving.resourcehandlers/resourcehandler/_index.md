## ResourceHandler class

This class is responsible for handling resources. It provides methods that allow you to control what will be done with the [`Resource`](/html/python-net/aspose.html.saving/resource), as well as what reference will be written to the parent [`Resource`](/html/python-net/aspose.html.saving/resource).
