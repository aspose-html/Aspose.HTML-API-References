## IImageData class

Creates an ImageData object from a given Uint8ClampedArray and the size of the image it contains. 
If no array is given, it creates an image of a black rectangle.



The IImageData type exposes the following members:

### Properties
| Property | Description |
| :- | :- |
| [width](/html/python-net/aspose.html.dom.canvas/iimagedata/width) | Is an unsigned long representing the actual width, in pixels, of the ImageData. |
| [height](/html/python-net/aspose.html.dom.canvas/iimagedata/height) | Is an unsigned long representing the actual height, in pixels, of the ImageData. |
| [data](/html/python-net/aspose.html.dom.canvas/iimagedata/data) | Is a Uint8ClampedArray representing a one-dimensional array containing the data in the RGBA order, <br/>with integer values between 0 and 255 (included). |
