The **Aspose.Html.Dom.Mutations**  provides opportunity to observe
DOM mutations and getting specific callback information.

### Classes
| Class | Description |
| :- | :- |
| [`MutationObserver`](/html/python-net/aspose.html.dom.mutations/mutationobserver) | A [`MutationObserver`](/html/python-net/aspose.html.dom.mutations/mutationobserver) object can be used to observe mutations to the tree of [`Node`](/html/python-net/aspose.html.dom/node). |
| [`MutationObserverInit`](/html/python-net/aspose.html.dom.mutations/mutationobserverinit) | This class represents an options collection which is used to configure [`MutationObserver`](/html/python-net/aspose.html.dom.mutations/mutationobserver). |
| [`MutationRecord`](/html/python-net/aspose.html.dom.mutations/mutationrecord) | A MutationRecord represents an individual DOM mutation. It is the object that is passed to [`MutationObserver`](/html/python-net/aspose.html.dom.mutations/mutationobserver)'s MutationCallback. |
