## page_layout_options property


Gets or sets the [`PageLayoutOptions`](/html/python-net/aspose.html.rendering/pagelayoutoptions). Default value is [`PageLayoutOptions.NONE`](/html/python-net/aspose.html.rendering/pagelayoutoptions#NONE), any other value will override the [`PageSetup.adjust_to_widest_page`](/html/python-net/aspose.html.rendering/pagesetup#adjust_to_widest_page) behaviour. Works only with HTML documents.
