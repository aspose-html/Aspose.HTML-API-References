## create_new {#aspose.html.HTMLDocument}

Creates a new [`HTMLFormElement`](/html/python-net/aspose.html/htmlformelement) and associated it with [`FormEditor`](/html/python-net/aspose.html.forms/formeditor). [`HTMLFormElement`](/html/python-net/aspose.html/htmlformelement) is created in the detached from the document state; in order to attach it to the document, please select proper location and use [`Node.append_child`](/html/python-net/aspose.html.dom/node/append_child) method.
