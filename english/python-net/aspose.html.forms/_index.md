The **Aspose.Html.Forms**  namespace provides access to
numerous objects (elements) which are responsible to
user interactivity within form as web page component.

### Classes
| Class | Description |
| :- | :- |
| [`ButtonElement`](/html/python-net/aspose.html.forms/buttonelement) | The ButtonElement represents a wrapper that is associated with the HTMLButtonElement. |
| [`DataListElement`](/html/python-net/aspose.html.forms/datalistelement) | The DataListElement represents a wrapper that is associated with the HTMLDataListElement |
| [`FormEditor`](/html/python-net/aspose.html.forms/formeditor) | This class represents the editor over the [`HTMLFormElement`](/html/python-net/aspose.html/htmlformelement) that creates a easier way for .net developers to edit the html forms. |
| [`FormElement`](/html/python-net/aspose.html.forms/formelement) | Represents base class for form elements. |
| [`FormSubmitter`](/html/python-net/aspose.html.forms/formsubmitter) | This class allows to prepare specified [`HTMLFormElement`](/html/python-net/aspose.html/htmlformelement), collects values from the form element, submit them to the remote server and receives a response. |
| [`InputElement`](/html/python-net/aspose.html.forms/inputelement) | The InputElement represents a wrapper that is associated with the HTMLInputElement. |
| [`OptionCollection`](/html/python-net/aspose.html.forms/optioncollection) | The OptionElements represents a wrapper that is associated with the IHTMLOptionsCollection |
| [`OptionElement`](/html/python-net/aspose.html.forms/optionelement) | The OptionElement represents a wrapper that is associated with the HTMLOptionElement |
| [`SelectElement`](/html/python-net/aspose.html.forms/selectelement) | The SelectElement represents a wrapper that is associated with the HTMLSelectElement |
| [`SubmissionResult`](/html/python-net/aspose.html.forms/submissionresult) | This class represents the result of the submitting form data to the server. |
| [`TextAreaElement`](/html/python-net/aspose.html.forms/textareaelement) | The TextAreaElement represents a wrapper that is associated with the HTMLTextAreaElement |
