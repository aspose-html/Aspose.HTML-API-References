## ICSSKeyframeRule interface

The `CSSKeyframeRule` interface describes an object representing a set of styles for a given keyframe. It corresponds to the contents of a single keyframe of a @keyframes at-rule.

```java
public interface ICSSKeyframeRule : ICSSRule
```

## Properties

| Name | Description |
| --- | --- |
| [getKeyText](../../com.aspose.html.dom.css/icsskeyframerule/keytext/) The keyText property of the CSSKeyframeRule interface represents the keyframe selector as a comma-separated list of percentage values. The from and to keywords map to 0% and 100%, respectively. |
| [getStyle](../../com.aspose.html.dom.css/icsskeyframerule/style/) The read-only CSSKeyframeRule.style property is the [`CSSStyleDeclaration`](../icssstyledeclaration/) interface for the declaration block of the `CSSKeyframeRule`. |
