## ICSS2Properties.Width property

This property specifies the [content width](https://www.w3.org/TR/1998/REC-CSS2-19980512/box.html#content-width) of boxes generated by block-level and [replaced](https://www.w3.org/TR/1998/REC-CSS2-19980512/conform.html#replaced-element) elements.

This property does not apply to non-replaced [inline-level](https://www.w3.org/TR/1998/REC-CSS2-19980512/visuren.html#inline-level) elements. The width of a non-replaced inline element's boxes is that of the rendered content within them (before any relative offset of children). Recall that inline boxes flow into [line boxes](https://www.w3.org/TR/1998/REC-CSS2-19980512/visuren.html#line-box). The width of line boxes is given by the their [containing block](https://www.w3.org/TR/1998/REC-CSS2-19980512/visuren.html#containing-block), but may be shorted by the presence of [floats](https://www.w3.org/TR/1998/REC-CSS2-19980512/visuren.html#floats).

The width of a replaced element's box is [intrinsic](https://www.w3.org/TR/1998/REC-CSS2-19980512/conform.html#intrinsic) and may be scaled by the user agent if the value of this property is different than 'auto'.

Values have the following meanings:

'[length](https://www.w3.org/TR/1998/REC-CSS2-19980512/syndata.html#value-def-length)' - Specifies a fixed width.'[percentage](https://www.w3.org/TR/1998/REC-CSS2-19980512/syndata.html#value-def-percentage)' - Specifies a percentage width. The percentage is calculated with respect to the width of the generated box's [containing block](https://www.w3.org/TR/1998/REC-CSS2-19980512/visuren.html#containing-block).auto - The width depends on the values of other properties. See the sections below.Note: Negative values for ['width'](https://www.w3.org/TR/1998/REC-CSS2-19980512/visudet.html#propdef-width) are illegal.
