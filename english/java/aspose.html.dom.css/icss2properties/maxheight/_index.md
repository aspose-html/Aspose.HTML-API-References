## ICSS2Properties.MaxHeight property

These two properties allow authors to constrain box heights to a certain range. Values have the following meanings:

'[length](https://www.w3.org/TR/1998/REC-CSS2-19980512/syndata.html#value-def-length)' - Specifies a fixed minimum or maximum computed height.'[percentage](https://www.w3.org/TR/1998/REC-CSS2-19980512/syndata.html#value-def-percentage)' - Specifies a percentage for determining the computed value. The percentage is calculated with respect to the height of the generated box's [containing block](https://www.w3.org/TR/1998/REC-CSS2-19980512/visuren.html#containing-block). If the height of the containing block is not specified explicitly (i.e., it depends on content height), the percentage value is interpreted like 'auto'.none - (Only on ['max-height'](https://www.w3.org/TR/1998/REC-CSS2-19980512/visudet.html#propdef-max-height)) No limit on the height of the box.
