## ICSS2Properties.LineHeight property

If the property is set on a [block-level](https://www.w3.org/TR/1998/REC-CSS2-19980512/visuren.html#block-level) element whose content is composed of [inline-level](https://www.w3.org/TR/1998/REC-CSS2-19980512/visuren.html#inline-level) elements, it specifies the minimal height of each generated inline box.

If the property is set on an [inline-level](https://www.w3.org/TR/1998/REC-CSS2-19980512/visuren.html#inline-level) element, it specifies the exact height of each box generated by the element. (Except for inline [replaced](https://www.w3.org/TR/1998/REC-CSS2-19980512/conform.html#replaced-element) elements, where the height of the box is given by the ['height'](https://www.w3.org/TR/1998/REC-CSS2-19980512/visudet.html#propdef-height) property.)

```java
public String LineHeight { get; set; }
```

### Return Value

line-height property
