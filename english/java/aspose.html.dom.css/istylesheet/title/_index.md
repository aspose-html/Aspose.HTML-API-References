## IStyleSheet.Title property

The title property of the [`StyleSheet`](../) interface returns the advisory title of the current style sheet.

The title is often specified in the [`ownerNode`](../ownernode/).

```java
public String Title { get; }
```

### Property Value

The parentStyleSheet attribute must return the parent [`CSS style sheet`](../../icssstylesheet/).
