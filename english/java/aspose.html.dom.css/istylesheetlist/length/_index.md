## IStyleSheetList.Length property

The length attribute must return the number of CSS style sheets represented by the collection. The range of valid child stylesheet indices is 0 to length-1 inclusive.

```java
public int Length { get; }
```

### Property Value

An integer indicating the number of items in the collection.
