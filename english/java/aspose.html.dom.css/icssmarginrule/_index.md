## ICSSMarginRule interface

The CSSMarginRule interface represents a margin at-rule (e.g. @top-left) in an @page at-rule.

```java
public interface ICSSMarginRule : ICSSRule
```

## Properties

| Name | Description |
| --- | --- |
| [getName](../../com.aspose.html.dom.css/icssmarginrule/name/) The name attribute must return the name of the margin at-rule. The @ character is not included in the name. |
| [getStyle](../../com.aspose.html.dom.css/icssmarginrule/style/) The style attribute must return a [`CSSStyleDeclaration`](../icssstyledeclaration/) object for the margin at-rule. |
