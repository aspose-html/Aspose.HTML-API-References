## FontEmbeddingRule enumeration

Vertegenwoordigt de regels voor het insluiten van lettertypen.

```csharp
public enum FontEmbeddingRule
```

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| Full | `1` | Alle gebruikte lettertypen worden ongewijzigd ingesloten. Houd er rekening mee dat sommige lettertypen kunnen worden gelicentieerd, waardoor het resulterende document niet meer kan worden bewerkt. Ook de ingesloten lettertypebestanden kunnen de grootte van het resulterende document aanzienlijk vergroten. |
| None | `2` | Lettertypen worden niet ingesloten. |
