## SVGSVGElement.CreateSVGTransformFromMatrix method

Maakt een SVGTransform-object buiten de documentstructuren. Het object wordt geïnitialiseerd naar de gegeven matrixtransformatie (dwz SVG_TRANSFORM_MATRIX). De waarden uit de parametermatrix worden gekopieerd, de matrixparameter wordt niet overgenomen als SVGTransform::matrix.

```csharp
public SVGTransform CreateSVGTransformFromMatrix(SVGMatrix matrix)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| matrix | SVGMatrix | De transformatiematrix. |

### Winstwaarde

Een SVGTransform-object.
