## TemplateLoadOptions.NamesAreCaseSensitive property

Eigenschap die specificeert of de namen van de template en data-elementen overeenkomen, ongeacht het hoofdlettergebruik of niet. Indien waar, is de overeenkomst hoofdlettergevoelig; indien onwaar, wordt hoofdlettergebruik genegeerd. Standaardwaarde is waar.

```csharp
public bool NamesAreCaseSensitive { get; set; }
```
