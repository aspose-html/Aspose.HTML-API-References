## ResourceHandlingOptions.MaxHandlingDepth property

Haalt of stelt de maximale diepte in van pagina's die zullen worden verwerkt. Diepte van 1 betekent dat alleen pagina's waarnaar rechtstreeks wordt verwezen vanuit het opgeslagen document, worden verwerkt. Als u deze eigenschap instelt op -1, worden alle pagina's verwerkt. Standaardwaarde is 0.

```csharp
public int MaxHandlingDepth { get; set; }
```
