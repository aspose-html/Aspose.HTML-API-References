## CSSPrimitiveValue.GetRectValue method

Deze methode wordt gebruikt om de Rect-waarde te verkrijgen. Als deze CSS-waarde geen rect-waarde bevat, wordt een DOMException gegenereerd. Wijziging van de overeenkomstige stijleigenschap kan worden bereikt met behulp van de Rect-interface.

```csharp
public abstract Rect GetRectValue()
```
