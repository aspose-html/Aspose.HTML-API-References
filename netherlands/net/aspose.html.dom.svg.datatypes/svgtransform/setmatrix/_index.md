## SVGTransform.SetMatrix method

Stelt het transformatietype in op SVG_TRANSFORM_MATRIX, waarbij de parametermatrix de nieuwe transformatie definieert. De waarden uit de parametermatrix worden gekopieerd, de matrixparameter vervangt niet SVGTransform::matrix.

```csharp
public void SetMatrix(SVGMatrix matrix)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| matrix | SVGMatrix | De nieuwe matrix voor de transformatie. |
