## HTMLOptionElement.Selected property

Vertegenwoordigt de huidige status van het overeenkomstige formulierbesturingselement, in een interactieve user-agent. Als u dit kenmerk wijzigt, wordt de status van het formulierbesturingselement gewijzigd, maar wordt de waarde van het in HTML geselecteerde kenmerk van het element niet gewijzigd.

public bool Selected { get; set; }
