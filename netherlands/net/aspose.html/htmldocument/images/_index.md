## HTMLDocument.Images property

Een verzameling van alle`IMG` elementen in een document. Het gedrag is beperkt tot`IMG` elementen voor achterwaartse compatibiliteit. Zoals voorgesteld door [[HTML4.01](http://www.w3.org/TR/1999/REC-html401-19991224)], om afbeeldingen op te nemen, mogen auteurs de`VOORWERP` element of de`IMG` element. Het wordt daarom aanbevolen om dit attribuut niet te gebruiken om de afbeeldingen in het document te vinden, maar`getElementsByTagName` met HTML 4.01 of`getElementsByTagNameNS` met XHTML 1.0.

public HTMLCollection Images { get; }
