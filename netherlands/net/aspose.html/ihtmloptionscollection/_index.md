## IHTMLOptionsCollection interface

Een`HTMLOptionsCollectie` is een lijst met knooppunten die het HTML optie-element vertegenwoordigen. Een individueel knooppunt kan worden benaderd door ordinale index of de knooppunten`naam` of`ID kaart` attributen. Collecties in de HTML DOM worden verondersteld live te zijn, wat betekent dat ze automatisch worden bijgewerkt wanneer het onderliggende document wordt gewijzigd.
