## HTMLInputElement.DefaultChecked property

Wanneer`type` heeft de waarde "radio" of "checkbox", deze vertegenwoordigt het HTML-gecontroleerde attribuut van het element. De waarde van dit attribuut verandert niet als de status van het corresponderende formulier control, in een interactieve user agent, verandert. Zie de gecontroleerde attribuutdefinitie in HTML 4.01.

```csharp
public bool DefaultChecked { get; set; }
```
