## HTMLInputElement.DefaultValue property

Wanneer de`type`attribuut van het element heeft de waarde "tekst", "bestand" of "wachtwoord", dit vertegenwoordigt de HTML-waarde attribuut van het element. De waarde van dit attribuut verandert niet als de inhoud van het overeenkomstige formulierbesturingselement, in een interactieve user-agent, verandert. Zie de definitie van het waardekenmerk in HTML 4.01.

```csharp
public string DefaultValue { get; set; }
```
