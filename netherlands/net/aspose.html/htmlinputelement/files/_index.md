## HTMLInputElement.Files property

Met het IDL-kenmerk van bestanden kunnen scripts toegang krijgen tot de geselecteerde bestanden van het element. Als het IDL-kenmerk van toepassing is, moet het bij ophalen een FileList-object retourneren dat de momenteel geselecteerde bestanden vertegenwoordigt. Hetzelfde object moet worden geretourneerd totdat de lijst met geselecteerde bestanden verandert. Als het IDL-attribuut niet van toepassing is, moet het in plaats daarvan null retourneren. [FILEAPI]

```csharp
public IFileList Files { get; }
```
