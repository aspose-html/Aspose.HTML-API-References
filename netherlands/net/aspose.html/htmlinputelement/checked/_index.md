## HTMLInputElement.Checked property

Wanneer de`type`attribuut van het element heeft de waarde "radio" of "checkbox", dit vertegenwoordigt de huidige status van het formulier control, in een interactieve user agent. Wijzigingen in dit attribuut wijzigen de status van het formulierbesturingselement, maar wijzigen niet de waarde van het HTML-gecontroleerde attribuut van het INPUT-element. Tijdens de verwerking van van een klikgebeurtenis op een invoerelement met een waarde "radio" of "checkbox", kunnen sommige implementaties de -waarde van deze eigenschap wijzigen voordat de gebeurtenis wordt verzonden in het -document. Als de standaardactie van de gebeurtenis wordt geannuleerd, kan de waarde van de eigenschap worden teruggezet naar de oorspronkelijke waarde. Dit betekent dat de waarde van deze eigenschap tijdens de afhandeling van klikgebeurtenissen afhankelijk is van de implementatie.

```csharp
public bool Checked { get; set; }
```
