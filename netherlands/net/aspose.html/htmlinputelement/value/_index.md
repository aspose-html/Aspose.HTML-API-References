## HTMLInputElement.Value property

Wanneer de`type` attribuut van het element heeft de waarde "text", "file" of "password", dit vertegenwoordigt de huidige inhoud van de corresponderende form control, in een interactieve user agent. Als u dit kenmerk wijzigt, verandert de inhoud van het formulierbesturingselement, maar verandert de waarde van het HTML-waardekenmerk van het element niet. Wanneer de`type`attribuut van het element heeft de waarde "button", "hidden", "submit", "reset", "image", "checkbox" of "radio", dit vertegenwoordigt de HTML-waarde attribuut van het element. Zie de definitie van het waardekenmerk in HTML 4.01.

```csharp
public string Value { get; set; }
```
