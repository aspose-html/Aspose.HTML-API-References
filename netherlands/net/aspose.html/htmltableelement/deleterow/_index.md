## HTMLTableElement.DeleteRow method

Verwijder een tabelrij.

```csharp
public void DeleteRow(int index)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| index | Int32 | De index van de rij die moet worden verwijderd. Deze index begint vanaf 0 en is relatief ten opzichte van de logische volgorde (niet documentvolgorde) van alle rijen in de tabel. Als de index -1 is, wordt de laatste rij in de tabel verwijderd. |
