## HTMLSelectElement.SelectedIndex property

De ordinale index van de geselecteerde optie, beginnend bij 0. De waarde -1 wordt geretourneerd als er geen element is geselecteerd. Als meerdere opties zijn geselecteerd, wordt de index van de eerst geselecteerde optie geretourneerd.

public int SelectedIndex { get; set; }
