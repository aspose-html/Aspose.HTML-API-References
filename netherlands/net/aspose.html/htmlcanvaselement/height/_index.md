## HTMLCanvasElement.Height property

Is een positief geheel getal dat het hoogte-HTML-attribuut van het canvaselement weergeeft, geïnterpreteerd in CSS-pixels. Als het attribuut niet is opgegeven, of als het is ingesteld op een ongeldige waarde, zoals een negatieve waarde, wordt de standaardwaarde van 150 gebruikt.

```csharp
public ulong Height { get; set; }
```
