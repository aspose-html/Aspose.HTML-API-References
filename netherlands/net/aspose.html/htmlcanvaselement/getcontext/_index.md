## HTMLCanvasElement.GetContext method

Retourneert een tekencontext op het canvas, of null als de context-ID niet wordt ondersteund. Met een tekencontext kunt u op het canvas tekenen.

```csharp
public object GetContext(string contextId)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| contextId | String | Is een tekenreeks die de context-ID bevat die de tekencontext definieert die aan het canvas is gekoppeld |

### Winstwaarde

Een RenderContext
