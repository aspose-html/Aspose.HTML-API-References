## ToDataURL() {#todataurl}

retourneert een gegevens-URI die een weergave van de afbeelding bevat in de indeling die is opgegeven door de parameter type (standaard ingesteld op PNG). De geretourneerde afbeelding heeft een resolutie van 96 dpi.

```csharp
public string ToDataURL()
```

### Winstwaarde

Een DOMString die de gevraagde gegevens-URI bevat.
