## CreatePattern(HTMLImageElement, string) {#createpattern_1}

Maakt een patroon met behulp van de opgegeven afbeelding (een CanvasImageSource). Het herhaalt de bron in de richtingen gespecificeerd door het herhalingsargument.

```csharp
public ICanvasPattern CreatePattern(HTMLImageElement image, string repetition)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| image | HTMLImageElement | Een HTMLImageElement dat moet worden gebruikt als de te herhalen afbeelding |
| repetition | String | Een tekenreeks die aangeeft hoe de afbeelding moet worden herhaald. |

### Winstwaarde

Een ondoorzichtig object dat een patroon beschrijft.
