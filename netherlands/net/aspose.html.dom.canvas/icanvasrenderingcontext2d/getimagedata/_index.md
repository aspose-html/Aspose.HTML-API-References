## ICanvasRenderingContext2D.GetImageData method

Retourneert een ImageData-object dat de onderliggende pixelgegevens vertegenwoordigt voor het gebied van het canvas dat wordt aangegeven door de rechthoek die begint bij (sx, sy) en een sw-breedte en sh-hoogte heeft. Deze methode wordt niet beïnvloed door de canvas-transformatiematrix.

```csharp
public IImageData GetImageData(double sx, double sy, double sw, double sh)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| sx | Double | De x-coördinaat van de linkerbovenhoek van de rechthoek waaruit de ImageData wordt geëxtraheerd. |
| sy | Double | De y-coördinaat van de linkerbovenhoek van de rechthoek waaruit de ImageData wordt geëxtraheerd. |
| sw | Double | De breedte van de rechthoek waaruit de ImageData wordt geëxtraheerd. |
| sh | Double | De hoogte van de rechthoek waaruit de ImageData wordt geëxtraheerd. |

### Winstwaarde

Een ImageData-object dat de afbeeldingsgegevens bevat voor de gegeven rechthoek van het canvas.
