## ICanvasRenderingContext2D.CreateRadialGradient method

```csharp
public ICanvasGradient CreateRadialGradient(double x0, double y0, double r0, double x1, double y1, 
    double r1)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x0 | Double | De x-as van de coördinaat van de startcirkel. |
| y0 | Double | De y-as van de coördinaat van de startcirkel |
| r0 | Double | De straal van de startcirkel. |

| y1 | Double | De y-as van de coördinaat van de eindcirkel. |
