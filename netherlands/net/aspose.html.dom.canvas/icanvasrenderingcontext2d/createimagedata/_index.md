## CreateImageData(double, double) {#createimagedata_1}

Maakt een nieuw, leeg ImageData-object met de opgegeven afmetingen. Alle pixels in het nieuwe object zijn transparant zwart.

```csharp
public IImageData CreateImageData(double sw, double sh)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| sw | Double | De breedte om het nieuwe ImageData-object te geven. |
| sh | Double | De hoogte om het nieuwe ImageData-object te geven. |

### Winstwaarde

Een nieuw ImageData-object met de opgegeven breedte en hoogte. Het nieuwe object is gevuld met transparante zwarte pixels.
