De **Aspose.Html.Dom.Canvas** naamruimte biedt interfaces voor kwalificatie van 2D-renderingdetails, evenals opties voor het vullen van en Path2D-klasse.

## Klassen

| Klas | Beschrijving |
| --- | --- |
| [Path2D](./path2d/) | De Path2D-interface van de Canvas 2D API wordt gebruikt om paden te declareren die later worden gebruikt op CanvasRenderingContext2D-objecten. De padmethoden van de CanvasRenderingContext2D-interface zijn ook aanwezig op deze interface en stellen u in staat om -paden te maken die u kunt behouden en naar behoefte opnieuw kunt afspelen op een canvas. |
## Interfaces

| Koppel | Beschrijving |
| --- | --- |
| [ICanvasDrawingStyles](./icanvasdrawingstyles/) | De ICanvasDrawingStyles-interface biedt methoden en eigenschappen die bepalen hoe lijnen worden getekend en hoe tekst wordt opgemaakt. |
| [ICanvasGradient](./icanvasgradient/) | Vertegenwoordigt een ondoorzichtig object dat een verloop beschrijft. |
| [ICanvasPathMethods](./icanvaspathmethods/) | De ICanvasPathMethods-interface wordt gebruikt om paden van objecten te manipuleren. |
| [ICanvasPattern](./icanvaspattern/) | Vertegenwoordigt een ondoorzichtig object dat een patroon beschrijft, gebaseerd op een afbeelding, een canvas of een video. |
| [ICanvasRenderingContext2D](./icanvasrenderingcontext2d/) | De ICanvasRenderingContext2D-interface wordt gebruikt voor het tekenen van rechthoeken, tekst, afbeeldingen en andere objecten op het canvaselement. Het biedt de 2D-renderingcontext voor het tekenoppervlak van een canvaselement. |
| [IImageData](./iimagedata/) | Maakt een ImageData-object van een gegeven Uint8ClampedArray en de grootte van de afbeelding die het bevat. Als er geen array wordt opgegeven, wordt een afbeelding van een zwarte rechthoek gemaakt. |
| [ITextMetrics](./itextmetrics/) | Vertegenwoordigt de dimensie van een tekst op het canvas. |
