## Arc(double, double, double, double, double) {#arc}

Voegt een boog toe aan het pad dat gecentreerd is op (x, y) positie met straal r beginnend bij startAngle en eindigend bij endAngle, in de gegeven richting tegen de klok in (standaard met de klok mee).

```csharp
public void Arc(double x, double y, double radius, double startAngle, double endAngle)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Double | De x-coördinaat van het middelpunt van de boog. |
| y | Double | De y-coördinaat van het middelpunt van de boog. |
| radius | Double | De straal van de boog. |
| startAngle | Double | De hoek waaronder de boog begint, met de klok mee gemeten vanaf de positieve x-as en uitgedrukt in radialen. |
| endAngle | Double | De hoek waaronder de boog eindigt, met de klok mee gemeten vanaf de positieve x-as en uitgedrukt in radialen. |

## Arc(double, double, double, double, double, bool) {#arc_1}

```csharp
public void Arc(double x, double y, double radius, double startAngle, double endAngle, 
    bool counterclockwise)
```

| Parameter | Type | Beschrijving |

| y | Double | De y-coördinaat van het middelpunt van de boog. |

| endAngle | Double | De hoek waaronder de boog eindigt, met de klok mee gemeten vanaf de positieve x-as en uitgedrukt in radialen. |
| counterclockwise | Boolean | Zorgt ervoor dat de boog tegen de klok in wordt getrokken tussen de twee hoeken. Standaard wordt het met de klok mee getekend. |
