## Ellipse(double, double, double, double, double, double, double) {#ellipse}

Voegt een ellips toe aan het pad dat is gecentreerd op (x, y) positie met de stralen radiusX en radiusY beginnend bij startAngle en eindigend bij endAngle in de gegeven richting tegen de klok in (standaard met de klok mee).

```csharp
public void Ellipse(double x, double y, double radiusX, double radiusY, double rotation, 
    double startAngle, double endAngle)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| x | Double | De x-as van de coördinaat voor het middelpunt van de ellips. |
| y | Double | De y-as van de coördinaat voor het middelpunt van de ellips. |
| radiusX | Double | De straal van de hoofdas van de ellips. |
| radiusY | Double | De straal van de kleine as van de ellips. |
| rotation | Double | De rotatie voor deze ellips, uitgedrukt in radialen. |
| startAngle | Double | Het startpunt, gemeten vanaf de x-as, van waaruit het wordt getekend, uitgedrukt in radialen. |
| endAngle | Double | De hoek van de eindellips waarnaar deze wordt getekend, uitgedrukt in radialen. |
