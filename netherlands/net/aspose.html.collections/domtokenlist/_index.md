## DOMTokenList class

De klasse DOMTokenList vertegenwoordigt een reeks door spaties gescheiden tokens. Het wordt geïndexeerd beginnend met 0, net als bij JavaScript Array-objecten. DOMTokenList is altijd hoofdlettergevoelig.

```csharp
public class DOMTokenList : DOMObject, IEnumerable<string>
```
