## PageSetup.AdjustToWidestPage property

फ़्लैग प्राप्त करता है या सेट करता है जो यह निर्धारित करता है कि पृष्ठ का आकार दस्तावेज़ में सबसे चौड़े पृष्ठ पर कब समायोजित किया जाएगा। यह विकल्प समय लेने वाला है इसलिए दस्तावेज़ प्रसंस्करण का समय दो बार बढ़ाया जा सकता है। समायोजन तभी होगा जब दस्तावेज़ में सबसे चौड़ा पृष्ठ निर्धारित पृष्ठ आकार से अधिक चौड़ा हो[`PageSetup`](../) . समायोजित पृष्ठ आकार दस्तावेज़ में सभी पृष्ठों के लिए उपयोग किया जाएगा।

```csharp
public bool AdjustToWidestPage { get; set; }
```
