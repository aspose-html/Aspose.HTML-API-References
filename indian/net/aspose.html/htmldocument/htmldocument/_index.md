का एक नया उदाहरण प्रारंभ करता है[`HTMLDocument`](../) कक्षा। कंस्ट्रक्टर समकालिक रूप से काम करता है, यह सभी बाहरी संसाधनों (छवियों, स्क्रिप्ट, आदि) के लोड होने की प्रतीक्षा करता है। दस्तावेज़ को एसिंक्रोनस रूप से लोड करने के लिए विधि का उपयोग करें[`Navigate`](../../../aspose.html.dom/document/navigate/) या इसके ओवरलोड्स. या आप उपयुक्त फ़्लैग्स सेट करके कुछ बाहरी संसाधनों को लोड करना अक्षम कर सकते हैं[`Security`](../../../aspose.html.dom/ibrowsingcontext/security/) . दस्तावेज़ लोड करना स्ट्रीम में वर्तमान स्थिति से शुरू होता है।

```csharp
public HTMLDocument(Stream content, Url baseUri, Configuration configuration)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| content | Stream | दस्तावेज़ सामग्री। |
| baseUri | Url | दस्तावेज़ का आधार URI। |
| configuration | Configuration | पर्यावरण विन्यास। |

### अपवाद

| अपवाद | स्थिति |
| --- | --- |
| ArgumentNullException | `baseUri` है`व्यर्थ`. |
