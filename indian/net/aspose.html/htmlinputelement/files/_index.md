## HTMLInputElement.Files property

फ़ाइलें IDL विशेषता स्क्रिप्ट को तत्व की चयनित फ़ाइलों तक पहुँचने की अनुमति देती हैं। प्राप्त करने पर, यदि IDL विशेषता लागू होती है, तो उसे एक FileList ऑब्जेक्ट वापस करना होगा जो वर्तमान चयनित फ़ाइलों का प्रतिनिधित्व करता है। जब तक चयनित फ़ाइलों की सूची नहीं बदल जाती, तब तक उसी वस्तु को लौटाया जाना चाहिए। यदि IDL विशेषता लागू नहीं होती है, तो उसे इसके बजाय शून्य होना चाहिए। [FILEAPI]

```csharp
public IFileList Files { get; }
```
