## HTMLInputElement.Value property

जब`प्रकार` तत्व की विशेषता का मान "टेक्स्ट", "फ़ाइल" या "पासवर्ड" है, यह एक इंटरैक्टिव उपयोगकर्ता एजेंट में संबंधित प्रपत्र नियंत्रण की वर्तमान सामग्री का प्रतिनिधित्व करता है। इस विशेषता को बदलने से प्रपत्र नियंत्रण की सामग्री बदल जाती है, लेकिन तत्व के HTML मान विशेषता का मान नहीं बदलता है। जब`प्रकार`तत्व की विशेषता का मान "बटन", "छिपा हुआ", "सबमिट", "रीसेट", "छवि", "चेकबॉक्स" या "रेडियो" है, यह तत्व के HTML मान विशेषता का प्रतिनिधित्व करता है। HTML 4.01. में मान विशेषता परिभाषा देखें

```csharp
public string Value { get; set; }
```
