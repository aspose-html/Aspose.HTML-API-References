द **Aspose.Html** नेमस्पेस में HTML दस्तावेज़ों में हेरफेर करने के लिए वर्ग और विधियाँ शामिल हैं, चाहे वे सरल हों या जटिल और चलते-फिरते।  **Aspose.HTML**for .NET डेवलपर्स को सम्मिलित करने, निकालने, HTML नोड्स को बदलने, CSS शैली की जानकारी निकालने, या तो NodeIterator, ट्रीवॉकर द्वारा HTML दस्तावेज़ के माध्यम से नेविगेट करने की अनुमति देता है जो ट्रैवर्सल विनिर्देशों, XPath या CSS चयनकर्ता प्रश्नों द्वारा प्रदान किया जाता है। यह स्क्रिप्टिंग भी प्रदान करता है जो to को जावास्क्रिप्ट के माध्यम से HTML DOM में हेरफेर करने की अनुमति देता है। HTML के साथ-साथ, यह API EPUB और MHTML को लोड करने की क्षमता भी प्रदान करता है। Aspose API अपने अंतर फ़ाइल प्रारूप रूपांतरण सुविधाओं के लिए प्रसिद्ध हैं और यह API HTML फ़ाइल को लोड करने और आउटपुट को PDF, XPS और JPEG, PNG, BMP और TIFF सहित रेखापुंज छवि प्रारूपों में प्रस्तुत करने की क्षमता भी प्रदान करता है।
