## HTMLFormElement.Enctype property

सबमिट किए गए फॉर्म का सामग्री प्रकार, आम तौर पर "आवेदन/x-www-form-urlencoded"। HTML 4.01 में enctype विशेषता परिभाषा देखें। इस विधि को लागू करते समय चालू होने वाले हैंडलर को ट्रिगर होने की गारंटी नहीं है। व्यवहार ऐतिहासिक कारणों से असंगत है और लेखकों को विशेष पर भरोसा नहीं करना चाहिए।

```csharp
public string Enctype { get; set; }
```
