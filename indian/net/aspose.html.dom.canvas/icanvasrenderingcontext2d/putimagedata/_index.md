## PutImageData(IImageData, double, double) {#putimagedata}

दिए गए ImageData ऑब्जेक्ट से डेटा को बिटमैप पर पेंट करता है। यदि एक गंदा आयत प्रदान किया जाता है, तो उस आयत के केवल पिक्सेल ही पेंट किए जाते हैं। यह विधि कैनवास परिवर्तन मैट्रिक्स से प्रभावित नहीं है।

```csharp
public void PutImageData(IImageData imagedata, double dx, double dy)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| imagedata | IImageData | एक ImageData ऑब्जेक्ट जिसमें पिक्सेल मानों की सरणी होती है। |
| dx | Double | क्षैतिज स्थिति (x-निर्देशांक) जिस पर छवि डेटा को गंतव्य कैनवास में रखना है। |
| dy | Double | कार्यक्षेत्र स्थिति (y-निर्देशांक) जिस पर छवि डेटा को गंतव्य कैनवास में रखना है। |

## PutImageData(IImageData, double, double, double, double, double, double) {#putimagedata_1}

दिए गए ImageData ऑब्जेक्ट से डेटा को बिटमैप पर पेंट करता है। यदि एक गंदा आयत प्रदान किया जाता है, तो उस आयत के केवल पिक्सेल ही पेंट किए जाते हैं। यह विधि कैनवास परिवर्तन मैट्रिक्स से प्रभावित नहीं है।

```csharp
public void PutImageData(IImageData imagedata, double dx, double dy, double dirtyX, double dirtyY, 
    double dirtyWidth, double dirtyHeight)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| imagedata | IImageData | एक ImageData ऑब्जेक्ट जिसमें पिक्सेल मानों की सरणी होती है। |
| dx | Double | क्षैतिज स्थिति (x-निर्देशांक) जिस पर छवि डेटा को गंतव्य कैनवास में रखना है। |
| dy | Double | कार्यक्षेत्र स्थिति (y-निर्देशांक) जिस पर छवि डेटा को गंतव्य कैनवास में रखना है। |
| dirtyX | Double | क्षैतिज स्थिति (एक्स-समन्वय)। आपके छवि डेटा के ऊपरी बाएँ हाथ के कोने का x निर्देशांक। 0 के लिए डिफ़ॉल्ट। |
| dirtyY | Double | लंबवत स्थिति (वाई-निर्देशांक)। आपके छवि डेटा के ऊपरी बाएँ कोने का y निर्देशांक। 0 के लिए डिफ़ॉल्ट। |
| dirtyWidth | Double | पेंट किए जाने वाले आयत की चौड़ाई। छवि डेटा की चौड़ाई के लिए डिफ़ॉल्ट। |
| dirtyHeight | Double | पेंट किए जाने वाले आयत की ऊँचाई। छवि डेटा की ऊंचाई के लिए डिफ़ॉल्ट। |
