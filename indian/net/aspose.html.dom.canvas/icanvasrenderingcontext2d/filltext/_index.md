## FillText(string, double, double, double) {#filltext_1}

दिए गए पाठ को दिए गए (x, y) स्थान पर ड्रा (भरता है) करता है।

```csharp
public void FillText(string text, double x, double y, double maxWidth)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| text | String | वर्तमान फ़ॉन्ट, textAlign, textBaseline, और दिशा मानों का उपयोग करके आरेखित किया जाने वाला पाठ। |
| x | Double | टेक्स्ट के शुरुआती बिंदु के लिए निर्देशांक का x अक्ष। |
| y | Double | टेक्स्ट के शुरुआती बिंदु के लिए निर्देशांक का y अक्ष। |
| maxWidth | Double | आकर्षित करने के लिए अधिकतम चौड़ाई। यदि निर्दिष्ट किया गया है, और स्ट्रिंग को इस चौड़ाई से अधिक व्यापक होने के लिए गणना की जाती है, तो फ़ॉन्ट को अधिक क्षैतिज रूप से संघनित फ़ॉन्ट का उपयोग करने के लिए समायोजित किया जाता है (यदि कोई उपलब्ध है या यथोचित पठनीय है तो वर्तमान फ़ॉन्ट को क्षैतिज रूप से स्केल करके संश्लेषित किया जा सकता है) या एक छोटा फ़ॉन्ट। |
