## Ellipse(double, double, double, double, double, double, double) {#ellipse}

उस पथ में एक दीर्घवृत्त जोड़ता है जो (x, y) स्थिति पर केंद्रित है, त्रिज्या त्रिज्या X और त्रिज्या Y के साथ startAngle पर शुरू होता है और एंटीक्लॉकवाइज़ (घड़ी की दिशा में डिफ़ॉल्ट) द्वारा दी गई दिशा में अंतकोण पर समाप्त होता है।

```csharp
public void Ellipse(double x, double y, double radiusX, double radiusY, double rotation, 
    double startAngle, double endAngle)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Double | दीर्घवृत्त के केंद्र के लिए निर्देशांक का x अक्ष। |
| y | Double | दीर्घवृत्त के केंद्र के लिए निर्देशांक का y अक्ष। |
| radiusX | Double | दीर्घवृत्त की प्रमुख-अक्ष त्रिज्या। |
| radiusY | Double | दीर्घवृत्त की लघु-अक्ष त्रिज्या। |
| rotation | Double | इस दीर्घवृत्त के लिए घूर्णन, रेडियन में व्यक्त किया गया। |
| startAngle | Double | प्रारंभिक बिंदु, एक्स अक्ष से मापा जाता है, जिससे इसे खींचा जाएगा, रेडियन में व्यक्त किया जाएगा। |
| endAngle | Double | अंत दीर्घवृत्त का कोण जिस पर इसे खींचा जाएगा, रेडियन में व्यक्त किया गया। |

## Ellipse(double, double, double, double, double, double, double, bool) {#ellipse_1}

उस पथ में एक दीर्घवृत्त जोड़ता है जो (x, y) स्थिति पर केंद्रित है, त्रिज्या त्रिज्या X और त्रिज्या Y के साथ startAngle पर शुरू होता है और एंटीक्लॉकवाइज़ (घड़ी की दिशा में डिफ़ॉल्ट) द्वारा दी गई दिशा में अंतकोण पर समाप्त होता है।

```csharp
public void Ellipse(double x, double y, double radiusX, double radiusY, double rotation, 
    double startAngle, double endAngle, bool anticlockwise)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x | Double | दीर्घवृत्त के केंद्र के लिए निर्देशांक का x अक्ष। |
| y | Double | दीर्घवृत्त के केंद्र के लिए निर्देशांक का y अक्ष। |
| radiusX | Double | दीर्घवृत्त की प्रमुख-अक्ष त्रिज्या। |
| radiusY | Double | दीर्घवृत्त की लघु-अक्ष त्रिज्या। |
| rotation | Double | इस दीर्घवृत्त के लिए घूर्णन, रेडियन में व्यक्त किया गया। |
| startAngle | Double | प्रारंभिक बिंदु, एक्स अक्ष से मापा जाता है, जिससे इसे खींचा जाएगा, रेडियन में व्यक्त किया जाएगा। |
| endAngle | Double | अंत दीर्घवृत्त का कोण जिस पर इसे खींचा जाएगा, रेडियन में व्यक्त किया गया। |
| anticlockwise | Boolean | एक वैकल्पिक बूलियन जो, यदि सत्य है, दीर्घवृत्त को वामावर्त (वामावर्त) खींचता है, अन्यथा दक्षिणावर्त दिशा में। |
