## Path2D.BezierCurveTo method

पथ में घन बेज़ियर वक्र जोड़ता है। इसके लिए तीन बिंदुओं की आवश्यकता है। पहले दो बिंदु नियंत्रण बिंदु हैं और तीसरा अंतिम बिंदु है। प्रारंभिक बिंदु वर्तमान पथ में अंतिम बिंदु है, जिसे बेजियर वक्र बनाने से पहले मूव टू () का उपयोग करके बदला जा सकता है।

```csharp
public void BezierCurveTo(double cp1x, double cp1y, double cp2x, double cp2y, double x, double y)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| cp1x | Double | पहले नियंत्रण बिंदु के लिए निर्देशांक का x अक्ष। |
| cp1y | Double | पहले नियंत्रण बिंदु के लिए निर्देशांक का y अक्ष। |
| cp2x | Double | दूसरे नियंत्रण बिंदु के लिए निर्देशांक का x अक्ष। |
| cp2y | Double | दूसरे नियंत्रण बिंदु के लिए निर्देशांक का y अक्ष। |
| x | Double | अंत बिंदु के लिए निर्देशांक का x अक्ष। |
| y | Double | अंत बिंदु के लिए निर्देशांक का y अक्ष। |
