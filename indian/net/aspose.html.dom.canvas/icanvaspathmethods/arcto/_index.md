## ICanvasPathMethods.ArcTo method

दिए गए नियंत्रण बिंदुओं और त्रिज्या के साथ पथ में एक चाप जोड़ता है, जो पिछले बिंदु से एक सीधी रेखा से जुड़ा होता है।

```csharp
public void ArcTo(double x1, double y1, double x2, double y2, double radius)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| x1 | Double | एक्स-अक्ष पहले नियंत्रण बिंदु के लिए निर्देशांक करता है। |
| y1 | Double | y-अक्ष पहले नियंत्रण बिंदु के लिए निर्देशांक करता है। |
| x2 | Double | एक्स-अक्ष दूसरे नियंत्रण बिंदु के लिए निर्देशांक करता है। |
| y2 | Double | दूसरे नियंत्रण बिंदु के लिए y-अक्ष निर्देशांक। |
| radius | Double | चाप की त्रिज्या। |
