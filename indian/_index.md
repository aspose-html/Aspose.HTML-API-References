## .NET के लिए Aspose.HTML

{{% alert color="primary" %}} 

![.NET उत्पाद लोगो के लिए Aspose.HTML](../home_1.png)


Aspose.HTML for .NET एक क्रॉस-प्लेटफ़ॉर्म क्लास लाइब्रेरी है जो आपके एप्लिकेशन को HTML हेरफेर कार्यों की एक विस्तृत श्रृंखला करने में सक्षम बनाती है। चूंकि HTML आमतौर पर अन्य तकनीकों के साथ संयोजन में उपयोग किया जाता है, Aspose.HTML न केवल HTML5 बल्कि CSS3, SVG, और HTML कैनवास विनिर्देशों का भी समर्थन करता है। Aspose.HTML का उपयोग करके, आप वेब पृष्ठों को लोड कर सकते हैं, सामग्री का विश्लेषण और संपादन कर सकते हैं, पृष्ठों को निम्न स्वरूपों में सहेज और परिवर्तित कर सकते हैं: HTML, XHTML, MHTML, SVG, Markdown, PDF, XPS,
EPUB, PNG, TIFF, JPEG, BMP और अन्य प्रारूप।

## जावा के लिए Aspose.HTML

{{% alert color="primary" %}} 

![जावा उत्पाद लोगो के लिए Aspose.HTML](../home_2.png)


जावा के लिए Aspose.HTML एक उन्नत HTML हेरफेर एपीआई है जिसे सीधे जावा अनुप्रयोगों के भीतर HTML हेरफेर कार्यों की एक विस्तृत श्रृंखला करने के लिए बनाया गया है। एपीआई निश्चित-लेआउट प्रारूपों जैसे पीडीएफ और एक्सपीएस, और कई रेखापुंज छवि प्रारूपों के लिए एक उच्च विश्वस्तता रेंडरिंग इंजन भी प्रदान करता है।
